Python Map

Introduction

The map() function executes a specified function for each item in an iterable. The item is sent to the function as a parameter.

Syntax

map(function, iterable)

Example:

map(function, iterables)
  • function: The function to execute for each item.
  • iterables: A sequence, collection or an iterator object. You can send as many iterables as you like, just make sure the function has one parameter for each iterable.

Usage with Function

Calculate the length of each word in the tuple:

def myfunc(n):
    return len(n)

x = map(myfunc, ('apple', 'banana', 'cherry'))

print(list(x))
# Output: [5, 6, 6]

Usage with Lambda

You can use lambda functions to keep the code concise:

numbers = [1, 2, 3, 4]
result = map(lambda x: x + x, numbers)
print(list(result))
# Output: [2, 4, 6, 8]

Mapping Multiple Iterators

If you pass multiple iterables, the function must accept that many arguments, and is applied to the items from all iterables in parallel.

numbers1 = [1, 2, 3]
numbers2 = [4, 5, 6]

result = map(lambda x, y: x + y, numbers1, numbers2)
print(list(result))
# Output: [5, 7, 9]

Worksheet

1. Write a program to triple all numbers in a given list.

nums = [1, 2, 3, 4, 5]
tripled = list(map(lambda x: x * 3, nums))
print(tripled)

2. Write a program to add three lists using map and lambda.

nums1 = [1, 2, 3]
nums2 = [4, 5, 6]
nums3 = [7, 8, 9]

result = list(map(lambda x, y, z: x + y + z, nums1, nums2, nums3))
print(result)

3. Write a program to listify the list of given strings individually.

color = ['Red', 'Blue', 'Black', 'White', 'Pink']
result = list(map(list, color))
print(result)

4. Write a program to calculate the square of the elements of a list.

nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
print(squared)

5. Write a program to convert all the characters in uppercase and lowercase and eliminate duplicate letters from a given sequence.

def change_cases(s):
    return str(s).upper(), str(s).lower()

chars = {'a', 'b', 'E', 'f', 'a', 'i', 'o', 'U', 'a'}
result = map(change_cases, chars)
print(set(result))

6. Write a program to add two given lists and find the difference between them.

nums1 = [6, 5, 3, 9]
nums2 = [0, 1, 7, 7]

result = list(map(lambda x, y: (x + y, x - y), nums1, nums2))
print(result)

7. Write a program to convert a given list of integers and a tuple of integers in a list of strings.

nums_list = [1, 2, 3, 4]
nums_tuple = (0, 1, 2, 3)

result_list = list(map(str, nums_list))
result_tuple = tuple(map(str, nums_tuple))

print(result_list)
print(result_tuple)

8. Write a program to create a new list taking specific elements from a tuple and convert a string value to integer.

student_data = [('Alberto Franco', '15/05/2002', '35kg'), ('Gino Mcneill', '17/05/2002', '37kg'), ('Ryan Parkes', '16/02/1999', '39kg'), ('Eesha Hinton', '25/09/1998', '35kg')]

students_weight = list(map(lambda x: int(x[2][:-2]), student_data))

9. Write a program to compute the square of first N Fibonacci numbers.

import itertools

n = 10
def fibonacci_nums(x=0, y=1):
    yield x
    while True:
        yield y
        x, y = y, x + y

print("First 10 Fibonacci numbers:")
result = list(itertools.islice(fibonacci_nums(), n))
print(result)

print("\nSquare of the said numbers:")
print(list(map(lambda x: x**2, result)))

10. Write a program to compute the sum of elements of a given array of integers.

from array import array

def array_sum(nums_arr):
    sum_val = 0
    for n in nums_arr:
        sum_val += n
    return sum_val

nums = array('i', [1, 2, 3, 4, 5, -15])
# Or simply using sum, but assuming map concept was intended, 
# map is for transformation, sum is reduction.
