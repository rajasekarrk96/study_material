Python XLSX (Excel)

Introduction

Python does not have built-in support for Excel files, but we can use the openpyxl library to read and write Excel 2010 xlsx/xlsm/xltx/xltm files.

Installation

To use it, you must install it using pip:

pip install openpyxl

Create & Write to Excel

You can create a workbook, access the active sheet, and assign values to cells.

Syntax:

wb = Workbook()
sheet = wb.active

Example:

from openpyxl import Workbook

# Create a workbook and select active sheet
wb = Workbook()
sheet = wb.active

# Assign values
sheet['A1'] = "Name"
sheet['B1'] = "Age"

sheet['A2'] = "Alice"
sheet['B2'] = 25

# Save the file
wb.save("sample.xlsx")
print("File saved successfully.")

Read from Excel

You can load an existing workbook and read values from cells.

from openpyxl import load_workbook

# Load the workbook
wb = load_workbook("sample.xlsx")
sheet = wb.active

# Read specific cell
print(sheet['A2'].value) # Output: Alice

# Iterate through rows
print("\nAll Data:")
for row in sheet.iter_rows(values_only=True):
    print(row)

Append Data

You can easily append rows to the end of the sheet.

from openpyxl import load_workbook

wb = load_workbook("sample.xlsx")
sheet = wb.active

# Data to append
new_data = [
    ("Bob", 30),
    ("Charlie", 35)
]

for row in new_data:
    sheet.append(row)

wb.save("sample.xlsx")
print("Data appended.")

Worksheet

1. Write a program to create a new Excel file named 'students.xlsx'.

from openpyxl import Workbook
wb = Workbook()
wb.save('students.xlsx')
print("Created")

2. Write a program to write 'Hello' to cell A1 of 'students.xlsx'.

from openpyxl import load_workbook
wb = load_workbook('students.xlsx')
ws = wb.active
ws['A1'] = 'Hello'
wb.save('students.xlsx')

3. Write a program to read the value from cell A1.

from openpyxl import load_workbook
wb = load_workbook('students.xlsx')
ws = wb.active
print(ws['A1'].value)

4. Write a program to append a list of values [1, 2, 3] as a row.

from openpyxl import load_workbook
wb = load_workbook('students.xlsx')
ws = wb.active
ws.append([1, 2, 3])
wb.save('students.xlsx')

5. Write a program to create a header row: Name, City, Age.

from openpyxl import Workbook
wb = Workbook()
ws = wb.active
ws.append(["Name", "City", "Age"])
wb.save('data.xlsx')

6. Write a program to read all values in the first column.

from openpyxl import load_workbook
wb = load_workbook('data.xlsx')
ws = wb.active
for row in ws.iter_rows(min_col=1, max_col=1, values_only=True):
    print(row[0])

7. Write a program to change the sheet title to 'MyData'.

from openpyxl import load_workbook
wb = load_workbook('data.xlsx')
ws = wb.active
ws.title = "MyData"
wb.save('data.xlsx')

8. Write a program to create a new sheet named 'Sheet2'.

from openpyxl import load_workbook
wb = load_workbook('data.xlsx')
wb.create_sheet("Sheet2")
wb.save('data.xlsx')

9. Write a program to access specific cell using text coordinates (e.g. cell(row=1, column=2)).

from openpyxl import load_workbook
wb = load_workbook('data.xlsx')
ws = wb.active
# Row 1, Col 2 is B1
print(ws.cell(row=1, column=2).value)

10. Write a program to iterate over all rows using values_only=True.

from openpyxl import load_workbook
wb = load_workbook('data.xlsx')
ws = wb.active
for row in ws.values:
    print(row)
