Python Zip

Introduction

The zip() function takes iterables (can be zero or more), aggregates them in a tuple, and returns it.

Syntax

zip(iterables)

Example:

zip(*iterables)
  • iterables: Built-in iterables (like: list, string, dict), or user-defined iterables.

Examples

Joining two tuples together

a = ("John", "Charles", "Mike")
b = ("Jenny", "Christy", "Monica")

x = zip(a, b)

# use the tuple() function to display a readable version of the result:
print(tuple(x))
# Output: (('John', 'Jenny'), ('Charles', 'Christy'), ('Mike', 'Monica'))

Handling Different Lengths

If the passed iterables have different lengths, the iterator with the least items decides the length of the new iterator.

a = ("John", "Charles", "Mike")
b = ("Jenny", "Christy", "Monica", "Vicky")

x = zip(a, b)

print(tuple(x))
# Output: (('John', 'Jenny'), ('Charles', 'Christy'), ('Mike', 'Monica'))

Creating a Dictionary

You can easily create a dictionary from two lists using zip() and dict(). 

keys = ['name', 'age', 'city']
values = ['Alice', 25, 'New York']

my_dict = dict(zip(keys, values))
print(my_dict)
# Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

Unzipping Values

The * operator can be used to unzip a list of tuples back into separate tuples.

pairs = [(1, 'a'), (2, 'b'), (3, 'c')]

numbers, letters = zip(*pairs)

print(numbers) # (1, 2, 3)
print(letters) # ('a', 'b', 'c')

Worksheet

1. Write a program to combine two lists into a list of tuples.

list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']
result = list(zip(list1, list2))
print(result)

2. Write a program to create a dictionary from two lists.

keys = ['x', 'y', 'z']
vals = [10, 20, 30]
result = dict(zip(keys, vals))
print(result)

3. Write a program to unzip a list of tuples into individual lists.

zipped = [(1, 4), (2, 5), (3, 6)]
l1, l2 = zip(*zipped)
print(list(l1))
print(list(l2))

4. Write a program to iterate over two lists simultaneously.

names = ["Anna", "Bob"]
ages = [24, 30]
for name, age in zip(names, ages):
    print(f"{name} is {age} years old")

5. Write a program to sort a list of tuples by the second element using zip.

# Actually simpler with sort(key=...), but demonstratively using zip for parallel sort idea
first = ['a', 'b', 'c']
second = [3, 1, 2]

# Zip, sort, unzip
zipped = zip(second, first)
sorted_zip = sorted(zipped)
result = list(zip(*sorted_zip))[1]
print(result)

6. Write a program to combine three lists using zip.

l1 = [1, 2]
l2 = [3, 4]
l3 = [5, 6]
print(list(zip(l1, l2, l3)))

7. Write a program to calculate the pairwise sum of two lists.

a = [1, 2, 3]
b = [4, 5, 6]
sums = [x + y for x, y in zip(a, b)]
print(sums)

8. Write a program to map keys to values, filtering out keys with None values using zip. 

keys = ['a', 'b', 'c']
values = [1, None, 3]
result = {k: v for k, v in zip(keys, values) if v is not None}
print(result)

9. Write a program to transpsose a matrix (list of lists) using zip.

matrix = [[1, 2], [3, 4], [5, 6]]
transpose = list(zip(*matrix))
print(transpose)

10. Write a program that zips two lists of different lengths and uses strict=True (Python 3.10+).

a = [1, 2]
b = [3, 4, 5]
try:
    print(list(zip(a, b, strict=True)))
except ValueError as e:
    print(e)
