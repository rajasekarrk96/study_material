Python Strings

Introduction

Strings in Python are surrounded by either single quotation marks, or double quotation marks.

"hello" is the same as 'hello'.

Multiline Strings

You can assign a multiline string to a variable by using three quotes:

a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)

Slicing Strings

You can return a range of characters by using the slice syntax.

Specify the start index and the end index, separated by a colon, to return a part of the string.

Syntax:

string_variable[start_index : end_index]

Example:

b = "Hello, World!"
print(b[2:5])   # llo
print(b[:5])    # Hello (from start)
print(b[2:])    # llo, World! (to end)

Negative Indexing

Use negative indexes to start the slice from the end of the string:

b = "Hello, World!"
print(b[-5:-2]) # orl

Modifying Strings

Python has a set of built-in methods that you can use on strings.

Upper Case

a = "Hello, World!"
print(a.upper())  # HELLO, WORLD!

Lower Case

a = "Hello, World!"
print(a.lower())  # hello, world!

Remove Whitespace

strip() removes whitespace from the beginning or the end:

a = " Hello, World! "
print(a.strip()) # "Hello, World!"

Replace String

a = "Hello, World!"
print(a.replace("H", "J")) # Jello, World!

Split String

split() returns a list where the text between the specified separator becomes the list items:

a = "Hello, World!"
print(a.split(",")) # ['Hello', ' World!']

String Concatenation

To concatenate, or combine, two strings you can use the + operator.

a = "Hello"
b = "World"
c = a + " " + b
print(c) # Hello World

String Formatting (f-strings)

To insert numbers into strings, use f-strings (available in Python 3.6+).

Put an f in front of the string, and add curly brackets {} containing variables:

age = 36
txt = f"My name is John, I am {age}"
print(txt)

Escape Characters

To insert characters that are illegal in a string, use an escape character \.

txt = "We are the so-called \"Vikings\" from the north."
  • \' Single Quote
  • \\ Backslash
  • \n New Line
  • \t Tab

String Methods

Method Description
capitalize() Converts the first character to upper case
count() Returns the number of times a specified value occurs
find() Searches the string for a specified value and returns the position
join() Joins the elements of an iterable to the end of the string

Worksheet

1. Write a program to find the length of a string without using len().

s = input("Enter a string: ")
count = 0
for char in s:
    count += 1
print("Length:", count)

2. Write a program to count the frequency of a specific character in a string.

s = input("Enter a string: ")
char = input("Enter character to count: ")
print("Count:", s.count(char))

3. Write a program to get a string made of the first 2 and the last 2 chars from a given a string.

s = input("Enter a string: ")
if len(s) < 2:
    print("Empty String")
else:
    new_str = s[:2] + s[-2:]
    print("Result:", new_str)

4. Write a program to check if a string is a palindrome.

s = input("Enter a string: ")
if s == s[::-1]:
    print("Palindrome")
else:
    print("Not a Palindrome")

5. Write a program to count the number of vowels in a string.

s = input("Enter a string: ")
vowels = "aeiouAEIOU"
count = 0
for char in s:
    if char in vowels:
        count += 1
print("Vowels:", count)

6. Write a program to remove duplicate characters from a string (preserving order).

s = input("Enter a string: ")
result = ""
for char in s:
    if char not in result:
        result += char
print("String without duplicates:", result)

7. Write a program to replace all spaces in a string with hyphens.

s = input("Enter a string: ")
print("Result:", s.replace(" ", "-"))

8. Write a program to count total number of words in a string.

s = input("Enter a string: ")
words = s.split()
print("Total words:", len(words))

9. Write a program to check if two strings are anagrams.

s1 = input("Enter first string: ")
s2 = input("Enter second string: ")
if sorted(s1) == sorted(s2):
    print("Anagrams")
else:
    print("Not Anagrams")

10. Write a program to check if a string contains only digits.

s = input("Enter a string: ")
if s.isdigit():
    print("Contains only digits")
else:
    print("Contains non-digits")
