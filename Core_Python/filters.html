Python Filters

Introduction

The filter() function extracts elements from an iterable (lists, tuples etc.) for which a function returns True.

It constructs an iterator from those elements of the iterable for which the function returns true.

Syntax

filter(function, iterable)

Example:

filter(function, iterable)
  • function: A function that tests if each element of a sequence true or not.
  • iterable: The iterable which is to be filtered, could be sets, lists, tuples, or containers of any iterators.

Filtering with a Function

In this example, we filter out vowels from a list of alphabets:

def fun(variable):
    letters = ['a', 'e', 'i', 'o', 'u']
    if (variable in letters):
        return True
    else:
        return False

sequence = ['g', 'e', 'e', 'j', 'k', 's', 'p', 'r']
filtered = filter(fun, sequence)

print('The filtered letters are:')
for s in filtered:
    print(s)

Filtering with Lambda

You can use lambda functions to make the code shorter:

seq = [0, 1, 2, 3, 5, 8, 13]

# result contains odd numbers of the list
result = filter(lambda x: x % 2 != 0, seq)
print(list(result))

# result contains even numbers of the list
result = filter(lambda x: x % 2 == 0, seq)
print(list(result))

Filtering with None

If the function argument is None, the identity function is assumed, meaning all elements that are false (like 0, False, None) are removed.

seq = [0, 1, 2, 3, False, 5, True, 8, 13]

result = filter(None, seq)

print(list(result))
# Output: [1, 2, 3, 5, True, 8, 13]

Worksheet

1. Write a program to filter positive numbers from a list.

numbers = [-10, 5, 12, -2, 0, 9]

positive_nums = list(filter(lambda x: x > 0, numbers))
print(positive_nums)

2. Write a program to filter even numbers from a list of 1 to 20.

numbers = range(1, 21)

evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)

3. Write a program to filter names that start with 'A'.

names = ["Andy", "Bob", "Alice", "Charlie", "Alex"]

starts_with_A = list(filter(lambda name: name.startswith('A'), names))
print(starts_with_A)

4. Write a program to remove empty strings from a list of strings.

str_list = ["Emma", "", "Jon", "", "Kelly"]

# Using None to filter falsy values (empty strings are falsy)
cleaned_list = list(filter(None, str_list))
print(cleaned_list)

5. Write a program to filter year values that are leap years.

years = [2000, 2001, 2004, 2100, 2020, 2025]

def is_leap(year):
    return (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

leap_years = list(filter(is_leap, years))
print(leap_years)

6. Write a program calculate the square of odd numbers using map and filter.

numbers = [1, 2, 3, 4, 5, 6]

# First filter odds, then map to square
odds = filter(lambda x: x % 2 != 0, numbers)
squared_odds = list(map(lambda x: x**2, odds))

print(squared_odds)

7. Write a program to filter words that are palindromes.

words = ["level", "world", "madam", "python", "racecar"]

palindromes = list(filter(lambda w: w == w[::-1], words))
print(palindromes)

8. Write a program to filter numbers divisible by both 3 and 5.

nums = [15, 30, 4, 9, 10, 45, 60]

divisible = list(filter(lambda x: x % 15 == 0, nums))
print(divisible)

9. Write a program to filter people older than 18 from a dictionary list.

people = [
    {"name": "John", "age": 25},
    {"name": "Jane", "age": 16},
    {"name": "Dave", "age": 30}
]

adults = list(filter(lambda p: p["age"] > 18, people))
print(adults)

10. Write a program to filter strings with length greater than 5.

words = ["apple", "banana", "kiwi", "watermelon", "orange"]

long_words = list(filter(lambda x: len(x) > 5, words))
print(long_words)
