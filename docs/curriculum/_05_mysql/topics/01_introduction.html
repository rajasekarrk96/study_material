Introduction

SQL (Structured Query Language) is a standard language used to store, retrieve, manage, and manipulate data in relational databases.

DBMS & RDBMS

🔹 DBMS (Database Management System)
A system software used to store, manage, and retrieve data efficiently.

🔹 RDBMS (Relational Database Management System)
A DBMS where data is stored in tables (rows & columns) and relationships are created using keys.

DDL - Data Definition Language

Used to define and manage database structure

🔹 CREATE – Creates database objects like tables, views, indexes.
🔹 ALTER – Modifies the structure of an existing table.
🔹 DROP – Deletes database objects permanently.
🔹 TRUNCATE – Removes all records from a table but keeps its structure.
🔹 RENAME – Changes the name of a database object.

DML - Data Manipulation Language

Used to manipulate data inside tables

🔹 INSERT – Adds new records into a table.
🔹 UPDATE – Modifies existing records in a table.
🔹 DELETE – Removes records from a table.

DRL / DQL - Data Retrieval / Query Language

Used to retrieve data from database

🔹 SELECT – Fetches required data from one or more tables.

TCL - Transaction Control Language

Used to control transactions

🔹 COMMIT – Permanently saves the changes.
🔹 ROLLBACK – Reverts changes made during a transaction.
🔹 SAVEPOINT – Creates a point within a transaction to rollback partially.

DCL - Data Control Language

Used to control access and permissions

🔹 GRANT – Provides user permissions on database objects.
🔹 REVOKE – Removes previously granted permissions.

Advanced Database Objects

🔹 Stored Procedure
A group of SQL statements stored in the database that performs a specific task.

🔹 Stored Function
A stored program that returns a single value.

🔹 Trigger
A database object that automatically executes when a specific event occurs on a table.
