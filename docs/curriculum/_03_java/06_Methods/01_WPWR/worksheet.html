Question 1: Sum of Two Numbers

Write a program with a method named getTotal that accepts two integers as an argument and return its sum. Call this method from main() and print the results.
import java.util.Scanner; class _01_ { static int getTotal(int a, int b) { return a + b; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter two numbers: "); int x = sn.nextInt(); int y = sn.nextInt(); System.out.println("Sum = " + getTotal(x, y)); sn.close(); } }
Enter two numbers: 10 20 Sum = 30

Question 2: Check Even or Odd

Write a method named isEven that accepts an int argument. The method should return true if the argument is even, or false otherwise. Also, write a program to test your method.
import java.util.Scanner; class _02_ { static boolean isEven(int n) { return n % 2 == 0; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); System.out.println("Is Even: " + isEven(n)); sn.close(); } }
Enter a number: 4 Is Even: true

Question 3: Base Raised to Exponent

Write a program with a method named power that accepts two integers, base and exponent, and returns the result of raising the base to the exponent. Call this method from the main() and print the result.
import java.util.Scanner; class _03_ { static int power(int base, int exp) { int result = 1; for (int i = 1; i <= exp; i++) result *= base; return result; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter base: "); int b = sn.nextInt(); System.out.print("Enter exponent: "); int e = sn.nextInt(); System.out.println("Result: " + power(b, e)); sn.close(); } }
Enter base: 2 Enter exponent: 3 Result: 8

Question 4: Check Prime Number

Write a method named isPrime that checks if the given number is prime. A prime number is a number that is evenly divisible only by itself and 1.
import java.util.Scanner; class _04_ { static boolean isPrime(int n) { if (n <= 1) return false; for (int i = 2; i <= n / 2; i++) if (n % i == 0) return false; return true; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); System.out.println("Is Prime: " + isPrime(n)); sn.close(); } }
Enter a number: 7 Is Prime: true

Question 5: Palindrome Number Check

Write a method that takes as input a nonnegative integer and returns true if the number is a palindrome; otherwise, it returns false.
import java.util.Scanner; class _05_ { static boolean isPalindrome(int n) { int rev = 0, temp = n; while (temp > 0) { rev = rev * 10 + temp % 10; temp /= 10; } return rev == n; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); System.out.println("Is Palindrome: " + isPalindrome(n)); sn.close(); } }
Enter a number: 121 Is Palindrome: true
