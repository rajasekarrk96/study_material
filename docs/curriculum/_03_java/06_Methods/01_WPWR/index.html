Methods With Parameters With Return (WPWR)

Section 1: Method Fundamentals

A method is a reusable block of code that performs a specific task. Methods help organize code, reduce repetition, and make programs easier to maintain. A method with parameters accepts input values, and with a return type, it sends output back to the caller.

Benefits of Methods

  • Code reusability - write once, use many times
  • Better organization and readability
  • Easier debugging and maintenance
  • Modularity - divide code into logical pieces

💡 WPWR Definition

WPWR = With Parameters With Return
This means the method accepts input parameters and returns a value to the caller.

🎯 Real-World Example

A calculator function: you input two numbers (parameters) and get the sum back (return value).

Section 2: Method Syntax and Structure

Every method has a specific structure with access modifiers, return type, name, and parameters:

Basic Syntax

accessModifier returnType methodName(paramType param1, paramType param2, ...) { // Method body // Perform operations return value; // Return result }

Components Breakdown

Component Description Example
accessModifier Visibility of method (public/private) public
returnType Type of value the method returns int, double, String
methodName Identifier for the method calculateSum
parameters Input values the method accepts int a, int b

Complete Example

public class Calculator { // Method with parameters and return type public int add(int num1, int num2) { int result = num1 + num2; return result; // Return the result } public static void main(String[] args) { Calculator calc = new Calculator(); int sum = calc.add(10, 20); System.out.println("Sum: " + sum); // Output: Sum: 30 } }

📝 Method Naming Convention

Use camelCase for method names (start lowercase, each word after starts uppercase): calculateTotal, getUserAge, etc.

Section 3: Parameters and Arguments

Parameters are variables defined in the method signature, while arguments are actual values passed when calling the method:

Parameter Types

// Method definition with parameters public void printDetails(String name, int age, double salary) { System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("Salary: $" + salary); } // Method call with arguments printDetails("Alice", 30, 50000.0); // "Alice" is argument for name parameter

Example 1: Multiple Parameters

public class EmployeeInfo { public void displayEmployee(String empName, int empID, String dept) { System.out.println("Employee: " + empName); System.out.println("ID: " + empID); System.out.println("Department: " + dept); } public static void main(String[] args) { EmployeeInfo emp = new EmployeeInfo(); emp.displayEmployee("John Doe", 101, "IT"); } }

Example 2: Parameter Type Conversion

public class TypeConversion { public void processData(int intValue, double doubleValue, String strValue) { System.out.println("Integer: " + intValue); System.out.println("Double: " + doubleValue); System.out.println("String: " + strValue); } public static void main(String[] args) { TypeConversion tc = new TypeConversion(); tc.processData(42, 3.14, "Hello"); } }

Example 3: Array as Parameter

public class ArrayParameter { public void printArray(int[] numbers) { for (int num : numbers) { System.out.print(num + " "); } System.out.println(); } public static void main(String[] args) { ArrayParameter ap = new ArrayParameter(); int[] data = {10, 20, 30, 40, 50}; ap.printArray(data); // Output: 10 20 30 40 50 } }

⚠️ Parameter Rules

  • Parameters must have both type and name
  • Multiple parameters are separated by commas
  • Order of arguments must match order of parameters
  • Type of arguments must match parameter types

Section 4: Return Values

The return statement sends a value back from the method to the caller:

Return Types

Return Type Description Example
void Method doesn't return anything public void display() { }
int Returns an integer value public int getAge() { return 25; }
double Returns a decimal value public double getPrice() { return 19.99; }
String Returns text public String getName() { return "John"; }
boolean Returns true or false public boolean isValid() { return true; }

Example 1: Returning Simple Value

public class MathOperations { public int multiply(int a, int b) { return a * b; } public double divide(double a, double b) { return a / b; } public static void main(String[] args) { MathOperations math = new MathOperations(); System.out.println("5 × 4 = " + math.multiply(5, 4)); // Output: 20 System.out.println("10 ÷ 2 = " + math.divide(10, 2)); // Output: 5.0 } }

Example 2: Returning String

public class StringReturn { public String getGrade(int marks) { if (marks >= 90) return "A+"; else if (marks >= 80) return "A"; else if (marks >= 70) return "B"; else return "C"; } public static void main(String[] args) { StringReturn sr = new StringReturn(); System.out.println("Grade: " + sr.getGrade(85)); // Output: Grade: A } }

Example 3: Returning Array

public class ArrayReturn { public int[] getEvenNumbers(int n) { int[] evens = new int[n]; int count = 0; for (int i = 2; count < n; i += 2) { evens[count] = i; count++; } return evens; // Return the array } public static void main(String[] args) { ArrayReturn ar = new ArrayReturn(); int[] result = ar.getEvenNumbers(5); for (int num : result) { System.out.print(num + " "); // Output: 2 4 6 8 10 } } }

🔍 Return Statement Rules

  • Must match the declared return type
  • Can only have one return statement that executes
  • Ends method execution immediately
  • void methods don't have return values

Section 5: Complete Method Examples

Real-world examples combining parameters and return values:

Example 1: Temperature Converter

public class TemperatureConverter { public double celsiusToFahrenheit(double celsius) { return (celsius * 9/5) + 32; } public double fahrenheitToCelsius(double fahrenheit) { return (fahrenheit - 32) * 5/9; } public static void main(String[] args) { TemperatureConverter tc = new TemperatureConverter(); System.out.println("0°C = " + tc.celsiusToFahrenheit(0) + "°F"); System.out.println("32°F = " + tc.fahrenheitToCelsius(32) + "°C"); } }

Example 2: String Manipulation

public class StringUtilities { public String reverseString(String str) { return new StringBuilder(str).reverse().toString(); } public int countVowels(String str) { int count = 0; for (char c : str.toLowerCase().toCharArray()) { if ("aeiou".indexOf(c) >= 0) count++; } return count; } public static void main(String[] args) { StringUtilities su = new StringUtilities(); System.out.println("Reversed: " + su.reverseString("Hello")); System.out.println("Vowels in 'Hello': " + su.countVowels("Hello")); } }

Example 3: Statistical Calculations

public class Statistics { public double findAverage(int[] numbers) { int sum = 0; for (int num : numbers) sum += num; return (double) sum / numbers.length; } public int findMax(int[] numbers) { int max = numbers[0]; for (int num : numbers) { if (num > max) max = num; } return max; } public static void main(String[] args) { Statistics stats = new Statistics(); int[] data = {45, 78, 92, 56, 88}; System.out.println("Average: " + stats.findAverage(data)); System.out.println("Maximum: " + stats.findMax(data)); } }

Example 4: Payment Calculator

public class PaymentCalculator { public double calculateNetSalary(double basicSalary, double taxRate) { double tax = basicSalary * taxRate / 100; return basicSalary - tax; } public double calculateBonus(double basicSalary, int yearsOfService) { if (yearsOfService >= 5) { return basicSalary * 0.20; // 20% bonus } else if (yearsOfService >= 2) { return basicSalary * 0.10; // 10% bonus } return 0; // No bonus } public static void main(String[] args) { PaymentCalculator pc = new PaymentCalculator(); double salary = 50000; System.out.println("Net Salary: $" + pc.calculateNetSalary(salary, 15)); System.out.println("Bonus: $" + pc.calculateBonus(salary, 6)); } }

💡 Best Practices

  • Methods should do one thing well
  • Use descriptive names
  • Keep methods short and focused
  • Document complex methods with comments
