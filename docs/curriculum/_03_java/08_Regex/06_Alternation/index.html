Regex Alternation

Regex Alternation

Match one pattern OR another using the | operator.

What is Alternation?

Alternation allows matching one of several patterns using the | (OR) operator.

// Matches "cat" OR "dog"
Pattern p = Pattern.compile("cat|dog");

Example: Matching Multiple Options

import java.util.regex.*;

public class AlternationDemo {
    public static void main(String[] args) {
        // Match Java, Python, or JavaScript
        String pattern = "Java|Python|JavaScript";
        
        String[] languages = {
            "I love Java",
            "Python is great",
            "C++ is fast",
            "JavaScript rocks"
        };
        
        Pattern p = Pattern.compile(pattern);
        for (String lang : languages) {
            Matcher m = p.matcher(lang);
            if (m.find()) {
                System.out.println(lang + " -> Found: " + m.group());
            } else {
                System.out.println(lang + " -> Not found");
            }
        }
    }
}
I love Java -> Found: Java
Python is great -> Found: Python
C++ is fast -> Not found
JavaScript rocks -> Found: JavaScript
