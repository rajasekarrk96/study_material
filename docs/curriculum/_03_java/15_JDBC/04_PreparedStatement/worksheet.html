Question 1: Hotel Reservation System - Room Booking

Problem Statement

Design a hotel reservation system using JDBC with PreparedStatements. This example demonstrates room booking functionality. A complete hotel reservation system would include:
  • Room booking and reservation management
  • Room availability checking (real-time)
  • Guest management (check-in, check-out)
  • Billing and invoice generation
  • Reporting (occupancy rates, revenue)
  • Room service and amenities tracking
This example focuses on the room booking operation using PreparedStatement for secure data insertion.

Database Setup: Create a database named hoteldb with a booking table containing columns: booking_id, room_number, and status.

Solution

import java.sql.*; class _05_ { public static void main(String[] args) { try { Connection con = DriverManager.getConnection( "jdbc:mysql://localhost:3306/hoteldb", "root", "password"); PreparedStatement ps = con.prepareStatement("INSERT INTO booking VALUES (?, ?, ?)"); ps.setInt(1, 501); ps.setString(2, "Room-101"); ps.setString(3, "Booked"); ps.executeUpdate(); System.out.println("Room Booked Successfully"); con.close(); } catch (Exception e) { e.printStackTrace(); } } }

Sample Output

Room Booked Successfully

Why PreparedStatement?

Advantages of PreparedStatement:
  • SQL Injection Prevention: Parameterized queries prevent malicious SQL code injection
  • Performance: Pre-compiled SQL statements are faster for repeated executions
  • Type Safety: Methods like setInt(), setString() ensure correct data types
  • Readability: Cleaner code with placeholders (?) instead of string concatenation
  • Automatic Escaping: Special characters are automatically escaped

PreparedStatement vs Statement

// ❌ BAD: Using Statement (vulnerable to SQL injection) Statement stmt = con.createStatement(); String sql = "INSERT INTO booking VALUES (" + id + ", '" + room + "', '" + status + "')"; stmt.executeUpdate(sql); // ✅ GOOD: Using PreparedStatement (secure) PreparedStatement ps = con.prepareStatement("INSERT INTO booking VALUES (?, ?, ?)"); ps.setInt(1, id); ps.setString(2, room); ps.setString(3, status); ps.executeUpdate();

Common PreparedStatement Methods

Setter Methods:
  • setInt(index, value) - Set integer parameter
  • setString(index, value) - Set string parameter
  • setDouble(index, value) - Set double parameter
  • setDate(index, value) - Set date parameter
  • setBoolean(index, value) - Set boolean parameter
  • setNull(index, sqlType) - Set NULL value
Execution Methods:
  • executeQuery() - For SELECT queries (returns ResultSet)
  • executeUpdate() - For INSERT, UPDATE, DELETE (returns int)
  • execute() - For any SQL statement (returns boolean)
