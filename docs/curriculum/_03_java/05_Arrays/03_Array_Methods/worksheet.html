Practice Questions: java.util.Arrays

Master the built-in utility methods of the Arrays class for sorting, searching, copying, and comparing arrays.

Question 1: Sorting and Binary Search

Write a Java program to sort an array and then find the index of a specific element using binary search.
import java.util.Arrays; class _M1_ { public static void main(String[] args) { int[] arr = {45, 12, 89, 23, 67, 34}; System.out.println("Original: " + Arrays.toString(arr)); // Sorting the array Arrays.sort(arr); System.out.println("Sorted: " + Arrays.toString(arr)); // Binary search for value 67 int index = Arrays.binarySearch(arr, 67); System.out.println("Index of 67: " + index); } }
Original: [45, 12, 89, 23, 67, 34] Sorted: [12, 23, 34, 45, 67, 89] Index of 67: 4

Question 2: Comparing Two Arrays

Write a Java program to check if two arrays are equal using the Arrays.equals() method.
import java.util.Arrays; class _M2_ { public static void main(String[] args) { int[] arr1 = {1, 2, 3, 4}; int[] arr2 = {1, 2, 3, 4}; int[] arr3 = {1, 2, 4, 3}; System.out.println("arr1 equals arr2: " + Arrays.equals(arr1, arr2)); System.out.println("arr1 equals arr3: " + Arrays.equals(arr1, arr3)); } }
arr1 equals arr2: true arr1 equals arr3: false

Question 3: Filling an Array

Write a Java program to initialize all elements of an array with a specific value using Arrays.fill().
import java.util.Arrays; class _M3_ { public static void main(String[] args) { int[] arr = new int[5]; // Fill entire array with 7 Arrays.fill(arr, 7); System.out.println("After fill: " + Arrays.toString(arr)); // Fill range index 1 to 3 (exclusive) with 9 Arrays.fill(arr, 1, 3, 9); System.out.println("After partial fill: " + Arrays.toString(arr)); } }
After fill: [7, 7, 7, 7, 7] After partial fill: [7, 9, 9, 7, 7]

Question 4: Copying an Array

Write a Java program to copy an array into a new array of a different size using Arrays.copyOf().
import java.util.Arrays; class _M4_ { public static void main(String[] args) { int[] original = {10, 20, 30}; // Copy with same size int[] copy1 = Arrays.copyOf(original, original.length); // Copy and expand size (new elements are 0) int[] copy2 = Arrays.copyOf(original, 5); System.out.println("Original: " + Arrays.toString(original)); System.out.println("Copy1: " + Arrays.toString(copy1)); System.out.println("Copy2 (Expanded): " + Arrays.toString(copy2)); } }
Original: [10, 20, 30] Copy1: [10, 20, 30] Copy2 (Expanded): [10, 20, 30, 0, 0]

Question 5: Deep String for 2D Arrays

Write a Java program to print a 2D array conveniently using Arrays.deepToString().
import java.util.Arrays; class _M5_ { public static void main(String[] args) { int[][] matrix = { {1, 2, 3}, {4, 5, 6} }; // Standard toString() doesn't work well for 2D arrays System.out.println("Standard toString(): " + Arrays.toString(matrix)); // deepToString() is used for nested arrays System.out.println("Deep toString(): " + Arrays.deepToString(matrix)); } }
Standard toString(): [[I@...] Deep toString(): [[1, 2, 3], [4, 5, 6]]
