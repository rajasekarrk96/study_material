Do-While Loop in Java

Execute code at least once, then repeat based on a condition.

Syntax

do {
    // Code to execute
    // Runs at least once
} while (condition);
  • Code block executes first, then condition is checked
  • Guarantees at least one execution
  • Loop continues while condition is true
  • Note the semicolon after while(condition)

Example: Print Numbers 1 to 5

public class DoWhileExample {
    public static void main(String[] args) {
        int i = 1;
        
        do {
            System.out.println("Number: " + i);
            i++;
        } while (i <= 5);
        
        System.out.println("Loop ended");
        
        // Example showing at least one execution
        int x = 10;
        do {
            System.out.println("This runs once even though x >= 10");
        } while (x < 10);
    }
}

Do-While vs While Loop

Do-While Loop While Loop
Condition checked AFTER execution Condition checked BEFORE execution
Executes at least once May not execute at all
Exit-controlled loop Entry-controlled loop
💡 Use Case: Use do-while when you need the code to run at least once, like menu-driven programs or input validation.
