Question 1: Create ArrayList, Add Colors & Print

Problem Statement

Write a Java program to create a new array list, add some colors (string) and print out the collection.

Solution

import java.util.ArrayList; class _01_ { public static void main(String[] args) { ArrayList<String> colors = new ArrayList<>(); colors.add("Red"); colors.add("Green"); colors.add("Blue"); colors.add("Yellow"); System.out.println(colors); } }

Sample Output

[Red, Green, Blue, Yellow]
Key Concepts:
  • ArrayList Creation: Dynamic array that can grow/shrink
  • add(): Appends element to the end of the list
  • Generic Type: <String> ensures type safety
  • toString(): Automatically called when printing
Advantages: Dynamic size, easy to use, maintains insertion order

Question 2: Iterate Through All Elements

Problem Statement

Write a Java program to iterate through all elements in an array list.

Solution

import java.util.ArrayList; class _02_ { public static void main(String[] args) { ArrayList<String> colors = new ArrayList<>(); colors.add("Red"); colors.add("Green"); colors.add("Blue"); for (String c : colors) { System.out.println(c); } } }

Sample Output

Red Green Blue
Key Concepts:
  • For-Each Loop: Enhanced for loop for easy iteration
  • Read-Only: Cannot modify list during for-each iteration
  • Cleaner Syntax: More readable than traditional for loop
  • Alternatives: Can use for loop with index or Iterator
Best Practice: Use for-each when you don't need the index

Question 3: Insert Element at First Position

Problem Statement

Write a Java program to insert an element into the array list at the first position.

Solution

import java.util.ArrayList; class _03_ { public static void main(String[] args) { ArrayList<String> colors = new ArrayList<>(); colors.add("Green"); colors.add("Blue"); colors.add(0, "Red"); System.out.println(colors); } }

Sample Output

[Red, Green, Blue]
Key Concepts:
  • add(index, element): Inserts element at specified position
  • Index 0: First position in the list
  • Shifting: Existing elements shift to the right
  • Performance: O(n) operation due to element shifting
Note: For frequent insertions at beginning, consider LinkedList

Question 4: Update Specific Array Element

Problem Statement

Write a Java program to update specific array element by given element.

Solution

import java.util.ArrayList; class _04_ { public static void main(String[] args) { ArrayList<String> colors = new ArrayList<>(); colors.add("Red"); colors.add("Green"); colors.add("Blue"); colors.set(1, "Yellow"); System.out.println(colors); } }

Sample Output

[Red, Yellow, Blue]
Key Concepts:
  • set(index, element): Replaces element at specified index
  • Index 1: Second element (0-based indexing)
  • Returns Old Value: set() returns the replaced element
  • Performance: O(1) constant time operation
Validation: Ensure index is within bounds (0 to size-1)

Question 5: Remove Third Element

Problem Statement

Write a Java program to remove the third element from an array list.

Solution

import java.util.ArrayList; class _05_ { public static void main(String[] args) { ArrayList<String> colors = new ArrayList<>(); colors.add("Red"); colors.add("Green"); colors.add("Blue"); colors.add("Yellow"); colors.remove(2); System.out.println(colors); } }

Sample Output

[Red, Green, Yellow]
Key Concepts:
  • remove(index): Removes element at specified position
  • Index 2: Third element (0-based indexing)
  • Shifting: Elements after removed index shift left
  • Alternative: remove(Object) removes by value
Returns: The removed element, useful for verification
