HashSet

HashSet in Java

Store unique elements without duplicates using HashSet.

What is HashSet?

HashSet stores unique elements. It doesn't allow duplicates and doesn't maintain insertion order.

  • No duplicate elements
  • Unordered collection
  • Fast add, remove, contains operations
  • Allows one null element

Common Methods

Method Description
add(E) Add element
remove(E) Remove element
contains(E) Check if exists
size() Get size
clear() Remove all

Example: Unique Numbers

import java.util.HashSet;

public class HashSetDemo {
    public static void main(String[] args) {
        HashSet numbers = new HashSet<>();
        
        // Add elements
        numbers.add(10);
        numbers.add(20);
        numbers.add(10);  // Duplicate - won't be added
        numbers.add(30);
        
        System.out.println("Set: " + numbers);
        System.out.println("Size: " + numbers.size());
        
        // Check if contains
        System.out.println("Contains 20? " + numbers.contains(20));
        
        // Remove
        numbers.remove(20);
        System.out.println("After removal: " + numbers);
        
        // Iterate
        for (int num : numbers) {
            System.out.println(num);
        }
    }
}
Set: [10, 20, 30]
Size: 3
Contains 20? true
After removal: [10, 30]
