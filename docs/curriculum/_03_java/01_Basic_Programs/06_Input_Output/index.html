Section 1: System.out Methods

Overview

System.out is an object used to display output to the console. It provides three main methods for printing: print(), println(), and printf(). Each serves a different purpose in displaying information to the user.

Methods Table

Method Description Example Output
print() Prints without newline System.out.print("Hello"); Hello
println() Prints with newline System.out.println("Hello"); Hello
(moves to next line)
printf() Formatted printing System.out.printf("%d", 42); 42

Code Example

public class SystemOutMethods { public static void main(String[] args) { // print() - no newline System.out.print("Hello "); System.out.print("World"); System.out.println(); // Move to next line // println() - with newline System.out.println("Java Programming"); System.out.println("Easy to Learn"); // Printing different data types System.out.println(42); // Integer System.out.println(3.14); // Double System.out.println(true); // Boolean System.out.println('A'); // Character // String concatenation String name = "John"; int age = 25; System.out.println("Name: " + name + ", Age: " + age); // printf() examples System.out.printf("Number: %d\n", 42); System.out.printf("Float: %.2f\n", 3.14159); System.out.printf("String: %s\n", "Hello"); } }

Output

Hello World Java Programming Easy to Learn 42 3.14 true A Name: John, Age: 25 Number: 42 Float: 3.14 String: Hello
Note: Use println() for most cases as it adds a newline automatically. Use print() when you need output without a newline. Use printf() for formatted output with specific formatting requirements.

Section 2: Scanner Class for Input

Overview

The Scanner class is used to read input from the user. It's part of the java.util package and provides various methods to read different data types from the console (System.in). To use Scanner, you must import java.util.Scanner first.

Basic Setup

import java.util.Scanner; Scanner input = new Scanner(System.in);

Scanner Methods Table

Method Data Type Example
nextInt() Integer int num = input.nextInt();
nextDouble() Double double price = input.nextDouble();
nextFloat() Float float height = input.nextFloat();
nextLine() String (entire line) String text = input.nextLine();
next() String (single word) String word = input.next();
nextBoolean() Boolean boolean flag = input.nextBoolean();

Code Example

import java.util.Scanner; public class ScannerClassExample { public static void main(String[] args) { Scanner input = new Scanner(System.in); // Read a string System.out.print("Enter your name: "); String name = input.nextLine(); // Read an integer System.out.print("Enter your age: "); int age = input.nextInt(); // Read a double System.out.print("Enter your height (in meters): "); double height = input.nextDouble(); // Display results System.out.println("\n--- Your Information ---"); System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("Height: " + height + " meters"); // Close the scanner input.close(); } }

Sample Interaction

Enter your name: Alice Johnson Enter your age: 22 Enter your height (in meters): 1.65 --- Your Information --- Name: Alice Johnson Age: 22 Height: 1.65 meters
Note: Always close the Scanner when done to free up resources using input.close(). After using nextInt(), nextDouble(), etc., use nextLine() carefully as it may read leftover newline characters.

Section 3: Reading Different Data Types

Overview

The Scanner class provides specific methods for reading different data types. Understanding how to use each method correctly is crucial for building interactive programs that accept various types of user input.

Code Example

import java.util.Scanner; public class ReadingDataTypes { public static void main(String[] args) { Scanner input = new Scanner(System.in); // Reading Integer System.out.print("Enter an integer: "); int intValue = input.nextInt(); // Reading Double System.out.print("Enter a decimal number: "); double doubleValue = input.nextDouble(); // Reading Float System.out.print("Enter a float value: "); float floatValue = input.nextFloat(); // Important: Clear the newline from input buffer input.nextLine(); // Reading String (single word) System.out.print("Enter a word: "); String word = input.next(); // Clear the newline again input.nextLine(); // Reading entire line as String System.out.print("Enter a full sentence: "); String sentence = input.nextLine(); // Reading Boolean System.out.print("Enter true or false: "); boolean boolValue = input.nextBoolean(); // Display all inputs System.out.println("\n--- Collected Data ---"); System.out.println("Integer: " + intValue); System.out.println("Double: " + doubleValue); System.out.println("Float: " + floatValue); System.out.println("Word: " + word); System.out.println("Sentence: " + sentence); System.out.println("Boolean: " + boolValue); input.close(); } }

Sample Output

Enter an integer: 42 Enter a decimal number: 3.14 Enter a float value: 2.5 Enter a word: Java Enter a full sentence: Java is amazing Enter true or false: true --- Collected Data --- Integer: 42 Double: 3.14 Float: 2.5 Word: Java Sentence: Java is amazing Boolean: true
Example: When mixing nextInt(), nextDouble() with nextLine(), always use nextLine() after numeric input to clear the input buffer and avoid skipping lines.

Section 4: Formatted Output

Overview

The printf() method allows you to format output using format specifiers. This is useful for aligning output, controlling decimal places, and creating organized displays like tables and reports.

Common Format Specifiers

Specifier Description Example Output
%d Integer printf("%d", 42) 42
%f Floating-point (6 decimal places) printf("%f", 3.14) 3.140000
%.2f Floating-point (2 decimal places) printf("%.2f", 3.14159) 3.14
%s String printf("%s", "Hello") Hello
%c Character printf("%c", 'A') A
%n Newline printf("Line1%nLine2") Line1
Line2

Code Example

public class FormattedOutput { public static void main(String[] args) { // Basic formatting System.out.printf("Integer: %d\n", 42); System.out.printf("String: %s\n", "Java"); System.out.printf("Character: %c\n", 'J'); // Decimal formatting double price = 19.567; System.out.printf("Price: $%.2f\n", price); // Width and alignment System.out.printf("%10d\n", 42); // Right-aligned in 10 spaces System.out.printf("%-10s|\n", "Left"); // Left-aligned in 10 spaces // Multiple values String product = "Laptop"; int quantity = 5; double unitPrice = 999.99; System.out.printf("%s: Qty=%d, Price=$%.2f\n", product, quantity, unitPrice); // Table formatting System.out.println("\n--- Student Grades ---"); System.out.printf("%-15s %10s %10s\n", "Name", "Score", "Grade"); System.out.println("---------------------------------------------"); System.out.printf("%-15s %10d %10s\n", "Alice", 95, "A"); System.out.printf("%-15s %10d %10s\n", "Bob", 87, "B"); System.out.printf("%-15s %10d %10s\n", "Charlie", 92, "A"); } }

Output

Integer: 42 String: Java Character: J Price: $19.57 42 Left | Laptop: Qty=5, Price=$999.99 --- Student Grades --- Name Score Grade --------------------------------------------- Alice 95 A Bob 87 B Charlie 92 A
Note: Use %n instead of \n in printf() for better portability. The format specifier can include width and precision information, such as %10d for 10-space width or %.2f for 2 decimal places.

Section 5: Practical Examples

Example 1: Simple Calculator

import java.util.Scanner; public class SimpleCalculator { public static void main(String[] args) { Scanner input = new Scanner(System.in); System.out.println("=== Simple Calculator ==="); System.out.print("Enter first number: "); double num1 = input.nextDouble(); System.out.print("Enter second number: "); double num2 = input.nextDouble(); double sum = num1 + num2; double difference = num1 - num2; double product = num1 * num2; double quotient = num1 / num2; System.out.println("\n--- Results ---"); System.out.printf("%.2f + %.2f = %.2f\n", num1, num2, sum); System.out.printf("%.2f - %.2f = %.2f\n", num1, num2, difference); System.out.printf("%.2f * %.2f = %.2f\n", num1, num2, product); System.out.printf("%.2f / %.2f = %.2f\n", num1, num2, quotient); input.close(); } }

Example 1 Output

=== Simple Calculator === Enter first number: 50 Enter second number: 15 --- Results --- 50.00 + 15.00 = 65.00 50.00 - 15.00 = 35.00 50.00 * 15.00 = 750.00 50.00 / 15.00 = 3.33

Example 2: Student Information Form

import java.util.Scanner; public class StudentForm { public static void main(String[] args) { Scanner input = new Scanner(System.in); System.out.println("=== Student Information Form ==="); System.out.print("Enter student name: "); String name = input.nextLine(); System.out.print("Enter roll number: "); int rollNumber = input.nextInt(); System.out.print("Enter marks in Math: "); double math = input.nextDouble(); System.out.print("Enter marks in English: "); double english = input.nextDouble(); System.out.print("Enter marks in Science: "); double science = input.nextDouble(); double average = (math + english + science) / 3; System.out.println("\n=== Student Report ==="); System.out.printf("Name: %s\n", name); System.out.printf("Roll No: %d\n", rollNumber); System.out.printf("Math: %.2f\n", math); System.out.printf("English: %.2f\n", english); System.out.printf("Science: %.2f\n", science); System.out.printf("Average: %.2f\n", average); input.close(); } }

Example 2 Output

=== Student Information Form === Enter student name: Priya Sharma Enter roll number: 101 Enter marks in Math: 95 Enter marks in English: 87 Enter marks in Science: 91 === Student Report === Name: Priya Sharma Roll No: 101 Math: 95.00 English: 87.00 Science: 91.00 Average: 91.00
Best Practices: Always provide clear prompts to users about what input is expected. Use appropriate Scanner methods for each data type. Close the Scanner object after use to prevent resource leaks. Validate user input when possible.
