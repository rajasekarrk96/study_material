Question 1: Declare and Initialize Variables

Problem Statement

Write a Java program that declares and initializes three variables: an integer for age, a double for height, and a String for name. Print all three values.

Sample Input and Output

Sample Input 1:
25
5.9
John

Sample Output 1:
Name: John
Age: 25
Height: 5.9

Sample Input 2:
30
6.1
Sarah

Sample Output 2:
Name: Sarah
Age: 30
Height: 6.1
Show Solution

Solution (Java Program)

import java.util.Scanner; public class DeclareVariables { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter age: "); int age = sn.nextInt(); System.out.print("Enter height: "); double height = sn.nextDouble(); sn.nextLine(); // Consume newline System.out.print("Enter name: "); String name = sn.nextLine(); System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("Height: " + height); sn.close(); } }

Step-by-Step Explanation

  1. Variable Declaration: We declare three variables with different data types: int age, double height, and String name.
  2. Reading Integer: nextInt() reads an integer value for age.
  3. Reading Double: nextDouble() reads a decimal value for height.
  4. Consume Newline: After reading numeric values, we call nextLine() to consume the leftover newline character.
  5. Reading String: nextLine() reads the entire line for the name.
  6. Display Values: Print all three variables using concatenation.

Question 2: Swap Two Numbers (With Temporary Variable)

Problem Statement

Write a Java program to swap two numbers using a temporary variable.

Sample Input and Output

Sample Input 1:
10
20

Sample Output 1:
Before swap: a = 10, b = 20
After swap: a = 20, b = 10

Sample Input 2:
5
15

Sample Output 2:
Before swap: a = 5, b = 15
After swap: a = 15, b = 5
Show Solution

Solution (Java Program)

import java.util.Scanner; public class SwapWithTemp { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int a = sn.nextInt(); System.out.print("Enter second number: "); int b = sn.nextInt(); System.out.println("Before swap: a = " + a + ", b = " + b); // Swapping using temporary variable int temp = a; a = b; b = temp; System.out.println("After swap: a = " + a + ", b = " + b); sn.close(); } }

Step-by-Step Explanation

  1. Read Two Numbers: Get values for variables a and b from the user.
  2. Display Before: Show the original values before swapping.
  3. Create Temporary Variable: int temp = a; - Store the value of a in a temporary variable.
  4. Assign b to a: a = b; - Now a has the value of b.
  5. Assign temp to b: b = temp; - Now b has the original value of a (stored in temp).
  6. Display After: Show the swapped values.

Question 3: Swap Two Numbers (Without Temporary Variable)

Problem Statement

Write a Java program to swap two numbers without using a temporary variable (using arithmetic operations).

Sample Input and Output

Sample Input 1:
12
8

Sample Output 1:
Before swap: a = 12, b = 8
After swap: a = 8, b = 12

Sample Input 2:
100
50

Sample Output 2:
Before swap: a = 100, b = 50
After swap: a = 50, b = 100
Show Solution

Solution (Java Program)

import java.util.Scanner; public class SwapWithoutTemp { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int a = sn.nextInt(); System.out.print("Enter second number: "); int b = sn.nextInt(); System.out.println("Before swap: a = " + a + ", b = " + b); // Swapping without temporary variable a = a + b; // a now contains sum b = a - b; // b now contains original a a = a - b; // a now contains original b System.out.println("After swap: a = " + a + ", b = " + b); sn.close(); } }

Step-by-Step Explanation

  1. Read Values: Get two integer values from the user.
  2. First Operation: a = a + b; - Store the sum of both numbers in a.
  3. Second Operation: b = a - b; - Subtract b from the sum to get the original value of a, store it in b.
  4. Third Operation: a = a - b; - Subtract the new b (original a) from the sum to get the original b, store it in a.
  5. Result: The values are now swapped without using any extra variable.
  6. Note: This method can cause overflow for very large numbers.

Question 4: Variable Scope Demonstration

Problem Statement

Write a Java program that demonstrates local variable scope by calculating the sum and product of two numbers in separate code blocks.

Sample Input and Output

Sample Input 1:
6
4

Sample Output 1:
Sum: 10
Product: 24

Sample Input 2:
9
3

Sample Output 2:
Sum: 12
Product: 27
Show Solution

Solution (Java Program)

import java.util.Scanner; public class VariableScope { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int num1 = sn.nextInt(); System.out.print("Enter second number: "); int num2 = sn.nextInt(); // Local variable in this block { int sum = num1 + num2; System.out.println("Sum: " + sum); } // Different local variable in this block { int product = num1 * num2; System.out.println("Product: " + product); } sn.close(); } }

Step-by-Step Explanation

  1. Method-Level Variables: num1 and num2 are accessible throughout the main method.
  2. Block Scope: Variables declared inside { } are only accessible within that block.
  3. First Block: sum is declared and used only within the first set of braces.
  4. Second Block: product is declared in a separate block and is independent of the first block.
  5. Scope Limitation: You cannot access sum outside its block, and you cannot access product outside its block.
  6. Best Practice: Declare variables in the smallest scope necessary.

Question 5: Final Variables (Constants)

Problem Statement

Write a Java program that calculates the area of a circle using a final variable for PI. The radius should be taken as input.

Sample Input and Output

Sample Input 1:
7

Sample Output 1:
Area of circle: 153.93804002589985

Sample Input 2:
10

Sample Output 2:
Area of circle: 314.1592653589793
Show Solution

Solution (Java Program)

import java.util.Scanner; public class CircleArea { public static void main(String[] args) { Scanner sn = new Scanner(System.in); // Final variable (constant) final double PI = 3.14159265358979323846; System.out.print("Enter radius: "); double radius = sn.nextDouble(); double area = PI * radius * radius; System.out.println("Area of circle: " + area); sn.close(); } }

Step-by-Step Explanation

  1. Final Keyword: final double PI - The final keyword makes PI a constant that cannot be changed.
  2. Constant Naming: Constants are typically named in ALL_CAPS by convention.
  3. Initialization: Final variables must be initialized when declared.
  4. Read Radius: Get the radius value from the user.
  5. Calculate Area: Use the formula: Area = π × r²
  6. Immutability: Once assigned, the value of PI cannot be changed. Attempting to do so will cause a compilation error.
