21. Multi-Select Dropdowns

📖 Introduction

Most lists allow collecting one item at a time, but some interfaces require selecting multiple items simultaneously. This could be a standard multi-select listbox or a complex grid where you hold CTRL or SHIFT to choose multiple rows.

💡 What You'll Learn

  • Handling Standard Multi-select Dropdowns
  • Using isMultiple() to verify dropdown type
  • Deselecting options (only valid for multi-selects)
  • Simulating CTRL+Click for custom grids

🔍 Detailed Explanation

1. Standard Multi-Select (Select Class)

If an HTML <select> tag has the multiple attribute, you can select multiple options using the standard Select class.

  • select.isMultiple(): Returns true if multiple selection is allowed.
  • select.deselectAll(): Clears all selections.
  • select.selectBy...(): Can be called multiple times to add to the selection.

2. Custom Multi-Select (Actions Class)

For modern UI grids (div/span based) that emulate list behavior, you typically need to hold the Control (Ctrl) or Command key while clicking items.

💻 Code Examples

Key techniques for multi-selection:

Standard Listbox

Select select = new Select(driver.findElement(By.id("colors")));

if(select.isMultiple()) {
    // Select Red and Blue
    select.selectByVisibleText("Red");
    select.selectByVisibleText("Blue");
    
    // Verify 2 items are selected
    List<WebElement> selected = select.getAllSelectedOptions();
    System.out.println("Selected count: " + selected.size());
    
    // Deselect Blue
    select.deselectByVisibleText("Blue");
}

Custom Grid (CTRL+Click)

Actions actions = new Actions(driver);
WebElement item1 = driver.findElement(By.id("row-1"));
WebElement item2 = driver.findElement(By.id("row-3"));

// Hold CTRL, Click Item 1, Click Item 2, Release CTRL
actions.keyDown(Keys.CONTROL)
       .click(item1)
       .click(item2)
       .keyUp(Keys.CONTROL)
       .perform();

⚠️ Common Pitfalls

Deselect Exception: Trying to use deselectAll() or deselectBy...() on a standard single-select dropdown will throw an UnsupportedOperationException.

🎯 Real-World Scenario

Practice Scenario

Goal: Manage a Skills Profile.

  1. Check if the "Skills" list allows multiple selection.
  2. Select "Java" and "Python".
  3. Deselect "Java".
  4. Print all currently selected skills.

📝 Interview Questions

Q1: How do you select all options in a multi-select dropdown?

Answer: Iterate through select.getOptions() and call selectBy... for each one, or verify isMultiple() first.
