13. Dropdown Handling

📖 Introduction

Dropdowns are common UI elements used to select options from a list. In Selenium, handling them depends on how they are built in the HTML. Some are standard HTML <select> tags, while others are "fancy" dynamic dropdowns made with <div> and <span> tags (like in Bootstrap or React apps).

💡 What You'll Learn

  • How to handle standard Static Dropdowns using the Select class
  • Methods to select by Index, Value, and Visible Text
  • Strategies to handle Dynamic (Custom) Dropdowns
  • How to handle Multi-Select Dropdowns

🔍 Detailed Explanation

1. Static Dropdowns (The Select Class)

If the dropdown is created using the <select> tag, Selenium makes it easy. You cannot just click options; you must use the Select class.

  • selectByVisibleText("Text"): Selects the option that displays the text "Text". (Recommended)
  • selectByValue("val"): Selects the option with value="val".
  • selectByIndex(0): Selects the first option. (Risky if order changes).

2. Dynamic Dropdowns

These are not <select> tags. They are interactive lists. To handle them:

  1. Click the dropdown to expand it.
  2. Wait for options to be visible.
  3. Find the list of options (usually li tags) using specific locators.
  4. Iterate through the list and click the one you want.

💻 Code Examples

Static Dropdown Example

// 1. Find the element
WebElement staticDropdown = driver.findElement(By.id("country_select"));

// 2. Wrap it with Select class
Select dropdown = new Select(staticDropdown);

// 3. Select option
dropdown.selectByVisibleText("India");
System.out.println("Selected: " + dropdown.getFirstSelectedOption().getText());

Dynamic Dropdown Example

// 1. Click to open
driver.findElement(By.id("autosuggest")).sendKeys("ind");

// 2. Find all matching options
List<WebElement> options = driver.findElements(By.cssSelector("li.ui-menu-item a"));

// 3. Iterate and click
for(WebElement option : options) {
    if(option.getText().equalsIgnoreCase("India")) {
        option.click();
        break;
    }
}

⚠️ Common Pitfalls

UnexpectedTagNameException: This error occurs if you try to use the Select class on an element that is NOT a <select> tag. Always check the HTML source first!

🎯 Real-World Scenario

Practice Scenario

Goal: handle a Country selection form.

  1. Select "United States" from the static dropdown.
  2. Print all available options in the console.
  3. Type "Can" in the dynamic search box and select "Canada" from the suggestions.

📝 Interview Questions

Q1: How do you check if a dropdown allows multiple selections?

Answer: Use the select.isMultiple() method. It returns a boolean.

Q2: How to deselect an option?

Answer: You can use deselectBy...() methods, but this only works for Multi-Select dropdowns. You cannot deselect an option in a standard single-select dropdown.
