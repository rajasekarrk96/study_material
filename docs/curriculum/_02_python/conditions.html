Conditional Statements

Definition

Conditional statements allow a program to make decisions based on conditions.

They execute certain blocks of code only if the condition is True.

Python supports three main conditional structures:

1. if Statement

Used to execute a block only when the condition is True.

Syntax:

if condition:
    # code to execute if condition is True

Example:

age = 18

if age >= 18:
    print("Eligible to vote")

2. if–else Statement

Runs one block if the condition is True, otherwise another block.

Syntax:

if condition:
    # code if True
else:
    # code if False

Example:

num = 5

if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")

3. if–elif–else Statement

Matches multiple conditions sequentially.

Syntax:

if condition1:
    # code if condition1 is True
elif condition2:
    # code if condition2 is True
else:
    # code if conditions are False

Example:

marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")

4. Nested if (if inside another if)

Example:

age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")
    else:
        print("ID required")
else:
    print("Not allowed")

5. Short-hand if / Ternary Operator

One-line if:

if 10 > 5: print("10 is greater")

Ternary expression:

x = 10
result = "Even" if x % 2 == 0 else "Odd"
print(result)

Example Program

num = int(input("Enter a number: "))

if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")
