Loops in Python

Loop

A loop is a programming structure that allows you to repeat a set of instructions multiple times until a condition tells it to stop.

A loop works through four essential parts:

  1. Initialization (init) → You set the starting point of the loop.
  2. Condition → The loop checks whether it should continue or stop.
  3. Process → The instructions that run again and again inside the loop.
  4. Iteration → The step that updates a value so the loop can move forward

Python has two main types of loops:

  • while loop
  • for loop

2. Python For Loop

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

Syntax:

for item in sequence:
    # code to execute

Example:

while condition:
    # code to execute

1. Python While Loop

The while loop executes a set of statements as long as a condition is true.

Syntax:

while condition:
    # code to execute

Example:

n=1 #initialization
while n<=10:    #condition
    print(n)    #process
    n+=1 #n=n+1 #iteration

For Loop

A for loop repeats a block of code a specific number of times. It uses a counter variable that automatically starts at a given value, increases by a fixed step, and stops before the end value.

This is usually done using the range() function.

Examples:

1. for x in range(10):

for x in range(10):   # 10 → end value (loop runs 0 to 9)
    print(x)
  • Start = 0 (default)
  • End = 10–1 = 9
  • Step = 1 (default)

3. Nested Loops

A nested loop is a loop inside a loop.

The "inner loop" will be executed one time for each iteration of the "outer loop".

Syntax:

for element in outer_sequence:
    for element in inner_sequence:
        # code

Example:

for x in range(1, 5): # start, end-1
    print(x)
  • Start = 1
  • End = 5–1 = 4
  • Step = 1

This prints 1, 2, 3, 4.

3. for x in range(1, 10, 1):

for x in range(1, 10, 1): # start, end-1, iteration(step)
    print(x)
  • Start = 1
  • End = 10–1 = 9
  • Step = 1

This prints 1 to 9, increasing by 1 each time.

1. Write a python program to check if two numbers are twin primes.

def is_prime(n):
    if n <= 1: return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

x = int(input("Enter first number: "))
y = int(input("Enter second number: "))

if is_prime(x) and is_prime(y) and abs(x - y) == 2:
    print(x, "and", y, "are Twin Primes")
else:
    print(x, "and", y, "are NOT Twin Primes")

2. Write a python program to find the perfect square root of a number without using built-in functions.

num=int(input('Enter a number: '))
found = False
for x in range(1, num + 1):
    if x * x == num:
        print(x, "is the square root of", num)
        found = True
        break
    if x * x > num:
        break

if not found:

n = int(input('Enter a number: '))
if n <= 0:
    print(n, "is not a power of two")
else:
    temp = n
    power = 0
    while temp > 1:
        if temp % 2 != 0:
            break
        temp //= 2
        power += 1
    
    if temp == 1:
        print(n, "is 2 to the power of", power)

5. Write a Python program to calculate both GCD and LCM without using a separate function. 

a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

gcd = 1
for i in range(1, min(a, b) + 1):
    if a % i == 0 and b % i == 0:
        gcd = i

lcm = (a * b) // gcd

print("GCD of", a, "and", b, "is:", gcd)
print("LCM of", a, "and", b, "is:", lcm)
