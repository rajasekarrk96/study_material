Python Web Scraping

Introduction

Web scraping is the process of extracting data from websites. Python is one of the most popular languages for this task due to its powerful libraries.

Installation

We will use requests to fetch HTML and beautifulsoup4 to parse it.

pip install requests
pip install beautifulsoup4

Fetching a Page

First, we need to download the web page using the requests library.

Syntax:

requests.get(url)

Example:

import requests

url = "https://www.example.com"
response = requests.get(url)

print(response.status_code) # 200 means success
print(response.text[:100])  # Print first 100 characters of HTML

Parsing HTML

Once we have the HTML, we can parse it using BeautifulSoup.

from bs4 import BeautifulSoup

soup = BeautifulSoup(response.content, 'html.parser')

print(soup.title) # <title>Example Domain</title>
print(soup.title.text) # Example Domain

Finding Elements

You can search for elements by tag name, class, id, etc.

# Find the first paragraph
p = soup.find('p')
print(p.text)

# Find all links
links = soup.find_all('a')
for link in links:
    print(link.text)

Extracting Attributes

You can access tag attributes (like 'href' in links) like a dictionary.

links = soup.find_all('a')
for link in links:
    href = link.get('href')
    print(href)

import requests

soup = BeautifulSoup(r.content, "html.parser")
print(soup.title)

4. Write a program to find all 'h1' tags in a soup object.

h1_tags = soup.find_all("h1")
for h1 in h1_tags:
    print(h1.text)

5. Write a program to find an element by its ID 'main'.

element = soup.find(id="main")

6. Write a program to find all elements with class 'highlight'.

items = soup.find_all(class_="highlight")
for item in items:
    print(item.text)

7. Write a program to extract the 'href' attribute from all links.

for link in links:

8. Write a program to find the parent of an element.

9. Write a program to find the next sibling of an element.
