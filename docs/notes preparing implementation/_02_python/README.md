# Python Learning Platform

A modern, interactive platform for learning Python programming, featuring detailed explanations, examples, and practice exercises.

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Interactive Learning**: Code examples and practice exercises
- **Progress Tracking**: Track your learning progress
- **Clean UI/UX**: Modern and intuitive interface
- **Topics Covered**:
  - Introduction to Python
  - Python Basics
  - Conditional Statements
  - Loops
  - (More topics to be added)

## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Start learning Python!

## Project Structure

```
.
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── main.js         # JavaScript functionality
└── README.md           # This file
```

## Adding New Content

To add new topics or modify existing content, edit the `contentData` object in `js/main.js`.
