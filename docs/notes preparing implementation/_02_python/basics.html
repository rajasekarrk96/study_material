Python Basics

Variable

A variable in Python is a named storage location in memory that is used to store data or a value that can be referenced and manipulated in a program.

It acts like a container that holds data, and the value stored can be changed anytime.

In Python, you don’t need to declare the data type — the type is decided automatically based on the value assigned.

Syntax

variable_name = value

Basic Example

x = 10        # x stores an integer
name = "Ram"  # name stores a string
pi = 3.14     # pi stores a float

print(x, name, pi)

Variable Reassignment (you can change the value anytime)

a = 5
a = "Hello"
print(a)   # Output: Hello

Rules for Naming Variables

  • Must start with a letter or underscore
  • Cannot start with a number
  • Can contain letters, numbers, and underscore
  • Case-sensitive (age and Age are different)
valid_name = 10
_valid = 20
name1 = "Ragul"   # valid

1name = 5         # ❌ invalid
my-name = 10      # ❌ invalid

Multiple Variable Assignment

a, b, c = 1, 2, 3
print(a, b, c)

Assigning Same Value to Many Variables

x = y = z = 100
print(x, y, z)

Dynamic Typing Example

Python allows changing the type of a variable anytime:

data = 10      # int
data = "text"  # now string
print(data)

Data types

Data types in Python represent the kind of value stored in a variable.

They determine what operations can be done on that value (like addition, indexing, etc.).

Python has the following basic data types:

1. Integer (int)

Numbers without decimals.

x = 10
print(type(x))  # <class 'int'>

2. Float (float)

Numbers with decimals.

y = 3.14
print(type(y))  # <class 'float'>

3. String (str)

Text or characters inside quotes.

name = "Ragul"
print(type(name))  # <class 'str'>

4. Boolean (bool)

Represents True or False.

a = True
b = False
print(type(a))  # <class 'bool'>

5. List

Ordered, changeable collection of values.

numbers = [1, 2, 3, 4]
print(type(numbers))  # <class 'list'>

6. Tuple

Ordered but unchangeable collection.

point = (5, 10)
print(type(point))  # <class 'tuple'>

7. Set

Unordered collection of unique values.

unique_values = {1, 2, 3, 3}
print(unique_values)          # {1, 2, 3}
print(type(unique_values))    # <class 'set'>

8. Dictionary (dict)

Stores data in key–value pairs.

student = {"name": "Ragul", "age": 21}
print(type(student))  # <class 'dict'>

Type Casting

Type casting is the process of converting one data type into another in Python.

It helps when you need to change values from string → integer, float → integer, etc.

Python provides built-in functions for casting.

Types of Type Casting

1. Implicit Type Casting

Python automatically converts one type to another when it is safe to do so.

Example

a = 5       # int
b = 2.5     # float

c = a + b   # Python converts a to float
print(c)           # 7.5
print(type(c))     # <class 'float'>

Python changes a from int → float automatically.

2. Explicit Type Casting

You manually convert one data type to another using functions like:

  • int()
  • float()
  • str()
  • bool()
  • list(), tuple(), etc.

Examples

Convert string → int

x = "25"
y = int(x)
print(y, type(y))

Arithmetic Operators

Arithmetic operators are symbols used to perform mathematical operations on numbers in Python.

List of Arithmetic Operators

1. Addition (+)

Adds two values.

a = 10 + 5
print(a)   # 15

2. Subtraction (-)

Subtracts one value from another.

a = 10 - 3
print(a)   # 7

3. Multiplication (*)

Multiplies two values.

a = 6 * 4
print(a)   # 24

4. Division (/)

Always returns a float.

a = 10 / 4
print(a)   # 2.5

5. Floor Division (//)

Returns the integer part of division (rounds down).

a = 10 // 4
print(a)   # 2

6. Modulus (%)

Gives the remainder of division.

a = 10 % 4
print(a)   # 2

7. Exponentiation (**)

Raises a number to the power of another number.

a = 2 ** 3
print(a)   # 8

Example Using All Operators

x = 10
y = 3

print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.333...
print(x // y)  # 3
print(x % y)   # 1
print(x ** y)  # 1000

Comparison Operators

Comparison operators are used to compare two values.

They return a Boolean result → True or False.

List of Comparison Operators

1. Equal to (==)

Checks if two values are equal.

print(5 == 5)   # True
print(5 == 3)   # False

2. Not equal to (!=)

Checks if two values are not equal.

print(5 != 3)   # True
print(5 != 5)   # False

3. Greater than (>)

Checks if left value is greater than right.

print(10 > 5)   # True
print(3 > 8)    # False

4. Less than (<)

Checks if left value is less than right.

print(2 < 7)   # True
print(9 < 1)   # False

5. Greater than or equal to (>=)

Checks if left value is greater or equal.

print(5 >= 5)   # True
print(6 >= 10)  # False

6. Less than or equal to (<=)

Checks if left value is less or equal.

print(3 <= 5)   # True
print(7 <= 2)   # False

Example Combining Comparison Operators

a = 10
b = 20

print(a == b)   # False
print(a != b)   # True
print(a < b)    # True
print(a > b)    # False
print(a <= b)   # True
print(a >= b)   # False

Logical Operators

Logical operators are used to combine multiple conditions.

They always return a Boolean value → True or False.

Python has three logical operators:

1. and

Returns True only if both conditions are True.

Example:

a = 10
b = 5

print(a > 5 and b < 10)   # True and True → True
print(a > 5 and b > 10)   # True and False → False

2. or

Returns True if at least one condition is True.

Example:

a = 10
b = 20

print(a > 5 or b < 10)    # True or False → True
print(a < 5 or b < 10)    # False or False → False

3. not

Reverses the Boolean value.

Example:

x = True
y = False

print(not x)   # False
print(not y)   # True

Combined Example

age = 18
marks = 75

print(age >= 18 and marks > 60)   # True
print(age < 18 or marks > 60)     # True
print(not(age > 18))              # True (because age > 18 is False)

Bitwise Operators

Bitwise operators perform operations on numbers at the binary (bit) level.

They compare or manipulate each bit of the number.

Example:
10 in binary → 1010
4 in binary → 0100

List of Bitwise Operators

1. AND (&)

Sets each bit to 1 if both bits are 1.

Example:

a = 10   # 1010
b = 4    # 0100

print(a & b)   # 8  (1000)

2. OR (|)

Sets each bit to 1 if at least one bit is 1.

Example:

a = 10   # 1010
b = 4    # 0100

print(a | b)   # 14 (1110)

3. XOR (^)

Sets each bit to 1 if bits are different.

Example:

a = 10   # 1010
b = 4    # 0100

print(a ^ b)   # 14 (1110)

4. NOT (~)

Inverts all bits (changes 0 to 1 and 1 to 0).

For positive numbers, result becomes negative using 2’s complement.

Example:

a = 10
print(~a)      # -11

5. Left Shift (<<)

Shifts bits to the left (adds zeros on the right).

Same as multiplying by 2.

Example:

a = 5    # 0101
print(a << 1)   # 10 (1010)
print(a << 2)   # 20 (10100)

6. Right Shift (>>)

Shifts bits to the right (removes bits on the right).

Same as dividing by 2.

Example:

a = 20   # 10100
print(a >> 1)   # 10 (1010)
print(a >> 2)   # 5  (101)

Assignment Operators

Assignment operators are used to assign values to variables.

They can also update the existing value using arithmetic or bitwise operations.

Basic Assignment Operator

1. = (Simple assignment)

Assigns a value to a variable.

x = 10
y = "Hello"

Compound Assignment Operators

These operators update the variable by performing an operation and assignment in one step.

2. += (Add and assign)

x = 10
x += 5       # x = x + 5
print(x)     # 15

3. -= (Subtract and assign)

x = 20
x -= 8       # x = x - 8
print(x)     # 12

4. *= (Multiply and assign)

x = 6
x *= 3       # x = x * 3
print(x)     # 18

5. /= (Divide and assign)

x = 10
x /= 4       # x = x / 4
print(x)     # 2.5

6. %= (Modulus and assign)

x = 10
x %= 3       # x = x % 3
print(x)     # 1

7. //= (Floor divide and assign)

x = 10
x //= 3      # x = x // 3
print(x)     # 3

8. **= (Power and assign)

x = 2
x **= 4      # x = x ** 4
print(x)     # 16

Identity Operators

Identity operators are used to compare memory locations of two objects.

They check whether two variables refer to the same object in memory, not just equal values.

Python has two identity operators:

1. is

Returns True if both variables point to the same memory location.

Example:

a = [1, 2, 3]
b = a

print(a is b)      # True (same object)

2. is not

Returns True if both variables do NOT refer to the same object.

Example:

a = [1, 2, 3]
b = [1, 2, 3]

print(a is b)       # False (different objects)
print(a is not b)   # True

Membership Operators

Membership operators are used to check whether a value exists inside a sequence such as:

  • string
  • list
  • tuple
  • set
  • dictionary (checks keys)

They return a Boolean value → True or False.

Python has two membership operators:

1. in

Returns True if the value is present in the sequence.

2. not in

Returns True if the value is not present in the sequence.

Example:

fruits = ["apple", "banana", "mango"]

print("apple" in fruits)       # True
print("orange" not in fruits)  # True

ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding system that assigns a numeric value to letters, digits, symbols, and control characters.

It uses 7 bits to represent each character and supports 128 characters (0–127).

Examples:

  • A → 65
  • a → 97
  • 0 → 48
  • Space → 32

Why ASCII is used?

It allows computers to store, process, and transmit text using numbers.

ASCII in Python

1. Get ASCII value of a character → ord()

print(ord('A'))   # 65

2. Convert ASCII number to character → chr()

print(chr(65))    # A
