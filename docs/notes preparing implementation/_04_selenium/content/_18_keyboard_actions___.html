18. Keyboard Actions

📖 Introduction

While sendKeys() is great for typing text, sometimes you need to perform special keyboard operations like pressing Enter, Tab, or holding down modifier keys (Shift, Ctrl, Alt). Selenium provides the Keys enum and the Actions class for these tasks.

💡 What You'll Learn

  • How to press special keys (Enter, Esc, Tab, etc.)
  • How to simulate key chords (Ctrl+C, Ctrl+V)
  • How to hold down keys while typing (Shift for Uppercase)

🔍 Detailed Explanation

The Keys Enum

You can pass special keys directly to sendKeys:

element.sendKeys(Keys.ENTER);
element.sendKeys(Keys.TAB);

Complex Key Interactions

For holding keys down, use the Actions class:

Actions act = new Actions(driver);
act.keyDown(Keys.SHIFT).sendKeys("hello").keyUp(Keys.SHIFT).perform();

💻 Code Examples

Copy and Paste (Ctrl+A, Ctrl+C, Ctrl+V)

WebElement input1 = driver.findElement(By.id("input1"));
WebElement input2 = driver.findElement(By.id("input2"));

Actions act = new Actions(driver);

// Select All and Copy from Input 1
input1.sendKeys(Keys.CONTROL, "a");
input1.sendKeys(Keys.CONTROL, "c");

// Paste into Input 2
input2.sendKeys(Keys.CONTROL, "v");

⚠️ Common Pitfalls

OS Differences: On Mac, you usually use Keys.COMMAND instead of Keys.CONTROL. This can make tests flaky across operating systems.

🎯 Real-World Scenario

Practice Scenario

Goal: Fill a form without using the mouse.

  1. Focus on the "Name" field.
  2. Type a name in LOWERCASE but hold SHIFT to make it UPPERCASE.
  3. Press TAB to move to the next field.
  4. Press ENTER to submit the form.
