Question 1: Add Two Matrices

Write a Java program to add two matrices of the same size.
class _11_ { public static void main(String[] args) { int[][] a = {{1, 2}, {3, 4}}; int[][] b = {{5, 6}, {7, 8}}; int[][] sum = new int[2][2]; System.out.println("Resulting Matrix Sum:"); for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { sum[i][j] = a[i][j] + b[i][j]; System.out.print(sum[i][j] + " "); } System.out.println(); } } }
Resulting Matrix Sum: 6 8 10 12

Question 2: Matrix Multiplication

Write a Java program to multiply two 2x2 matrices.
class _12_ { public static void main(String[] args) { int[][] a = {{1, 2}, {3, 4}}; int[][] b = {{2, 0}, {1, 2}}; int[][] c = new int[2][2]; System.out.println("Matrix Multiplication Result:"); for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { for (int k = 0; k < 2; k++) { c[i][j] += a[i][k] * b[k][j]; } System.out.print(c[i][j] + " "); } System.out.println(); } } }
Matrix Multiplication Result: 4 4 10 8

Question 3: Transpose of a Matrix

Write a Java program to find the transpose of a 2x3 matrix.
class _13_ { public static void main(String[] args) { int[][] a = {{1, 2, 3}, {4, 5, 6}}; System.out.println("Transpose of the Matrix:"); for (int i = 0; i < 3; i++) { for (int j = 0; j < 2; j++) { System.out.print(a[j][i] + " "); } System.out.println(); } } }
Transpose of the Matrix: 1 4 2 5 3 6

Question 4: Sparse Matrix Check

Determine if a matrix is sparse (more than half of elements are zero).
class _14_ { public static void main(String[] args) { int[][] a = { {0, 0, 3}, {0, 0, 0}, {1, 0, 0} }; int count = 0; int size = a.length * a[0].length; for (int[] row : a) { for (int val : row) { if (val == 0) count++; } } if (count > size / 2) System.out.println("The matrix is a Sparse Matrix"); else System.out.println("The matrix is NOT a Sparse Matrix"); } }
The matrix is a Sparse Matrix

Question 5: Odd & Even Frequency

Write a Java program to find the frequency of odd and even numbers in a matrix.
class _15_ { public static void main(String[] args) { int[][] a = { {1, 2}, {3, 4}, {5, 6} }; int odd = 0, even = 0; for (int[] row : a) { for (int val : row) { if (val % 2 == 0) even++; else odd++; } } System.out.println("Frequency of Even numbers: " + even); System.out.println("Frequency of Odd numbers: " + odd); } }
Frequency of Even numbers: 3 Frequency of Odd numbers: 3
