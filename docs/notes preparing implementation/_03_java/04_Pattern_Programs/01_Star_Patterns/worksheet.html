Question 1: Right Triangle Star Pattern

Problem Statement

Write a program to print a right-angled triangle pattern using stars.

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*
**
***
****
*****
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class RightTriangle { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= i; j++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Nested loops are fundamental to patterns: Outer loop (i) controls rows, inner loop (j) controls columns. This row-column relationship is the foundation of all 2D pattern programming.
  2. Inner loop depends on outer: Inner loop runs from 1 to i, meaning row 1 has 1 star, row 2 has 2 stars, etc. This dependency creates the triangle shape.
  3. println() after inner loop: After printing all stars for a row, println() moves cursor to next line. Understanding when to use print() vs println() is crucial for pattern control.
  4. Practice makes perfect: This simple pattern teaches the template for all pattern problems. Master this, and you can adapt it to create any 2D shape by changing loop boundaries and print logic.

Question 2: Square Star Pattern

Problem Statement

Print a square pattern of stars with all sides equal.

Sample Input and Output

Sample Input 1:
4

Sample Output 1:
****
****
****
****
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class SquarePattern { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= n; j++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Independent loops create squares: Both loops run from 1 to n independently. This creates n rows with n stars each - a perfect square. Understanding when loops should depend on each other vs be independent is key.
  2. Simplest nested loop pattern: This is the baseline pattern. All other patterns build on this by adding conditions or changing loop boundaries. Always start simple before adding complexity.
  3. Constant iterations per row: Every row has exactly n stars because inner loop always runs n times regardless of outer loop value. Compare this to the triangle where inner loop depends on i.
  4. Building block for hollow patterns: To make this hollow, you'd add if conditions checking positions. The square structure remains, but logic inside determines what to print.

Question 3: Inverted Right Triangle

Problem Statement

Print an inverted right triangle starting with N stars and decreasing.

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*****
****
***
**
*
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class InvertedTriangle { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = n; i >= 1; i--) { for (int j = 1; j <= i; j++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Countdown in outer loop: Starting from n and decrementing (i--) reverses the pattern. First row has n stars, second has n-1, etc. Direction of counting matters for pattern orientation.
  2. Inner loop still depends on i: Even though i counts down, inner loop still runs 1 to i, so it naturally decreases. The dependency creates the inverse triangle automatically.
  3. Alternative approach exists: Could count i up (1 to n) and inner loop as (1 to n-i+1). Both work! Multiple solutions exist - choose what makes most logical sense.
  4. Pattern reversal technique: Reversing outer loop direction is a common technique to flip patterns. Apply this concept to any upward pattern to make it inverted.

Question 4: Hollow Square Pattern

Problem Statement

Print a hollow square with stars on borders only, spaces inside.

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*****
* *
* *
* *
*****
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class HollowSquare { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= n; j++) { if (i == 1 || i == n || j == 1 || j == n) { System.out.print("*"); } else { System.out.print(" "); } } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Condition-based printing: Same loop structure as filled square, but if condition controls what prints. Print star if on border (first/last row OR first/last column), else print space.
  2. Understanding OR logic: i==1 || i==n checks top/bottom rows. j==1 || j==n checks left/right columns. Combining these with OR catches all border positions.
  3. Hollow pattern template: This conditional approach works for hollowing any shape. Identify border conditions, print symbol there, space elsewhere. Applies to triangles, pyramids, diamonds.
  4. Mind the else clause: Printing space in else is crucial. Without it, inner positions print nothing (collapsed), breaking alignment. Spaces maintain structure even when invisible.

Question 5: Reverse Right Triangle

Problem Statement

Print a right triangle aligned to the right (spaces on left, stars on right).

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*
**
***
****
*****
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class ReverseTriangle { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= n - i; j++) { System.out.print(" "); } for (int k = 1; k <= i; k++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Two inner loops for two sections: First inner loop prints spaces (n-i spaces), second prints stars (i stars). Total per row is always n characters, maintaining alignment.
  2. Spaces first, stars second: Order matters! Spaces must print before stars to push stars to the right. Reversing this would create a left-aligned triangle.
  3. Complementary relationship: As i increases, spaces decrease (n-i) and stars increase (i). They always sum to n, keeping width constant. This balance is key to aligned patterns.
  4. Foundation for pyramids: This space+symbol pattern is core to centered pyramids. Pyramid is just this pattern but with stars on both sides of center. Master this to understand pyramid logic.
