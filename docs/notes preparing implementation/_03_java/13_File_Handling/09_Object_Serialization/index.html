Object Serialization

Serialization in Java

Convert objects to byte streams for storage or transmission.

What is Serialization?

Serialization converts an object into a byte stream. Deserialization recreates the object from bytes.

  • Implement Serializable interface
  • Save object state to file
  • Restore object from file

Serialization Example

import java.io.*;

class Student implements Serializable {
    String name;
    int age;
    
    public Student(String n, int a) {
        name = n;
        age = a;
    }
}

public class SerializeDemo {
    public static void main(String[] args) {
        try {
            Student s = new Student("Alice", 20);
            
            FileOutputStream file = new FileOutputStream("student.ser");
            ObjectOutputStream out = new ObjectOutputStream(file);
            out.writeObject(s);
            out.close();
            
            System.out.println("Object serialized");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Deserialization Example

import java.io.*;

public class DeserializeDemo {
    public static void main(String[] args) {
        try {
            FileInputStream file = new FileInputStream("student.ser");
            ObjectInputStream in = new ObjectInputStream(file);
            
            Student s = (Student) in.readObject();
            in.close();
            
            System.out.println("Name: " + s.name);
            System.out.println("Age: " + s.age);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
Name: Alice
Age: 20
