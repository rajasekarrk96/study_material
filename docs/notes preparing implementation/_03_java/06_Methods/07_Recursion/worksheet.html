Question 1: Factorial using Recursion

Write a Java method to find the factorial of a given number using recursion.
import java.util.Scanner; class _01_ { static int fact(int n) { if (n == 0) return 1; return n * fact(n - 1); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); System.out.println("Factorial of " + n + " is: " + fact(n)); sn.close(); } }
Enter a number: 5 Factorial of 5 is: 120

Question 2: Sum of Digits using Recursion

Write a recursive method that takes an integer and returns the sum of its digits. For example, 126 → 1 + 2 + 6 = 9.
import java.util.Scanner; class _02_ { static int sumOfDigits(int n) { if (n == 0) return 0; return (n % 10) + sumOfDigits(n / 10); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); System.out.println("Sum of digits: " + sumOfDigits(n)); sn.close(); } }
Enter a number: 1234 Sum of digits: 10

Question 3: Reverse a String using Recursion

Write a recursive method that returns the reverse of a given string.
import java.util.Scanner; class _03_ { static String reverse(String str) { if (str.isEmpty()) return str; return reverse(str.substring(1)) + str.charAt(0); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a string: "); String s = sn.next(); System.out.println("Reversed: " + reverse(s)); sn.close(); } }
Enter a string: Hello Reversed: olleH

Question 4: Nth Fibonacci Number

Write a recursive method to find the Nth term in the Fibonacci sequence (0, 1, 1, 2, 3, 5, 8, 13...).
import java.util.Scanner; class _04_ { static int fib(int n) { if (n <= 1) return n; return fib(n - 1) + fib(n - 2); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter n: "); int n = sn.nextInt(); System.out.println("Fibonacci term at " + n + " is: " + fib(n)); sn.close(); } }
Enter n: 6 Fibonacci term at 6 is: 8
