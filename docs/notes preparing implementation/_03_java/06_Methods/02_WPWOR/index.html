Methods With Parameters Without Return (WPWOR)

Methods With Parameters Without Return

Methods that accept input but don't return any value.

What is WPWOR?

WPWOR = With Parameters Without Return

These methods accept input parameters to work with but don't return any value to the caller. They use void as the return type.

  • Accept input through parameters
  • Perform actions or operations
  • Don't return any value (void)
  • Useful for displaying output, updating state, or performing actions
💡 Key Point: Use WPWOR when you need to process input but don't need to send a result back.

Syntax

public void methodName(dataType parameter1, dataType parameter2) {
    // Method body
    // Perform operations with parameters
    // No return statement needed
}
Example Structure: 
public void displayMessage(String name) {
    System.out.println("Hello, " + name);
}

Example 1: Display User Info

public class UserInfo {
    public void displayUser(String name, int age) {
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
    }
    
    public static void main(String[] args) {
        UserInfo user = new UserInfo();
        user.displayUser("Alice", 25);
    }
}
Name: Alice
Age: 25

Example 2: Print Pattern

public class PatternPrinter {
    public void printStars(int count) {
        for (int i = 0; i < count; i++) {
            System.out.print("* ");
        }
        System.out.println();
    }
    
    public static void main(String[] args) {
        PatternPrinter printer = new PatternPrinter();
        printer.printStars(5);
        printer.printStars(3);
    }
}

Example 3: Update Values

public class Calculator {
    public void addAndDisplay(int a, int b) {
        int sum = a + b;
        System.out.println(a + " + " + b + " = " + sum);
    }
    
    public void multiplyAndDisplay(int a, int b) {
        int product = a * b;
        System.out.println(a + " × " + b + " = " + product);
    }
    
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        calc.addAndDisplay(5, 3);
        calc.multiplyAndDisplay(4, 6);
    }
}
5 + 3 = 8
4 × 6 = 24

When to Use WPWOR

✓ Best Used For:
• Displaying information to console
• Updating object state
• Printing reports or patterns
• Logging messages
• Performing actions without needing a result

⚠️ Not Suitable For:
• Calculations that need to return results
• Data transformations
• Value retrieval operations
