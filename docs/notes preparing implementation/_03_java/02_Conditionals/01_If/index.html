If Statement

Section 1: What is an If Statement?

An if statement is a fundamental control structure in Java that allows you to execute code conditionally. It tests a Boolean condition and, if that condition evaluates to true, the code block inside the if statement is executed. If the condition is false, the code is skipped.

Key Concepts

  • Boolean Condition: A condition that evaluates to either true or false
  • Conditional Execution: Code that runs only when a specific condition is met
  • Decision Making: Allow your program to make choices based on data or user input

💡 Important Note

If statements are essential for creating dynamic programs that respond to different situations. Without conditional statements, programs would always execute the same sequence of operations.

🎯 Real-World Example

Think of an if statement like a security gate. If you have the correct pass (condition is true), you enter the building (code executes). If you don't have the pass (condition is false), you don't enter.

Section 2: If Statement Syntax

The basic structure of an if statement is simple and follows a consistent pattern:

Basic Syntax

if (condition) { // Code to execute if condition is true }

Components Breakdown

Component Description Example
if Keyword that starts the conditional statement if
condition Expression that evaluates to true or false age > 18
{} Braces that contain the code block { System.out.println("Adult"); }

Complete Example with Explanation

public class Main { public static void main(String[] args) { int temperature = 30; if (temperature > 25) { System.out.println("It's hot outside!"); } } }

📝 Remember

The condition must be enclosed in parentheses. The curly braces {} define the scope of the code that belongs to the if statement. Even a single statement should ideally be enclosed in braces for clarity.

Section 3: Single If Statement Examples

Let's look at practical examples of simple if statements that test different types of conditions:

Example 1: Age Verification

public class AgeCheck { public static void main(String[] args) { int age = 20; if (age >= 18) { System.out.println("You are an adult."); } // Output: You are an adult. } }

🔍 Explanation

The condition age >= 18 is evaluated. Since age is 20, which is greater than or equal to 18, the condition is true. The message "You are an adult." is printed.

Example 2: Grade Evaluation

public class GradeCheck { public static void main(String[] args) { int marks = 85; if (marks >= 80) { System.out.println("Grade: A"); System.out.println("Excellent performance!"); } // Output: Grade: A // Excellent performance! } }

🔍 Explanation

Here we have multiple statements inside the if block. All of them will execute if the condition is true. Since 85 >= 80, both print statements execute.

Example 3: String Equality Check

public class StringCheck { public static void main(String[] args) { String username = "admin"; if (username.equals("admin")) { System.out.println("Access granted!"); } // Output: Access granted! } }

🔍 Explanation

For strings, use the .equals() method to compare values. The condition checks if username equals "admin", which it does, so the message is printed.

Example 4: Number Range Check

public class RangeCheck { public static void main(String[] args) { int score = 50; if (score > 0 && score <= 100) { System.out.println("Valid score!"); } // Output: Valid score! } }

🔍 Explanation

This example uses the AND operator && to check multiple conditions. Both conditions must be true for the code to execute. Since 50 > 0 AND 50 <= 100, the message prints.

Section 4: Multiple Conditions

You can combine multiple conditions using logical operators to create more complex decision-making scenarios:

Logical Operators

Operator Symbol Meaning Example
AND && Both conditions must be true age > 18 && hasLicense
OR || At least one condition must be true day.equals("Saturday") || day.equals("Sunday")
NOT ! Reverses the boolean value !isRaining

Example 1: AND Operator

public class MultipleConditions { public static void main(String[] args) { int age = 25; boolean hasLicense = true; if (age >= 18 && hasLicense) { System.out.println("You can drive!"); } // Output: You can drive! } }

🔍 Explanation

Both conditions must be true: age must be >= 18 AND hasLicense must be true. Since both are satisfied, the message prints.

Example 2: OR Operator

public class OrExample { public static void main(String[] args) { int day = 6; // Saturday = 6, Sunday = 7 if (day == 6 || day == 7) { System.out.println("It's the weekend!"); } // Output: It's the weekend! } }

🔍 Explanation

At least one condition must be true. Since day equals 6 (Saturday), the condition is true. Only one part of the OR needs to be true.

Example 3: NOT Operator

public class NotExample { public static void main(String[] args) { boolean isRaining = false; if (!isRaining) { System.out.println("Let's go to the park!"); } // Output: Let's go to the park! } }

🔍 Explanation

The NOT operator ! reverses the value. Since isRaining is false, !isRaining becomes true, so the code executes.

Example 4: Complex Combined Conditions

public class ComplexCondition { public static void main(String[] args) { int age = 22; boolean isCitizen = true; boolean hasIncome = true; if ((age >= 18 && isCitizen) || (hasIncome && age >= 21)) { System.out.println("Eligible to vote and work!"); } // Output: Eligible to vote and work! } }

🔍 Explanation

Parentheses help organize complex conditions. This checks if either group of conditions is true. Here, the first group (age >= 18 AND isCitizen) is true, so the entire condition is true.

Section 5: Nested If Statements

Nested if statements are if statements inside other if statements. They allow you to create more sophisticated decision-making logic by testing conditions at multiple levels.

Structure of Nested If

if (firstCondition) { if (secondCondition) { // Code executes if both conditions are true } }

Example 1: Age and Income Verification

public class LoanEligibility { public static void main(String[] args) { int age = 30; int income = 50000; if (age >= 18) { if (income >= 30000) { System.out.println("Eligible for loan!"); } } // Output: Eligible for loan! } }

🔍 Explanation

First, we check if age >= 18. If true, we then check if income >= 30000. Only if both conditions are true does the person become eligible for a loan.

Example 2: Grade Assignment System

public class GradeAssignment { public static void main(String[] args) { int marks = 88; if (marks >= 80) { if (marks >= 90) { System.out.println("Grade: A+"); } if (marks >= 85 && marks < 90) { System.out.println("Grade: A"); } if (marks >= 80 && marks < 85) { System.out.println("Grade: A-"); } } // Output: Grade: A } }

🔍 Explanation

This creates multiple levels of checking. First, we verify the marks are >= 80. Then we check the exact range to assign the appropriate grade. Since 88 falls in the 85-90 range, "Grade: A" is printed.

Example 3: Discount Calculation

public class DiscountCalculator { public static void main(String[] args) { double amount = 5000; boolean isMember = true; if (amount > 1000) { if (isMember) { double discount = amount * 0.20; // 20% discount for members System.out.println("Discount: $" + discount); System.out.println("Final Price: $" + (amount - discount)); } else { double discount = amount * 0.10; // 10% discount for non-members System.out.println("Discount: $" + discount); System.out.println("Final Price: $" + (amount - discount)); } } // Output: Discount: $1000.0 // Final Price: $4000.0 } }

🔍 Explanation

First, we check if the amount exceeds 1000. If yes, we then check if the customer is a member. Members get 20% discount, non-members get 10%.

Example 4: Three-Level Nested Decision

public class JobFitness { public static void main(String[] args) { int age = 28; int experience = 5; double gpa = 3.8; if (age >= 22) { if (experience >= 3) { if (gpa >= 3.5) { System.out.println("Senior Position Offered!"); } else { System.out.println("Junior Position Offered!"); } } else { System.out.println("Internship Position Offered!"); } } else { System.out.println("Too young. Apply after graduation."); } // Output: Senior Position Offered! } }

🔍 Explanation

This demonstrates three levels of nesting. Each level further refines the decision. First check age, then experience, then GPA. Based on all three factors, an appropriate position is offered.

⚠️ Best Practice Warning

While nested if statements are powerful, deeply nested code can become hard to read and maintain. If you have more than 3 levels of nesting, consider using else-if statements or switch statements for better code readability.
