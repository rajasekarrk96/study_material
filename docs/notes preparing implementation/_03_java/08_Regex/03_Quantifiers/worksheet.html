Question 1: Add Dash Before and After Every Vowel

Problem Statement

Write a Java program to add a dash (-) before and after every vowel in a given string.

Vowels to match: a, e, i, o, u (both lowercase and uppercase)

Examples:
  • "education" → "-e-d-u-c-a-t-i--o-n"
  • "hello" → "h-e-ll-o-"
  • "Java" → "J-a-v-a-"

Solution

class _04_ { public static void main(String[] args) { String str = "education"; String result = str.replaceAll("([aeiouAEIOU])", "-$1-"); System.out.println(result); } }

Sample Output

Input: "education" Output: -e-d-u-c-a-t-i--o-n Input: "hello" Output: h-e-ll-o- Input: "Java" Output: J-a-v-a- Input: "programming" Output: pr-o-gr-a-mm-i-ng Input: "AEIOU" Output: -A--E--I--O--U-

Regex Pattern Explanation

Pattern: ([aeiouAEIOU]) Replacement: -$1- Breakdown: - ( : Start of capturing group - [aeiouAEIOU] : Character class matching any vowel (lowercase or uppercase) - aeiou : Lowercase vowels - AEIOU : Uppercase vowels - ) : End of capturing group - $1 : Reference to captured group (the matched vowel) How it works: 1. Find each vowel in the string 2. Capture it in group 1 3. Replace with: dash + captured vowel + dash 4. Result: "-vowel-"

Understanding Capturing Groups

// Capturing Group Syntax (pattern) : Captures the matched text $1, $2, ... : References captured groups in replacement Example: Pattern: "([aeiou])" Input: "hello" Matches: "e" and "o" Replacement: "-$1-" Result: "h-e-ll-o-" // Multiple Groups Example Pattern: "(\\w)(\\d)" Input: "a1b2" Replacement: "$2$1" Result: "1a2b" (swaps letter and digit)

Alternative Approaches

// Approach 1: Case-insensitive flag (not in replaceAll) Pattern pattern = Pattern.compile("([aeiou])", Pattern.CASE_INSENSITIVE); Matcher matcher = pattern.matcher(str); String result = matcher.replaceAll("-$1-"); // Approach 2: Without capturing group (less efficient) String result = str.replaceAll("a", "-a-") .replaceAll("e", "-e-") .replaceAll("i", "-i-") .replaceAll("o", "-o-") .replaceAll("u", "-u-") .replaceAll("A", "-A-") .replaceAll("E", "-E-") .replaceAll("I", "-I-") .replaceAll("O", "-O-") .replaceAll("U", "-U-"); // Approach 3: Using lookahead/lookbehind (advanced) String result = str.replaceAll("(?=[aeiouAEIOU])", "-") .replaceAll("(?<=[aeiouAEIOU])", "-");
Key Concepts:
  • Character Class []: Matches any single character from the set
  • Capturing Group (): Captures matched text for later reference
  • Backreference $1: Refers to the first captured group
  • replaceAll(): Replaces all occurrences matching the pattern
Common Pitfalls:
  • Consecutive vowels create double dashes: "aa" → "-a--a-"
  • Forgetting uppercase vowels will miss matches
  • Using replace() instead of replaceAll() only replaces first occurrence
Real-World Applications:
  • Text formatting and decoration
  • String transformation and manipulation
  • Pattern-based text processing
