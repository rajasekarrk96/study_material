Regex Character Classes

Section 1: Regular Expressions Basics

Regular Expressions (Regex) are powerful patterns used to match, search, and manipulate text. They provide a concise and flexible means to find strings that match a particular pattern. Regular expressions are essential for text processing, validation, and string manipulation.

Common Uses of Regex

  • Email validation
  • Phone number validation
  • Password strength checking
  • Text search and replace
  • Data extraction and parsing

💡 Key Concept

A regular expression is a pattern of characters used to match or find text. In Java, regex patterns are typically defined as String literals.

🎯 Real-World Example

Validating email formats: an email must have an @ symbol followed by a domain name - regex can check this pattern automatically.

Section 2: Character Classes Overview

A character class defines a set of characters that can match at a single position in the text:

What is a Character Class?

// [abc] - matches any single character a, b, or c // [0-9] - matches any single digit 0-9 // [a-z] - matches any lowercase letter // [A-Z] - matches any uppercase letter

Basic Syntax

Pattern Meaning Example
[abc] Any of a, b, or c [cat] matches c, a, or t
[^abc] NOT a, b, or c [^0-9] matches any non-digit
[a-z] Range a to z Matches any lowercase letter
[0-9] Range 0 to 9 Matches any digit

🔍 Understanding Character Classes

Character classes are enclosed in square brackets [ ]. They define a SET of characters, and the regex matches ANY ONE character from that set.

Section 3: Standard Character Classes [abc] and [^abc]

The simplest character classes match specific characters or exclude specific characters:

Example 1: Matching Specific Characters

import java.util.regex.*; public class BasicCharClasses { public static void main(String[] args) { // Pattern [aeiou] matches any vowel String pattern = "[aeiou]"; String text = "Hello World"; Pattern p = Pattern.compile(pattern); Matcher m = p.matcher(text); System.out.println("Vowels found:"); while (m.find()) { System.out.println(m.group()); // Prints each vowel found } } }

Output

Vowels found:
e
o
o

Example 2: Excluding Characters with [^]

public class ExcludingCharacters { public static void main(String[] args) { // [^0-9] matches any character that is NOT a digit String pattern = "[^0-9]+"; String text = "abc123def456"; Pattern p = Pattern.compile(pattern); Matcher m = p.matcher(text); System.out.println("Non-digit sequences:"); while (m.find()) { System.out.println(m.group()); } } }

Output

Non-digit sequences:
abc
def

Example 3: Range Patterns

public class RangePatterns { public static void main(String[] args) { String text = "The Price is $50 for item #42"; // [0-9] matches any digit Pattern digits = Pattern.compile("[0-9]"); Matcher m = digits.matcher(text); System.out.println("Digits found: " + m.group()); // [a-z] matches lowercase letters Pattern lowercase = Pattern.compile("[a-z]"); m = lowercase.matcher(text); System.out.print("Lowercase letters: "); while (m.find()) { System.out.print(m.group() + " "); } } }

⚠️ Important

Character classes match ONE character at a time. To match multiple characters, you need quantifiers (covered later).

Section 4: Predefined Classes \\d, \\w, \\s

Java provides predefined character classes as shortcuts for common patterns:

Predefined Classes Table

Class Equivalent Meaning Example
\\d [0-9] Any digit Matches: 5
\\D [^0-9] Non-digit Matches: a
\\w [a-zA-Z0-9_] Word character Matches: a, 5, _
\\W [^a-zA-Z0-9_] Non-word character Matches: @, #, !
\\s Whitespace Space, tab, newline Matches: space
\\S Non-whitespace Any non-whitespace Matches: a, 5

Example 1: Using \\d

public class DigitClass { public static void main(String[] args) { String pattern = "\\d\\d\\d-\\d\\d\\d\\d"; // Phone pattern: XXX-XXXX String text = "Call me at 555-1234 for details"; Pattern p = Pattern.compile(pattern); Matcher m = p.matcher(text); if (m.find()) { System.out.println("Phone number found: " + m.group()); } } }

Output

Phone number found: 555-1234

Example 2: Using \\w and \\W

public class WordCharacters { public static void main(String[] args) { // \\w matches word characters String pattern = "\\w+"; // One or more word characters String text = "Hello_World @123 Java-Script"; Pattern p = Pattern.compile(pattern); Matcher m = p.matcher(text); System.out.println("Word sequences:"); while (m.find()) { System.out.println(m.group()); } } }

Output

Word sequences:
Hello_World
123
Java
Script

Example 3: Using \\s

public class WhitespaceClass { public static void main(String[] args) { // \\s matches whitespace String text = "Java Programming Language"; String[] words = text.split("\\s+"); // Split by whitespace System.out.println("Words:"); for (String word : words) { System.out.println(word); } } }

Output

Words:
Java
Programming
Language

💡 Escaping Backslash

In Java strings, backslash is an escape character, so we use \\ instead of \ in regex patterns.

Section 5: Pattern Matching Examples

Practical examples combining character classes for real-world pattern matching:

Example 1: Email Validation (Simple)

public class EmailValidation { public static void main(String[] args) { // Simple email pattern: word characters @ word characters . letters String pattern = "[\\w]+@[\\w]+\\.[a-z]{2,}"; String[] emails = { "john@example.com", "alice.smith@company.co.uk", "invalid@email", "test123@domain.org" }; Pattern p = Pattern.compile(pattern); for (String email : emails) { Matcher m = p.matcher(email); System.out.println(email + ": " + (m.find() ? "Valid" : "Invalid")); } } }

Example 2: Extracting Numbers

public class ExtractNumbers { public static void main(String[] args) { String text = "Invoice: 12345, Amount: 999.99, Code: 5678"; String pattern = "\\d+"; // One or more digits Pattern p = Pattern.compile(pattern); Matcher m = p.matcher(text); System.out.println("Numbers found:"); while (m.find()) { System.out.println(m.group()); } } }

Output

Numbers found:
12345
999
99
5678

Example 3: Validating Alphanumeric

public class AlphanumericValidation { public static void main(String[] args) { String pattern = "[a-zA-Z0-9]+"; // Only letters and digits String[] inputs = { "JavaCode123", "Hello_World", "Test2024", "Valid!", "abc" }; Pattern p = Pattern.compile("^" + pattern + "$"); // ^ and $ match start and end for (String input : inputs) { Matcher m = p.matcher(input); System.out.println(input + ": " + (m.matches() ? "Valid" : "Invalid")); } } }

Output

JavaCode123: Valid
Hello_World: Invalid
Test2024: Valid
Valid!: Invalid
abc: Valid

Example 4: Replacing Non-Digits

public class ReplaceNonDigits { public static void main(String[] args) { String phone = "Phone: (555) 123-4567"; String digits = phone.replaceAll("\\D", ""); // Replace all non-digits with empty System.out.println("Original: " + phone); System.out.println("Digits only: " + digits); } }

Output

Original: Phone: (555) 123-4567
Digits only: 5551234567

💡 Regex Methods in Java

  • Pattern.compile() - Creates a Pattern object
  • Pattern.matcher() - Creates a Matcher for the string
  • Matcher.find() - Finds next match
  • Matcher.matches() - Checks if entire string matches
  • String.replaceAll() - Replace using regex
  • String.split() - Split using regex
