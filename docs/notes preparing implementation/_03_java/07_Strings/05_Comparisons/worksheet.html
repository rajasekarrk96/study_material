Question 1: == vs equals() Comparison

Problem Statement

Write a Java program that demonstrates the difference between == and equals() when comparing strings. Create strings using both literals and new keyword, then compare them using both methods.

Solution

import java.util.Scanner; class _01_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a string: "); String input = sn.nextLine(); String s1 = input; // Literal String s2 = input; // Same literal String s3 = new String(input); // new keyword String s4 = new String(input); // Another new System.out.println("\n=== Using == (Reference Comparison) ==="); System.out.println("s1 == s2: " + (s1 == s2)); // true (same pool reference) System.out.println("s1 == s3: " + (s1 == s3)); // false (different locations) System.out.println("s3 == s4: " + (s3 == s4)); // false (different heap objects) System.out.println("\n=== Using equals() (Content Comparison) ==="); System.out.println("s1.equals(s2): " + s1.equals(s2)); // true System.out.println("s1.equals(s3): " + s1.equals(s3)); // true System.out.println("s3.equals(s4): " + s3.equals(s4)); // true System.out.println("\n✓ == compares memory addresses (references)"); System.out.println("✓ equals() compares actual content (values)"); System.out.println("✓ Always use equals() for string comparison!"); sn.close(); } }

Sample Output

Enter a string: Java === Using == (Reference Comparison) === s1 == s2: true s1 == s3: false s3 == s4: false === Using equals() (Content Comparison) === s1.equals(s2): true s1.equals(s3): true s3.equals(s4): true ✓ == compares memory addresses (references) ✓ equals() compares actual content (values) ✓ Always use equals() for string comparison!
Key Concepts:
  • == Operator: Compares object references (memory addresses)
  • equals() Method: Compares string content character-by-character
  • Pool vs Heap: Affects == results but not equals() results
  • Best Practice: Always use equals() for string content comparison
Warning: Using == for strings can lead to unexpected bugs

Question 2: compareTo() for Lexicographic Ordering

Problem Statement

Write a Java program that uses compareTo() to: • Compare two strings lexicographically • Sort an array of strings in alphabetical order • Explain the return values (negative, zero, positive)

Solution

import java.util.Scanner; class _02_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first string: "); String s1 = sn.nextLine(); System.out.print("Enter second string: "); String s2 = sn.nextLine(); int result = s1.compareTo(s2); System.out.println("\ncompareTo() result: " + result); if (result < 0) { System.out.println("\"" + s1 + "\" comes BEFORE \"" + s2 + "\""); } else if (result > 0) { System.out.println("\"" + s1 + "\" comes AFTER \"" + s2 + "\""); } else { System.out.println("\"" + s1 + "\" is EQUAL to \"" + s2 + "\""); } // Sorting example System.out.print("\nHow many strings to sort? "); int n = sn.nextInt(); sn.nextLine(); // consume newline String[] arr = new String[n]; for (int i = 0; i < n; i++) { System.out.print("Enter string " + (i + 1) + ": "); arr[i] = sn.nextLine(); } // Bubble sort using compareTo() for (int i = 0; i < n - 1; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr[j].compareTo(arr[j + 1]) > 0) { String temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } System.out.println("\nSorted order:"); for (String str : arr) { System.out.println(str); } sn.close(); } }

Sample Output

Enter first string: Apple Enter second string: Banana compareTo() result: -1 "Apple" comes BEFORE "Banana" How many strings to sort? 4 Enter string 1: Dog Enter string 2: Apple Enter string 3: Cat Enter string 4: Banana Sorted order: Apple Banana Cat Dog
Key Concepts:
  • Return Values: Negative (less), 0 (equal), Positive (greater)
  • Lexicographic: Dictionary order based on Unicode values
  • Case Sensitive: Uppercase letters come before lowercase
  • Sorting: Perfect for alphabetical sorting algorithms
Formula: s1.compareTo(s2) returns s1 - s2 (first differing character)

Question 3: equalsIgnoreCase() for Case-Insensitive Comparison

Problem Statement

Write a Java program that implements a login system where username comparison is case-insensitive. Use equalsIgnoreCase() to validate user input against stored credentials.

Solution

import java.util.Scanner; class _03_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); String storedUsername = "Admin"; String storedPassword = "Pass123"; System.out.println("=== Login System ==="); System.out.print("Enter username: "); String username = sn.next(); System.out.print("Enter password: "); String password = sn.next(); // Username: case-insensitive boolean usernameMatch = username.equalsIgnoreCase(storedUsername); // Password: case-sensitive boolean passwordMatch = password.equals(storedPassword); System.out.println("\n=== Validation Results ==="); System.out.println("Username check (case-insensitive):"); System.out.println(" Input: " + username); System.out.println(" Stored: " + storedUsername); System.out.println(" Match: " + usernameMatch); System.out.println("\nPassword check (case-sensitive):"); System.out.println(" Input: " + password); System.out.println(" Stored: " + storedPassword); System.out.println(" Match: " + passwordMatch); if (usernameMatch && passwordMatch) { System.out.println("\n✓ LOGIN SUCCESSFUL!"); } else { System.out.println("\n✗ LOGIN FAILED!"); if (!usernameMatch) System.out.println(" - Invalid username"); if (!passwordMatch) System.out.println(" - Invalid password"); } sn.close(); } }

Sample Output

=== Login System === Enter username: admin Enter password: Pass123 === Validation Results === Username check (case-insensitive): Input: admin Stored: Admin Match: true Password check (case-sensitive): Input: Pass123 Stored: Pass123 Match: true ✓ LOGIN SUCCESSFUL! --- Another attempt --- Enter username: ADMIN Enter password: pass123 Username check (case-insensitive): Input: ADMIN Stored: Admin Match: true Password check (case-sensitive): Input: pass123 Stored: Pass123 Match: false ✗ LOGIN FAILED! - Invalid password
Key Concepts:
  • equalsIgnoreCase(): Ignores case differences in comparison
  • Use Cases: Usernames, email addresses, commands
  • Security: Passwords should use equals() (case-sensitive)
  • User-Friendly: Reduces login errors from caps lock
Pattern: Case-insensitive for identifiers, case-sensitive for secrets

Question 4: compareToIgnoreCase() for Case-Insensitive Sorting

Problem Statement

Write a Java program that sorts strings in case-insensitive alphabetical order using compareToIgnoreCase(). Compare the results with regular compareTo() sorting.

Solution

import java.util.Scanner; class _04_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("How many strings? "); int n = sn.nextInt(); sn.nextLine(); String[] arr1 = new String[n]; String[] arr2 = new String[n]; for (int i = 0; i < n; i++) { System.out.print("Enter string " + (i + 1) + ": "); arr1[i] = sn.nextLine(); arr2[i] = arr1[i]; // Copy for second sort } // Sort using compareTo() (case-sensitive) for (int i = 0; i < n - 1; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr1[j].compareTo(arr1[j + 1]) > 0) { String temp = arr1[j]; arr1[j] = arr1[j + 1]; arr1[j + 1] = temp; } } } // Sort using compareToIgnoreCase() (case-insensitive) for (int i = 0; i < n - 1; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr2[j].compareToIgnoreCase(arr2[j + 1]) > 0) { String temp = arr2[j]; arr2[j] = arr2[j + 1]; arr2[j + 1] = temp; } } } System.out.println("\n=== Case-Sensitive Sort (compareTo) ==="); for (String str : arr1) { System.out.println(str); } System.out.println("\n=== Case-Insensitive Sort (compareToIgnoreCase) ==="); for (String str : arr2) { System.out.println(str); } sn.close(); } }

Sample Output

How many strings? 5 Enter string 1: apple Enter string 2: Banana Enter string 3: CHERRY Enter string 4: date Enter string 5: Elderberry === Case-Sensitive Sort (compareTo) === Banana CHERRY Elderberry apple date === Case-Insensitive Sort (compareToIgnoreCase) === apple Banana CHERRY date Elderberry
Key Concepts:
  • compareTo(): Uppercase before lowercase (A-Z then a-z)
  • compareToIgnoreCase(): True alphabetical order
  • Unicode Order: compareTo() follows ASCII/Unicode values
  • User Expectations: Case-insensitive usually more intuitive
Tip: Use compareToIgnoreCase() for user-facing alphabetical lists

Question 5: String Comparison Methods Summary

Problem Statement

Write a comprehensive Java program that demonstrates all string comparison methods with the same pair of strings. Show equals(), equalsIgnoreCase(), compareTo(), compareToIgnoreCase(), and == operator with detailed explanations.

Solution

import java.util.Scanner; class _05_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first string: "); String s1 = sn.nextLine(); System.out.print("Enter second string: "); String s2 = sn.nextLine(); System.out.println("\n=== COMPARISON METHODS SUMMARY ==="); // 1. == operator System.out.println("\n1. == (Reference Comparison)"); System.out.println(" s1 == s2: " + (s1 == s2)); System.out.println(" Compares: Memory addresses"); System.out.println(" Use for: Reference equality (rarely needed)"); // 2. equals() System.out.println("\n2. equals() (Content Comparison)"); System.out.println(" s1.equals(s2): " + s1.equals(s2)); System.out.println(" Compares: Exact content (case-sensitive)"); System.out.println(" Use for: Standard string comparison"); // 3. equalsIgnoreCase() System.out.println("\n3. equalsIgnoreCase() (Case-Insensitive)"); System.out.println(" s1.equalsIgnoreCase(s2): " + s1.equalsIgnoreCase(s2)); System.out.println(" Compares: Content ignoring case"); System.out.println(" Use for: Usernames, commands, emails"); // 4. compareTo() System.out.println("\n4. compareTo() (Lexicographic Order)"); int result1 = s1.compareTo(s2); System.out.println(" s1.compareTo(s2): " + result1); System.out.println(" Returns: " + getCompareResult(result1)); System.out.println(" Use for: Sorting (case-sensitive)"); // 5. compareToIgnoreCase() System.out.println("\n5. compareToIgnoreCase() (Case-Insensitive Order)"); int result2 = s1.compareToIgnoreCase(s2); System.out.println(" s1.compareToIgnoreCase(s2): " + result2); System.out.println(" Returns: " + getCompareResult(result2)); System.out.println(" Use for: Sorting (case-insensitive)"); // Summary System.out.println("\n=== QUICK REFERENCE ==="); System.out.println("✓ For equality: use equals()"); System.out.println("✗ Avoid == for string comparison!"); sn.close(); } public static String getCompareResult(int result) { if (result < 0) return "Negative (s1 < s2)"; else if (result > 0) return "Positive (s1 > s2)";
