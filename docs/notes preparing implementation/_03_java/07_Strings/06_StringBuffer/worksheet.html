StringBuffer - Practice

Practice Questions

Practice mutable string operations using the StringBuffer class.

Question 1: Insert Character

Write a Java program to insert a char ‘v’ into a given string "Jaa Functions" at a specified position 2 using StringBuffer.
class _01_ { public static void main(String[] args) { StringBuffer sb = new StringBuffer("Jaa Functions"); sb.insert(2, 'v'); System.out.println("After insertion: " + sb); } }
After insertion: Java Functions

Question 2: Reverse String

Write a Java program to reverse a given string "Happy Birthday" using StringBuffer.
class _02_ { public static void main(String[] args) { StringBuffer sb = new StringBuffer("Happy Birthday"); sb.reverse(); System.out.println("Reversed: " + sb); } }
Reversed: yadhtriB yppaH

Question 3: Insert Substring

Write a Java program to insert a substring "not " into a given string "It’s good" at a specified position 6 using StringBuffer.
class _03_ { public static void main(String[] args) { StringBuffer sb = new StringBuffer("It’s good"); sb.insert(6, "not "); System.out.println("Modified string: " + sb); } }
Modified string: It’s not good

Question 4: Delete Substring

Write a Java program to delete a specific substring "and Python" from a given string "Java and Python is a pure object oriented program” using StringBuffer.
class _04_ { public static void main(String[] args) { StringBuffer sb = new StringBuffer("Java and Python is a pure object oriented program"); int start = sb.indexOf("and Python"); int end = start + "and Python".length(); if (start != -1) { sb.delete(start, end); } System.out.println("After deletion: " + sb); } }
After deletion: Java is a pure object oriented program

Question 5: Replace Substring

Write a Java program to replace a specific substring "Hello" from position 0 to 5 with another substring "Goodbye" in a given string "Hello, World!" using StringBuffer.
class _05_ { public static void main(String[] args) { StringBuffer sb = new StringBuffer("Hello, World!"); sb.replace(0, 5, "Goodbye"); System.out.println("After replacement: " + sb); } }
After replacement: Goodbye, World!
