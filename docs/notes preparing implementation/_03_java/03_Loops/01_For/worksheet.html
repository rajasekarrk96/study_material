Question 1: Sum of Even and Odd Digits of a Number

Problem Statement

Write a program to get an integer and then print the sum of the even and odd digits.

Solution

import java.util.Scanner; class _01_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); int evenSum = 0, oddSum = 0; for (; n > 0; n = n / 10) { int d = n % 10; if (d % 2 == 0) evenSum += d; else oddSum += d; } System.out.println("Sum of Even digits = " + evenSum); System.out.println("Sum of Odd digits = " + oddSum); sn.close(); } }

Sample Output

Enter a number: 12345 Sum of Even digits = 6 Sum of Odd digits = 9
Key Concepts:
  • Digit Extraction: n % 10 gets last digit, n / 10 removes it
  • For Loop: Continues while n > 0
  • Conditional Sum: Check if digit is even or odd
  • Accumulation: evenSum and oddSum track totals
Logic: 12345 → digits: 5(odd), 4(even), 3(odd), 2(even), 1(odd)

Question 2: HCF of Two Numbers

Problem Statement

Write a program to calculate HCF (Highest Common Factor) of two given numbers.

Solution

import java.util.Scanner; class _02_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int a = sn.nextInt(); System.out.print("Enter second number: "); int b = sn.nextInt(); int hcf = 1; int min = (a < b) ? a : b; for (int i = 1; i <= min; i++) { if (a % i == 0 && b % i == 0) hcf = i; } System.out.println("HCF = " + hcf); sn.close(); } }

Sample Output

Enter first number: 24 Enter second number: 36 HCF = 12
Key Concepts:
  • HCF/GCD: Largest number that divides both numbers
  • Optimization: Loop only up to minimum of two numbers
  • Divisibility Check: Both a % i == 0 and b % i == 0
  • Update HCF: Keep updating to find highest common factor
Example: Factors of 24: 1,2,3,4,6,8,12,24; Factors of 36: 1,2,3,4,6,9,12,18,36 → HCF = 12

Question 3: Happy Number Check

Problem Statement

Write a program to find if the given number is a happy number or not. Examples: 7, 19 are happy numbers.

Solution

import java.util.Scanner; class _03_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter number: "); int n = sn.nextInt(); int temp = n; for (; n != 1 && n != 4; ) { int sum = 0; for (; n > 0; n = n / 10) { int d = n % 10; sum += d * d; } n = sum; } if (n == 1) System.out.println(temp + " is a Happy Number"); else System.out.println(temp + " is not a Happy Number"); sn.close(); } }

Sample Output

Enter number: 19 19 is a Happy Number Enter number: 20 20 is not a Happy Number
Key Concepts:
  • Happy Number: Sum of squares of digits eventually reaches 1
  • Cycle Detection: If reaches 4, it will loop forever (not happy)
  • Nested Loops: Outer loop repeats process, inner sums squares
  • Example: 19 → 1²+9² = 82 → 8²+2² = 68 → 6²+8² = 100 → 1²+0²+0² = 1 ✓
Termination: Loop stops when n == 1 (happy) or n == 4 (unhappy cycle)

Question 4: Decimal to Binary Conversion

Problem Statement

Write a Java program to convert a decimal number to binary number.

Solution

import java.util.Scanner; class _04_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter decimal number: "); int n = sn.nextInt(); int binary = 0, place = 1; for (; n > 0; n = n / 2) { int rem = n % 2; binary += rem * place; place *= 10; } System.out.println("Binary = " + binary); sn.close(); } }

Sample Output

Enter decimal number: 13 Binary = 1101
Key Concepts:
  • Binary Conversion: Repeatedly divide by 2, collect remainders
  • Place Value: Build binary number from right to left
  • Remainder: n % 2 gives binary digit (0 or 1)
  • Process: 13 → 13/2=6 rem 1 → 6/2=3 rem 0 → 3/2=1 rem 1 → 1/2=0 rem 1 → 1101
Algorithm: Divide by 2, store remainder, repeat until quotient is 0

Question 5: Series: 1, 1, 2, 3, 4, 9, 8, 27, 16, 81…

Problem Statement

Write a program to print the given series: 1, 1, 2, 3, 4, 9, 8, 27, 16, 81… HINT: Odd positions have 2^n values (n=0,1,2…), even positions have 3^n values (n=0,1,2…)

Solution

import java.util.Scanner; class _05_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter number of terms: "); int n = sn.nextInt(); int p2 = 0, p3 = 0; for (int i = 1; i <= n; i++) { if (i % 2 != 0) { System.out.print((int)Math.pow(2, p2) + " "); p2++; } else { System.out.print((int)Math.pow(3, p3) + " "); p3++; } } sn.close(); } }

Sample Output

Enter number of terms: 10 1 1 2 3 4 9 8 27 16 81
Key Concepts:
  • Alternating Pattern: Odd positions use powers of 2, even use powers of 3
  • Math.pow(): Calculates power (2^0, 2^1, 2^2... and 3^0, 3^1, 3^2...)
  • Position Check: i % 2 != 0 for odd positions
  • Series: Pos 1:2^0=1, Pos 2:3^0=1, Pos 3:2^1=2, Pos 4:3^1=3, Pos 5:2^2=4...
Pattern: Odd: 2^0, 2^1, 2^2, 2^3... | Even: 3^0, 3^1, 3^2, 3^3...
