While Loop

While Loop in Java

Execute code repeatedly as long as a condition remains true.

What is a While Loop?

A while loop repeats a block of code as long as a specified condition is true. The condition is checked before each iteration.

while (condition) {
    // Code to execute
    // Update condition variable
}
  • Condition: Boolean expression checked before each iteration
  • Loop Body: Code that executes when condition is true
  • Update: Must change condition to eventually become false
💡 Key Point: Always update the loop variable inside the loop to avoid infinite loops.

Example 1: Count from 1 to 5

public class WhileExample {
    public static void main(String[] args) {
        int count = 1;
        
        while (count <= 5) {
            System.out.println("Count: " + count);
            count++;
        }
    }
}

Example 2: Sum Numbers

public class SumExample {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        
        while (i <= 10) {
            sum += i;
            i++;
        }
        
        System.out.println("Sum of 1 to 10: " + sum);
    }
}
Sum of 1 to 10: 55

Example 3: User Input Validation

import java.util.Scanner;

public class InputValidation {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        String password = "";
        
        while (!password.equals("java123")) {
            System.out.print("Enter password: ");
            password = input.nextLine();
            
            if (!password.equals("java123")) {
                System.out.println("Wrong password! Try again.");
            }
        }
        
        System.out.println("Access granted!");
        input.close();
    }
}

Loop Control Statements

break Statement
int i = 1;
while (i <= 10) {
    if (i == 5) {
        break;  // Exit loop when i is 5
    }
    System.out.println(i);
    i++;
}
// Output: 1 2 3 4
continue Statement
int i = 0;
while (i < 5) {
    i++;
    if (i == 3) {
        continue;  // Skip when i is 3
    }
    System.out.println(i);
}
// Output: 1 2 4 5

Common Mistakes

❌ Infinite Loop (Forgot to update): 
int count = 1;
while (count <= 5) {
    System.out.println(count);
    // Missing count++ - infinite loop!
}
✓ Correct Version: 
int count = 1;
while (count <= 5) {
    System.out.println(count);
    count++;  // Update variable
}
⚠️ Remember: The condition is checked BEFORE each iteration. If false initially, the loop body never executes.
