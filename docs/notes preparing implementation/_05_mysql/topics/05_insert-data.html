Insert Data

The INSERT statement is used to add new records (rows) into a table.

Insert data into all columns

🔹 Definition
Adds a new record providing values for every column in the table.

🔹 Syntax
INSERT INTO table_name\nVALUES (value1, value2, value3);
🔹 Example
INSERT INTO students\nVALUES (1, 'Ravi', 21);
idnameage
1Ravi21

Insert data into specific columns

🔹 Definition
Adds a new record providing values only for some columns. Missing columns will be NULL or Use Default.

🔹 Syntax
INSERT INTO table_name (column1, column2)\nVALUES (value1, value2);
🔹 Example
INSERT INTO students (id, name)\nVALUES (2, 'Anu');
idnameage
1Ravi21
2AnuNULL

Insert multiple rows at once

🔹 Definition
Efficiently adds multiple records in a single SQL statement.

🔹 Syntax
INSERT INTO table_name (column1, column2)\nVALUES\n(value1, value2),\n(value3, value4);
🔹 Example
INSERT INTO students (id, name, age)\nVALUES\n(3, Kumar, 22),\n(4, Divya, 20);
Query OK, 2 rows affected
📌 Records for Kumar and Divya added.

Practice Worksheet

Q1. Insert one record into the students table.
Query:
INSERT INTO students VALUES (1, 'Arun', 20);

Output:
Query OK, 1 row affected
Q2. Insert multiple records into the students table.
Query:
INSERT INTO students VALUES
(2, 'Priya', 21),
(3, 'Kiran', 22);

Output:
Query OK, 2 rows affected
Q3. Insert values into selected columns only.
Query:
INSERT INTO students (id, name) VALUES (4, 'Meena');

Output:
Query OK, 1 row affected
Q4. Insert data into a table from another table.
Query:
INSERT INTO student_backup
SELECT * FROM students;

Output:
Query OK, 4 rows affected
Q5. Display all records from the students table.
Query:
SELECT * FROM students;

Output:

| 1  | Arun   | 20   |
| 2  | Priya  | 21   |
| 3  | Kiran  | 22   |
| 4  | Meena  | NULL |
