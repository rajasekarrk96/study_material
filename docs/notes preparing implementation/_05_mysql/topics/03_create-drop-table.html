Create / Drop Table

The CREATE TABLE statement is used to create a new table in a database with specified columns and data types.

CREATE TABLE

🔹 Definition
Used to create a new table in a database with specified columns and data types.

🔹 Syntax
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype
);
🔹 Example
CREATE TABLE students (
    student_id INT,
    name VARCHAR(50),
    age INT
);
Query OK, 0 rows affected
📌 A new table named students is created.

CREATE TABLE IF NOT EXISTS

🔹 Definition
Used to create a table only if it does not already exist.
If the table exists, no error occurs.

🔹 Syntax
CREATE TABLE IF NOT EXISTS table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype
);
🔹 Example
CREATE TABLE IF NOT EXISTS students (
    student_id INT,
    name VARCHAR(50),
    age INT
);
Query OK, 0 rows affected
📌 No error is shown.

DROP TABLE

🔹 Definition
Used to permanently delete a table from the database along with all its data and structure.

🔹 Syntax
DROP TABLE table_name;
🔹 Example
DROP TABLE students;
Query OK, 0 rows affected
✅ Table students is deleted completely.

DROP TABLE IF EXISTS

🔹 Definition
Prevents error if the table does not exist.

🔹 Syntax
DROP TABLE IF EXISTS table_name;
🔹 Example
DROP TABLE IF EXISTS students;
Query OK, 0 rows affected (or Warning if not exists)

Drop Multiple Tables

🔹 Definition
Used to delete multiple tables in a single command.

🔹 Syntax
DROP TABLE table1, table2, table3;
🔹 Example
DROP TABLE students, employees, departments;
Query OK, 0 rows affected
