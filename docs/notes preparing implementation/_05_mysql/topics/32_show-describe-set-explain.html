Show / Describe / Set / Explain

Commands used for database inspection, structure analysis, and configuration in MySQL.

SHOW

Used to display information about databases, tables, columns, or server status.

🔹 Syntax
SHOW DATABASES;\nSHOW TABLES;\nSHOW COLUMNS FROM employees;\nSHOW CREATE TABLE employees;
Result (SHOW TABLES):
Tables_in_company
employees
departments

DESCRIBE (DESC)

Used to see the structure of a table, including field names, data types, and keys.

🔹 Syntax
DESCRIBE employees;\n-- OR\nDESC employees;
Result:
FieldTypeNullKeyDefaultExtra
emp_idintNOPRINULLauto_increment
namevarchar(50)YESNULL
salaryintYESNULL

SET

Used to assign values to system variables or user-defined variables.

🔹 Syntax
SET @total = 100;\nSET autocommit = 0;
🔹 Example
SELECT @total;
Result:
@total
100

EXPLAIN

Used to analyze the query execution plan to understand how MySQL processes a query.

🔹 Syntax
EXPLAIN SELECT * FROM employees WHERE salary > 50000;
Result (simplified):
idselect_typetabletypekeyrows
1SIMPLEemployeesALLNULL5
