IF (MySQL)

IF (MySQL)

The IF() function is used to apply conditional logic within a query, returning one value if a condition is TRUE and another if it is FALSE.

Basic Concept & Syntax

🔹 Logic
The IF() function works similarly to a ternary operator in programming languages.

🔹 Syntax
IF(condition, value_if_true, value_if_false)

Example Table: students

Sample data used for the IF examples.

student_idnamemarks
1Arun85
2Ravi45
3Meena72

Example 1: Pass or Fail

🔹 Goal
Determine if a student passed based on a threshold of 50 marks.

🔹 Example
SELECT name, marks,\nIF(marks >= 50, 'Pass', 'Fail') AS result\nFROM students;
Result:
namemarksresult
Arun85Pass
Ravi45Fail
Meena72Pass

Example 2: IF with calculation

🔹 Goal
Add 5 grace marks to those who passed, while keeping others the same.

🔹 Example
SELECT name,\nIF(marks >= 50, marks + 5, marks) AS final_marks\nFROM students;
Result:
namefinal_marks
Arun90
Ravi45
Meena77

Example 3: IF with UPDATE

🔹 Goal
Update a status column in bulk using conditional logic.

🔹 Example
UPDATE students\nSET status = IF(marks >= 50, 'Pass', 'Fail');
✔ Directly updates table records based on the condition.

Practice Worksheet

Q1. Display age category using IF.
Query:
SELECT name, age,
       IF(age >= 23, 'Adult', 'Young') AS age_category
FROM students;

Output:
+--------+-----+-------------+
| name   | age | age_category|
+--------+-----+-------------+
| Arun   | 22  | Young       |
| Priya  | 23  | Adult       |
| Kiran  | 23  | Adult       |
| Meena  | 24  | Adult       |
+--------+-----+-------------+
Q2. Mark scholarship eligibility using IF.
Query:
SELECT name, age,
       IF(age < 23, 'Eligible', 'Not Eligible') AS scholarship
FROM students;

Output:
+--------+-----+-------------+
| name   | age | scholarship |
+--------+-----+-------------+
| Arun   | 22  | Eligible    |
| Priya  | 23  | Not Eligible|
| Kiran  | 23  | Not Eligible|
| Meena  | 24  | Not Eligible|
+--------+-----+-------------+
Q3. Calculate marks status using IF (Pass/Fail).
Query:
SELECT name, marks,
       IF(marks >= 50, 'Pass', 'Fail') AS result
FROM students;

Output:
+--------+-------+--------+
| name   | marks | result |
+--------+-------+--------+
| Arun   | 85    | Pass   |
| Priya  | 45    | Fail   |
| Kiran  | 90    | Pass   |
| Meena  | 40    | Fail   |
+--------+-------+--------+
Q4. Display message based on department using IF.
Query:
SELECT name, dept_id,
       IF(dept_id = 1, 'IT Dept', 'Other Dept') AS dept_name
FROM students;

Output:
+--------+---------+------------+
| name   | dept_id | dept_name  |
+--------+---------+------------+
| Arun   | 1       | IT Dept    |
| Priya  | 2       | Other Dept |
| Kiran  | 1       | IT Dept    |
| Meena  | NULL    | Other Dept |
+--------+---------+------------+
Q5. Use nested IF to assign grade based on marks.
Query:
SELECT name, marks,
       IF(marks >= 85, 'A',
          IF(marks >= 70, 'B', 'C')) AS grade
FROM students;

Output:
+--------+-------+-------+
| name   | marks | grade |
+--------+-------+-------+
| Arun   | 85    | A     |
| Priya  | 45    | C     |
| Kiran  | 90    | A     |
| Meena  | 40    | C     |
+--------+-------+-------+
