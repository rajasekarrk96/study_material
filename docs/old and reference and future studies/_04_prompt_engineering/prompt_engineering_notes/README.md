## How To Use

1. Study modules in order.
2. In each notebook, move from small topics to larger frameworks.
3. Rewrite the sample prompts in your own words.
4. Keep a separate experiment notebook for your own prompt tests.
5. Revisit the evaluation, safety, and iteration modules repeatedly.
