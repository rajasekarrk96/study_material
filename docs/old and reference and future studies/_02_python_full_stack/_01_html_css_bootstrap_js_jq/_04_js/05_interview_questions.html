1. Two Sum (Leetcode #1) (Easy)

Problem: Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice. Time complexity goal: O(N).

Goal: Test algorithmic thinking and usage of Maps/HashMaps in JS.

View Solution 
function twoSum(nums, target) {
    const map = new Map();
    for (let i = 0; i < nums.length; i++) {
        const complement = target - nums[i];
        if (map.has(complement)) {
            return [map.get(complement), i];
        }
        map.set(nums[i], i);
    }
    return [];
}

2. Valid Palindrome (Easy)

Problem: A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Write a function isPalindrome(s) returning true or false.

Goal: Test string manipulation and regular expressions.

View Solution 
function isPalindrome(s) {
    // Remove non-alphanumeric and lowercase
    const cleanStr = s.replace(/[^a-zA-Z0-9]/g, '').toLowerCase();
    
    // Two pointers
    let left = 0;
    let right = cleanStr.length - 1;
    
    while(left < right) {
        if(cleanStr[left] !== cleanStr[right]) return false;
        left++;
        right--;
    }
    return true;
}

3. Currying in JavaScript (Medium)

Problem: Write a function sum(a)(b)(c) that returns the sum of a, b, and c. It should only be callable like sum(1)(2)(3) // returns 6.

Goal: Validate profound understanding of closures and returning functions from functions.

View Solution 
function sum(a) {
    return function(b) {
        return function(c) {
            return a + b + c;
        }
    }
}

4. Deep Clone an Object (Medium)

Problem: Write a function deepClone(obj) that creates a deep copy of a nested JavaScript object (copies values, not just references). Do not use JSON.parse(JSON.stringify(obj)).

Goal: Test recursion and understanding of primitives vs references in JS.

View Solution Sketch 
function deepClone(obj) {
    if (obj === null || typeof obj !== 'object') {
        return obj; // Return primitives immediately
    }
    // Handle Array or Object
    let copy = Array.isArray(obj) ? [] : {};
    for (let key in obj) {
        if (obj.hasOwnProperty(key)) {
            copy[key] = deepClone(obj[key]); // Recursive call
        }
    }
    return copy;
}

5. Implement a Debounce Function (Hard)

Problem: Implement a debounce(func, wait) function. It should return a new version of the function that delays its execution until after wait milliseconds have elapsed since the last time it was invoked. (Often used for search bars to prevent firing API calls on every keystroke).

Goal: Highly common FAANG interview question. Tests closures, setTimeout, and the this / arguments context binding.

View Solution Sketch 
function debounce(func, wait) {
    let timeoutId;
    
    return function(...args) {
        // Clear the previous timer
        clearTimeout(timeoutId);
        
        // Start a new timer
        timeoutId = setTimeout(() => {
            // Call the original function with right context/args
            func.apply(this, args); 
        }, wait);
    };
}
