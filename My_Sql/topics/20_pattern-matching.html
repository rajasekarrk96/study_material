Pattern Matching (LIKE)

Pattern matching is used to search for values that match a specific pattern. The main operators are LIKE and REGEXP (or RLIKE).

LIKE Operator & Wildcards

🔹 Definition
Used to match simple patterns using wildcards.

🔹 Syntax
SELECT *\nFROM table_name\nWHERE column LIKE 'pattern';
SymbolMeaning
%Any number of characters (including zero)
_Exactly one character

Examples of LIKE patterns

🔹 Practical Usage

🔹 Example
-- Starts with 'A'\nSELECT * FROM students WHERE name LIKE 'A%';\n\n-- Ends with 'n'\nSELECT * FROM students WHERE name LIKE '%n';\n\n-- Contains 'ra'\nSELECT * FROM students WHERE name LIKE '%ra%';\n\n-- Exactly 4 characters\nSELECT * FROM students WHERE name LIKE '____';
✔ Success: Filters rows matching the character patterns.

NOT LIKE

🔹 Definition
Used to exclude certain patterns from the result set.

🔹 Example
SELECT * FROM students\nWHERE name NOT LIKE 'A%';
✔ Displays names that do NOT start with 'A'.

REGEXP (MySQL) / RLIKE

🔹 Definition
Used for advanced pattern matching using regular expressions.

🔹 Syntax
SELECT *\nFROM table_name\nWHERE column REGEXP 'pattern';
🔹 Example
-- Starts with A or R\nSELECT * FROM students WHERE name REGEXP '^[AR]';\n\n-- Ends with 'n'\nSELECT * FROM students WHERE name REGEXP 'n$';\n\n-- Contains 'ra' or 'ri'\nSELECT * FROM students WHERE name REGEXP 'ra|ri';
✔ Advanced matching applied using regex.

Practice Worksheet

Q1. Display students whose name starts with 'A'.
Query:
SELECT * FROM students
WHERE name LIKE 'A%';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
+----+--------+------+
Q2. Display students whose name ends with 'a'.
Query:
SELECT * FROM students
WHERE name LIKE '%a';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 4  | Meena  | 24   |
+----+--------+------+
Q3. Display students whose name contains 'ri'.
Query:
SELECT * FROM students
WHERE name LIKE '%ri%';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
+----+--------+------+
Q4. Display students whose name does NOT start with 'K'.
Query:
SELECT * FROM students
WHERE name NOT LIKE 'K%';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q5. Display students whose name has exactly 5 characters.

+----+--------+------+
| id | name   | age  |
+----+--------+------+
