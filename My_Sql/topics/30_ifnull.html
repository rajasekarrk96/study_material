IFNULL

IFNULL

The IFNULL() function is used to replace NULL values with a specified value.

Basic Concept & Syntax

🔹 Logic
If the expression is NULL, this function returns a specified value. If the expression is NOT NULL, it returns the expression itself.

🔹 Syntax
IFNULL(expression, value_if_null)

Example Table: employees

Sample data showing some NULL bonus values.

emp_idnamebonus
1Arun5000
2RaviNULL
3Meena3000

Example 1: Replace NULL with 0

🔹 Goal
Display a value of 0 instead of 'NULL' for employees without a bonus.

🔹 Example
SELECT name, IFNULL(bonus, 0) AS bonus\nFROM employees;
Result:
namebonus
Arun5000
Ravi0
Meena3000

Example 2: IFNULL with calculation

🔹 Goal
Ensure mathematical operations don't result in NULL when adding columns.

🔹 Example
SELECT name, salary + IFNULL(bonus, 0) AS total_salary\nFROM employees;
✔ Prevents the entire sum from becoming NULL if the bonus is missing.

Example 3: IFNULL with aggregate function

🔹 Goal
Provide a fallback value for aggregate results.

🔹 Example
SELECT IFNULL(SUM(bonus), 0) AS total_bonus\nFROM employees;
✔ Ensures the query returns 0 instead of NULL if no bonuses exist in the filtered set.

Query:

FROM students;

| Kiran  | kiran@example.com |

+--------+---------------+
