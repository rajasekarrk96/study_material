Triggers

A Trigger is a special type of stored procedure that automatically executes in response to certain events (INSERT, UPDATE, DELETE) on a table.

Basic Concept

🔹 Logic
Triggers are automatic and cannot be called manually. They are used for data validation, auditing, and maintaining database integrity.

Types of Triggers

MySQL supports triggers for various events and execution timings:

TypeDescription
BEFORE INSERTExecutes before a row is inserted.
AFTER INSERTExecutes after a row is inserted.
BEFORE UPDATEExecutes before a row is updated.
AFTER UPDATEExecutes after a row is updated.
BEFORE DELETEExecutes before a row is deleted.
AFTER DELETEExecutes after a row is deleted.

Syntax (MySQL)

🔹 Keywords

  • FOR EACH ROW: The trigger runs for every affected row.
  • NEW: Refers to the incoming value (INSERT/UPDATE).
  • OLD: Refers to the existing value (UPDATE/DELETE).

🔹 Syntax
CREATE TRIGGER trigger_name\nBEFORE|AFTER INSERT|UPDATE|DELETE\nON table_name\nFOR EACH ROW\nBEGIN\n    -- SQL statements\nEND;

Example: Student Insertion Audit

🔹 Goal
Log every new student added to the main table into an audit table.

🔹 Example
-- 1. Create audit table\nCREATE TABLE students_log (\n    student_id INT,\n    student_name VARCHAR(50),\n    action_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP\n);\n\n-- 2. Create Trigger\nDELIMITER //\nCREATE TRIGGER after_student_insert\nAFTER INSERT ON students\nFOR EACH ROW\nBEGIN\n    INSERT INTO students_log(student_id, student_name)\n    VALUES (NEW.student_id, NEW.student_name);\nEND //\nDELIMITER ;

Result Visualization

🔹 Testing
Inserting a student automatically populates the log table.

🔹 Example
INSERT INTO students(student_id, student_name, marks)\nVALUES (3, 'Bob', 78);

Practice Worksheet

Q1. Create a BEFORE INSERT trigger to set default age to 18 if not provided.
Query:
DELIMITER $$

CREATE TRIGGER before_student_insert
BEFORE INSERT ON students
FOR EACH ROW
BEGIN
    IF NEW.age IS NULL THEN
        SET NEW.age = 18;
    END IF;
END $$

DELIMITER ;

Usage Example:
INSERT INTO students(name) VALUES('Sita');
SELECT * FROM students;

Output:
+----+------+-----+
| id | name | age |
+----+------+-----+
| ...| Sita | 18  |
+----+------+-----+
Q2. Create an AFTER INSERT trigger to log new student names.
Query:
DELIMITER $$

CREATE TABLE student_log(
    log_id INT AUTO_INCREMENT PRIMARY KEY,
    student_name VARCHAR(50),
    log_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) $$

DELIMITER ;

DELIMITER $$
CREATE TRIGGER after_student_insert
AFTER INSERT ON students
FOR EACH ROW
BEGIN
    INSERT INTO student_log(student_name) VALUES (NEW.name);
END $$
DELIMITER ;

Usage Example:
INSERT INTO students(name, age) VALUES('Ravi', 22);
SELECT * FROM student_log;

Output:
+--------+--------------+---------------------+
| log_id | student_name | log_time            |
+--------+--------------+---------------------+
| 1      | Ravi         | 2025-12-29 11:00:00 |
+--------+--------------+---------------------+
Q3. BEFORE UPDATE trigger to prevent decreasing age.
Query:
DELIMITER $$

CREATE TRIGGER before_age_update
BEFORE UPDATE ON students
FOR EACH ROW
BEGIN
    IF NEW.age < OLD.age THEN
        SET NEW.age = OLD.age;
    END IF;
END $$

DELIMITER ;

Usage Example:
UPDATE students SET age = 20 WHERE name='Ravi';
SELECT * FROM students WHERE name='Ravi';

Output:
+----+------+-----+
| id | name | age |
+----+------+-----+
| ...| Ravi | 22  |
+----+------+-----+
Q4. AFTER DELETE trigger to log deleted student.
Query:
DELIMITER $$

CREATE TRIGGER after_student_delete
AFTER DELETE ON students
FOR EACH ROW
BEGIN
    INSERT INTO student_log(student_name) VALUES (OLD.name);
END $$

DELIMITER ;

Usage Example:
DELETE FROM students WHERE name='Sita';
SELECT * FROM student_log;

Output:
+--------+--------------+---------------------+
| log_id | student_name | log_time            |
+--------+--------------+---------------------+
| 2      | Sita         | 2025-12-29 11:05:00 |
+--------+--------------+---------------------+
Q5. BEFORE INSERT trigger to capitalize student names.
Query:
DELIMITER $$

CREATE TRIGGER before_name_insert
BEFORE INSERT ON students
FOR EACH ROW
BEGIN
    SET NEW.name = UPPER(NEW.name);
END $$

DELIMITER ;

Usage Example:
INSERT INTO students(name, age) VALUES('meena', 24);
SELECT * FROM students;

Output:
+----+------+-----+
| id | name | age |
+----+------+-----+
| ...| MEENA| 24  |
+----+------+-----+
