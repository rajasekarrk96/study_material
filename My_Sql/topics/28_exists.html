Exists

The EXISTS operator is used to test for the existence of any record in a subquery.

Basic Concept & Syntax

🔹 Logic
The EXISTS operator returns TRUE if the subquery returns one or more records. It is often more efficient than IN for large datasets.

🔹 Syntax
SELECT column_name\nFROM table1\nWHERE EXISTS (\n    SELECT 1\n    FROM table2\n    WHERE condition\n);

Example 1: Customers who have orders

🔹 Goal
Find names of customers who have placed at least one order.

🔹 Example
SELECT name\nFROM customers c\nWHERE EXISTS (\n    SELECT 1\n    FROM orders o\n    WHERE o.cust_id = c.cust_id\n);
Result:
name
Arun
Meena

Example 2: Customers with no orders (NOT EXISTS)

🔹 Goal
Find names of customers who haven't placed any orders yet.

🔹 Example
SELECT name\nFROM customers c\nWHERE NOT EXISTS (\n    SELECT 1\n    FROM orders o\n    WHERE o.cust_id = c.cust_id\n);
Result:
name
Ravi

Practice Worksheet

Q1. Display students who belong to a department (using EXISTS).
Query:
SELECT *
FROM students s
WHERE EXISTS (
    SELECT 1
    FROM department d
    WHERE s.dept_id = d.dept_id
);

Output:
+----+--------+---------+
| id | name   | dept_id |
+----+--------+---------+
| 1  | Arun   | 1       |
| 2  | Priya  | 2       |
| 3  | Kiran  | 1       |
+----+--------+---------+
Q2. Display students who do NOT belong to any department.
Query:
SELECT *
FROM students s
WHERE NOT EXISTS (
    SELECT 1
    FROM department d
    WHERE s.dept_id = d.dept_id
);

Output:
+----+--------+---------+
| id | name   | dept_id |
+----+--------+---------+
| 4  | Meena  | NULL    |
+----+--------+---------+
Q3. Display departments that have at least one student.
Query:
SELECT *
FROM department d
WHERE EXISTS (
    SELECT 1
    FROM students s
    WHERE s.dept_id = d.dept_id
);

Output:
+---------+------------+
| dept_id | dept_name  |
+---------+------------+
| 1       | IT         |
| 2       | HR         |
+---------+------------+
Q4. Display departments with no students.
Query:
SELECT *
FROM department d
WHERE NOT EXISTS (
    SELECT 1
    FROM students s
    WHERE s.dept_id = d.dept_id
);

Output:
+---------+------------+
| dept_id | dept_name  |
+---------+------------+
| 3       | Finance    |
+---------+------------+
Q5. Display students whose department exists and age > 22.
Query:
SELECT *
FROM students s
WHERE EXISTS (
    SELECT 1
    FROM department d
    WHERE s.dept_id = d.dept_id
) AND age > 22;

Output:
+----+--------+---------+
| id | name   | dept_id |
+----+--------+---------+
| 2  | Priya  | 2       |
| 3  | Kiran  | 1       |
+----+--------+---------+
