Case

The CASE statement is used to add conditional logic to SQL queries, similar to IF-ELSE in programming.

Basic Syntax

🔹 Structure
CASE goes through conditions and returns a value when the first condition is met.

🔹 Syntax
CASE\n    WHEN condition1 THEN result1\n    WHEN condition2 THEN result2\n    ELSE resultN\nEND

Example Table: students

Sample data used for the CASE examples.

student_idnamemarks
1Arun85
2Ravi45
3Meena72
4Sita30

Example 1: Grade students based on marks

🔹 Goal
Assign a text label based on the marks range.

🔹 Example
SELECT name, marks,\nCASE\n    WHEN marks >= 75 THEN 'Distinction'\n    WHEN marks >= 50 THEN 'Pass'\n    ELSE 'Fail'\nEND AS result\nFROM students;
Result:
namemarksresult
Arun85Distinction
Ravi45Fail
Meena72Pass
Sita30Fail

Example 2: CASE with ORDER BY

🔹 Goal
Sort the result set based on custom logic rather than raw data values.

🔹 Example
SELECT name, marks\nFROM students\nORDER BY\nCASE\n    WHEN marks >= 75 THEN 1\n    WHEN marks >= 50 THEN 2\n    ELSE 3\nEND;
✔ Groups results by category (Distinction first, then Pass, then Fail).

Example 3: CASE with UPDATE

🔹 Goal
Update a column with values based on conditions.

🔹 Example
UPDATE students\nSET result =\nCASE\n    WHEN marks >= 50 THEN 'Pass'\n    ELSE 'Fail'\nEND;
✔ Column 'result' is populated for all students based on their marks.

Example 4: CASE with GROUP BY

🔹 Goal
Group aggregate results based on a calculated condition.

🔹 Example
SELECT\nCASE\n    WHEN marks >= 50 THEN 'Pass'\n    ELSE 'Fail'\nEND AS status,\nCOUNT(*) AS total_students\nFROM students\nGROUP BY status;
Result:
statustotal_students
Pass2
Fail2

Practice Worksheet

Q1. Display student names with age group (Minor / Adult).
Query:
SELECT name,
       CASE 
           WHEN age < 18 THEN 'Minor'
           ELSE 'Adult'
       END AS age_group
FROM students;

Output:
+--------+-----------+
| name   | age_group |
+--------+-----------+
| Arun   | Adult     |
| Priya  | Adult     |
| Kiran  | Adult     |
| Meena  | Adult     |
+--------+-----------+
