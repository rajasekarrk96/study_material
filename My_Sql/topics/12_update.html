Update

The UPDATE statement is used to modify existing records in a table.

Basic UPDATE

🔹 Definition
Used to change the value of one or more columns for specific rows matching a condition.

🔹 Syntax
UPDATE table_name\nSET column1 = value1\nWHERE condition;
🔹 Example
UPDATE students\nSET age = 22\nWHERE id = 1;

UPDATE Multiple Columns

🔹 Definition
Modifies multiple fields for the same record in a single statement.

🔹 Example
UPDATE students\nSET age = 23,\n    name = 'Ravi Kumar'\nWHERE id = 1;
✔ Name and age updated for student ID 1

UPDATE Multiple Rows

🔹 Definition
Applies changes to all rows that meet the WHERE condition.

🔹 Example
UPDATE students\nSET age = age + 1\nWHERE age < 21;
✔ Incremented age for all students younger than 21

UPDATE Without WHERE (⚠️ Risky)

🔹 Warning
If you omit the WHERE clause, ALL records in the table will be updated!

🔹 Example
UPDATE students\nSET age = 25;
📌 Updates all rows in the students table.

UPDATE with SAFE UPDATE Mode (MySQL)

🔹 Rule
In MySQL Safe Mode, you must use a WHERE clause with a key column (like Primary Key).

🔹 Example
UPDATE students\nSET age = 24\nWHERE id = 2;
❌ Without WHERE → Error Code: 1175

Practice Worksheet

Q1. Update age of student with id = 1 to 21.
Query:
UPDATE students
SET age = 21
WHERE id = 1;

Output:
Query OK, 1 row affected
Q2. Update name of student with id = 2.
Query:
UPDATE students
SET name = 'Priya Sharma'
WHERE id = 2;

Output:
Query OK, 1 row affected
Q3. Update age to 23 for all students whose age is NULL.
Query:
UPDATE students
SET age = 23
WHERE age IS NULL;

Output:
Query OK, 1 row affected
Q4. Increase age of all students by 1 year.
Query:
UPDATE students
SET age = age + 1;

Output:
Query OK, 4 rows affected
Q5. Display updated records from students table.
Query:
SELECT * FROM students;
