Alter Table

The ALTER statement is used to modify the structure of an existing table.

Add a New Column

🔹 Definition
Adds a new column to an existing table.

🔹 Syntax
ALTER TABLE table_name\nADD column_name datatype;
🔹 Example
ALTER TABLE students\nADD email VARCHAR(50);
✔ Column 'email' added to students table.

Add Multiple Columns

🔹 Definition
Adds several columns in a single SQL statement.

🔹 Example
ALTER TABLE students\nADD phone VARCHAR(15),\nADD city VARCHAR(30);
✔ Columns 'phone' and 'city' added.

Modify Column Datatype / Size

🔹 Definition
Changes the data type or constraints of an existing column.

🔹 Example
-- MySQL\nALTER TABLE students\nMODIFY age INT;\n\n-- Change name & type\nALTER TABLE students\nCHANGE phone mobile_no VARCHAR(15);
✔ Column modifications applied.

Rename a Column

🔹 Definition
Changes the name of an existing column.

🔹 Example
-- MySQL 8+ / PostgreSQL\nALTER TABLE students\nRENAME COLUMN name TO full_name;
✔ Column 'name' renamed to 'full_name'.

Drop a Column

🔹 Definition
Permanently removes a column and its data from the table.

🔹 Example
ALTER TABLE students\nDROP COLUMN city;
✔ Column 'city' deleted.
