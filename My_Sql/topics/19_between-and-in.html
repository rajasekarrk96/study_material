BETWEEN & IN

Both BETWEEN and IN are used for filtering data in the WHERE clause.

BETWEEN Operator

🔹 Definition
Used to filter values within a range (inclusive).

🔹 Syntax
SELECT *\nFROM table_name\nWHERE column BETWEEN value1 AND value2;
🔹 Example
SELECT *\nFROM students\nWHERE age BETWEEN 18 AND 22;
idnameage

BETWEEN with Dates

🔹 Usage
Filters records based on a date range.

🔹 Example
SELECT *\nFROM orders\nWHERE order_date BETWEEN '2024-01-01' AND '2024-01-31';
✔ Returns all orders placed in January 2024.

IN Operator

🔹 Definition
Used to match multiple specific values from a list.

🔹 Syntax
SELECT *\nFROM table_name\nWHERE column IN (value1, value2, value3);
🔹 Example
SELECT *\nFROM students\nWHERE city IN ('Chennai', 'Delhi');
✔ Returns students from either Chennai or Delhi.

NOT BETWEEN & NOT IN

🔹 Negation
Excludes values within the specified range or list.

🔹 Example
-- NOT BETWEEN\nSELECT * FROM students WHERE age NOT BETWEEN 18 AND 22;\n\n-- NOT IN\nSELECT * FROM students WHERE city NOT IN ('Chennai', 'Delhi');
⚠️ Note: If the column contains NULL, NOT IN may return no rows.

Practice Worksheet

Q1. Display students whose age is between 22 and 24.
Query:
SELECT * FROM students
WHERE age BETWEEN 22 AND 24;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q2. Display students whose age is NOT between 22 and 24.
Query:
SELECT * FROM students
WHERE age NOT BETWEEN 22 AND 24;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 5  | Ravi   | 25   |
+----+--------+------+
Q3. Display students whose name is IN the given list.
Query:
SELECT * FROM students
WHERE name IN ('Arun', 'Meena');

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 4  | Meena  | 24   |
+----+--------+------+
Q4. Display students whose name is NOT IN the given list.
Query:
SELECT * FROM students
WHERE name NOT IN ('Arun', 'Meena');

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 5  | Ravi   | 25   |
+----+--------+------+
