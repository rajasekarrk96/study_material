Limit

The LIMIT clause is used to restrict the number of rows returned in the result set.

Basic Concept & Syntax

🔹 Logic
LIMIT is commonly used for pagination or to show only a top number of records. (MySQL, PostgreSQL)

🔹 Syntax
SELECT column_name\nFROM table_name\nLIMIT number;

Example Table: employees

Sample data used for the LIMIT examples.

emp_idnamesalary
1Arun50000
2Ravi40000
3Meena60000
4Sita45000
5Karthi55000

Example 1: Show only first 3 records

🔹 Goal
Retrieve only the first three rows from the result set.

🔹 Example
SELECT * FROM employees LIMIT 3;
Result:
emp_idnamesalary
1Arun50000
2Ravi40000
3Meena60000

Example 2: LIMIT with ORDER BY

🔹 Goal
Find the top 2 highest earners.

🔹 Example
SELECT name, salary\nFROM employees\nORDER BY salary DESC\nLIMIT 2;
Result:
namesalary
Meena60000
Karthi55000

Example 3: LIMIT with OFFSET

🔹 Goal
Skip a specific number of records before returning the result.

🔹 Example
SELECT * FROM employees LIMIT 2 OFFSET 2;
✔ Skips first 2 rows (Arun, Ravi) and shows next 2 (Meena, Sita).

Example 4: LIMIT for pagination

🔹 Goal
Standard approach for web applications to load data in pages.

🔹 Example
SELECT * FROM employees\nORDER BY emp_id\nLIMIT 5 OFFSET 10;
✔ Shows the 3rd page of results (5 records per page, skipping first 10).

Practice Worksheet

Q1. Display first 3 students from the table.
Query:
SELECT * FROM students
LIMIT 3;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
+----+--------+------+
Q2. Display 2 students starting from the second record.
Query:
SELECT * FROM students
LIMIT 1, 2;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
+----+--------+------+
Q3. Display top 1 student with highest age (using ORDER BY).
Query:
SELECT * FROM students
ORDER BY age DESC
LIMIT 1;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 4  | Meena  | 24   |
+----+--------+------+
Q4. Display first 2 students with age > 22.
Query:
SELECT * FROM students
WHERE age > 22
LIMIT 2;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
+----+--------+------+
Q5. Display 2 students after skipping first 2 records with age > 22.
Query:
SELECT * FROM students
WHERE age > 22
LIMIT 2, 2;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 4  | Meena  | 24   |
+----+--------+------+
