Union / Union All

The UNION and UNION ALL operators are used to combine the result-set of two or more SELECT statements.

Example Tables

Sample data from two different tables used for the following examples.

Table 1: customers
cust_idnamecity
1ArunChennai
2RaviMadurai
3SitaTrichy
Table 2: suppliers
supp_idnamecity
101KumarChennai
102MeenaCoimbatore
103RaniMadurai

UNION Example

🔹 Goal
Combines cities from both tables and removes duplicate values.

🔹 Syntax
SELECT city FROM customers\nUNION\nSELECT city FROM suppliers;
🔹 Example
SELECT city FROM customers\nUNION\nSELECT city FROM suppliers;
Result:
city
Chennai
Madurai
Trichy
Coimbatore

✔ Duplicate cities (Chennai, Madurai) appear only once.

UNION ALL Example

🔹 Goal
Combines cities from both tables and keeps all values, including duplicates.

🔹 Syntax
SELECT city FROM customers\nUNION ALL\nSELECT city FROM suppliers;
🔹 Example
SELECT city FROM customers\nUNION ALL\nSELECT city FROM suppliers;
Result:
city
Chennai
Madurai
Trichy
Chennai
Coimbatore
Madurai

Practice Worksheet

Q1. Display all student names from both tables using UNION (removes duplicates).
Query:
SELECT name FROM students
UNION
SELECT name FROM new_students;

Output:
+--------+
| name   |
+--------+
| Arun   |
| Priya  |
| Kiran  |
| Meena  |
| Ravi   |
| Sita   |
+--------+
Q2. Display all student names from both tables using UNION ALL (keeps duplicates).
Query:
SELECT name FROM students
UNION ALL
SELECT name FROM new_students;

Output:
+--------+
| name   |
+--------+
| Arun   |
| Priya  |
| Kiran  |
| Meena  |
| Arun   |
| Ravi   |
| Sita   |
+--------+
Q3. Count total distinct student names from both tables using UNION.
Query:
SELECT COUNT(*) AS total_names
FROM (
    SELECT name FROM students
    UNION
    SELECT name FROM new_students
) AS combined;

Output:
+------------+
| total_names|
+------------+
| 6          |
+------------+
Q4. Count total student names including duplicates using UNION ALL.
Query:
SELECT COUNT(*) AS total_names
FROM (
    SELECT name FROM students
    UNION ALL
    SELECT name FROM new_students
) AS combined;

Output:
+------------+
| total_names|
+------------+
| 7          |
+------------+
Q5. Display names from both tables ordered alphabetically using UNION.
