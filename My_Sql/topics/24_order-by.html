Order By

The ORDER BY clause is used to sort the result set in ascending (default) or descending order.

Basic Concept & Syntax

🔹 Note
By default, ORDER BY sorts the records in ascending order. To sort the records in descending order, use the DESC keyword.

🔹 Syntax
SELECT column1, column2, ...\nFROM table_name\nORDER BY column1, column2, ... ASC|DESC;

Example Table: employees

Sample data used for the following examples.

emp_idnamedepartmentsalary
1ArunIT50000
2RaviHR40000
3MeenaIT60000
4SitaHR45000

Example 1: Sort by salary (Ascending)

🔹 Goal
Sort employees from lowest salary to highest (Default behavior).

🔹 Example
SELECT name, salary \nFROM employees\nORDER BY salary;
Result:
namesalary
Ravi40000
Sita45000
Arun50000
Meena60000

Example 2: Sort by salary (Descending)

🔹 Goal
Sort employees from highest salary to lowest using the DESC keyword.

🔹 Example
SELECT name, salary \nFROM employees\nORDER BY salary DESC;
Result:
namesalary
Meena60000
Arun50000
Sita45000
Ravi40000

Example 3: Sort by multiple columns

🔹 Goal
Sort by department first (Ascending), and then by salary (Descending) within each department.

🔹 Example
SELECT * \nFROM employees\nORDER BY department ASC, salary DESC;
Result:
emp_idnamedepartmentsalary
4SitaHR45000
2RaviHR40000
3MeenaIT60000
1ArunIT50000

✔ Groups by department
✔ Highest salary first within each department

Practice Worksheet

Q1. Display all students ordered by age ascending.
Query:
SELECT * FROM students
ORDER BY age ASC;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q2. Display all students ordered by age descending.
Query:
SELECT * FROM students
ORDER BY age DESC;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 4  | Meena  | 24   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 1  | Arun   | 22   |
+----+--------+------+
Q3. Order students by name alphabetically.
Query:
SELECT * FROM students
ORDER BY name ASC;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
| 2  | Priya  | 23   |
+----+--------+------+
Q4. Order by multiple columns (age ascending, name descending).
Query:
SELECT * FROM students
ORDER BY age ASC, name DESC;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 3  | Kiran  | 23   |
| 2  | Priya  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q5. Display first 2 students ordered by age ascending.
Query:
SELECT * FROM students
ORDER BY age ASC
LIMIT 2;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 3  | Kiran  | 23   |
+----+--------+------+
