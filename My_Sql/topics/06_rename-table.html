Rename Table

In SQL, RENAME is used to change the name of a table.

Rename a table (MySQL / Oracle)

🔹 Definition
Changes the name of an existing table to a new name.

🔹 Syntax
RENAME TABLE old_table_name TO new_table_name;
🔹 Example
RENAME TABLE students TO learners;
Result: Table name changed from students → learners
