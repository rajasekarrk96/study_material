Subqueries

A Subquery is a query inside another query. It is also called a nested query.

Basic Syntax

🔹 Note
A subquery is embedded within the WHERE, FROM, or SELECT clause of another query.

🔹 Syntax
SELECT column_name\nFROM table_name\nWHERE column_name operator\n(SELECT column_name FROM table_name);

Example Table: employees

Sample data used for the subquery examples.

Single-Row Subquery

🔹 Goal
Returns exactly one value. Used with comparison operators (=, >, <, etc.).

🔹 Example
-- Find employee with highest salary\nSELECT name, salary\nFROM employees\nWHERE salary = (SELECT MAX(salary) FROM employees);
Result:
namesalary
Meena60000

Multiple-Row Subquery

🔹 Goal
Returns a list of values. Used with operators like IN, ANY, or ALL.

🔹 Example
-- Departments where at least one employee earns more than 50,000\nSELECT name, department\nFROM employees\nWHERE department IN (\n    SELECT department\n    FROM employees\n    WHERE salary > 50000\n);
Result:
namedepartment
MeenaIT
SitaFinance

ANY / ALL Subquery

🔹 ANY returns true if any one condition matches.
🔹 ALL returns true only if all conditions match.

🔹 Example
-- ANY Example: Salary > any HR salary\nSELECT name, salary FROM employees\nWHERE salary > ANY (\n    SELECT salary FROM employees WHERE department = 'HR'\n);\n\n-- ALL Example: Salary > all HR salaries\nSELECT name, salary FROM employees\nWHERE salary > ALL (\n    SELECT salary FROM employees WHERE department = 'HR'\n);
✔ ANY returns anyone who earns more than 40,000 (minimum HR salary).
✔ ALL returns anyone who earns more than 40,000 (maximum HR salary in this sample).

Subquery in SELECT

🔹 Goal
Used to include an aggregate value next to individual row data.

🔹 Example
SELECT name,\n       (SELECT AVG(salary) FROM employees) AS avg_salary\nFROM employees;
✔ Displays each employee's name along with the company-wide average salary.

Correlated Subquery

🔹 Goal
A subquery that depends on values from the outer query for its results.

🔹 Example
-- Employees earning more than their own department average\nSELECT name, department, salary\nFROM employees e\nWHERE salary > (\n    SELECT AVG(salary)\n    FROM employees\n    WHERE department = e.department\n);
Result:
namedepartmentsalary
MeenaIT60000

Practice Worksheet

Q1. Single-row subquery – Get students from the department with dept_id = 1.
Query:
SELECT * 
FROM students
WHERE dept_id = (SELECT dept_id FROM department WHERE dept_name = 'IT');

Output:
+----+-------+-----+---------+
| id | name  | age | dept_id |
+----+-------+-----+---------+
| 1  | Arun  | 22  | 1       |
| 3  | Kiran | 23  | 1       |
+----+-------+-----+---------+
Q2. Multi-row subquery – Students in departments with more than 1 student.
Query:
SELECT * 
FROM students
WHERE dept_id IN (
    SELECT dept_id 
    FROM students
    GROUP BY dept_id
    HAVING COUNT(*) > 1
);

Output:
+----+-------+-----+---------+
| id | name  | age | dept_id |
+----+-------+-----+---------+
| 1  | Arun  | 22  | 1       |
| 3  | Kiran | 23  | 1       |
+----+-------+-----+---------+
Q3. Correlated subquery – Students older than the average age.
Query:
SELECT * 
FROM students s1
WHERE age > (SELECT AVG(age) FROM students s2);

Output:
+----+-------+-----+---------+
| id | name  | age | dept_id |
+----+-------+-----+---------+
| 4  | Meena | 24  | NULL    |
+----+-------+-----+---------+
Q4. Subquery in SELECT – Show student name and department name.
Query:
SELECT name,
       (SELECT dept_name FROM department WHERE dept_id = students.dept_id) AS dept_name
FROM students;

Output:
+-------+------------+
| name  | dept_name  |
+-------+------------+
| Arun  | IT         |
| Priya | HR         |
| Kiran | IT         |
| Meena | NULL       |
+-------+------------+
Q5. Subquery with EXISTS – Students in departments that exist.
Query:
SELECT *
FROM students s
WHERE EXISTS (
    SELECT 1
    FROM department d
    WHERE s.dept_id = d.dept_id
);

Output:
+----+-------+-----+---------+
| id | name  | age | dept_id |
+----+-------+-----+---------+
| 1  | Arun  | 22  | 1       |
| 2  | Priya | 23  | 2       |
| 3  | Kiran | 23  | 1       |
+----+-------+-----+---------+
