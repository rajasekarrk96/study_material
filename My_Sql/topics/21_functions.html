Functions

SQL functions are predefined operations used to perform calculations or transformations on data.

Aggregate Functions

🔹 Definition
Used to calculate a single value from multiple rows.

🔹 Example
SELECT COUNT(*) AS total_students, AVG(age) AS avg_age\nFROM students;
FunctionDescription
COUNT()Counts rows
SUM()Adds values
AVG()Average
MIN()Minimum
MAX()Maximum

Scalar Functions - String

🔹 Definition
Operate on each row's string data.

🔹 Example
SELECT UPPER(name), CONCAT(name, ' (Student)') FROM students;
FunctionExample
UPPER()UPPER(name)
LOWER()LOWER(name)
CONCAT()CONCAT(f, ' ', l)
LENGTH()LENGTH(name)
SUBSTRING()SUBSTRING(n,1,3)

Scalar Functions - Numeric

🔹 Definition
Perform mathematical operations on individual numeric values.

🔹 Example
SELECT salary, ROUND(salary, 1) FROM employees;
FunctionExample
ROUND()ROUND(s, 2)
CEIL()CEIL(4.2) → 5
FLOOR()FLOOR(4.7) → 4
ABS()ABS(-10) → 10

Scalar Functions - Date & Time

🔹 Definition
Used to manipulate date and time values.

🔹 Example
SELECT NOW(), DATEDIFF('2024-12-31', '2024-01-01');
FunctionDescription
NOW()Current & time
CURDATE()Current date
DATEDIFF()Date difference
DATE_ADD()Add interval

Conversion Functions

🔹 Definition
Used to convert a value from one data type to another.

🔹 Example
SELECT CAST(salary AS CHAR) FROM employees;\nSELECT CONVERT(salary, CHAR) FROM employees;
✔ Success: Data types converted.

Practice Worksheet

Q1. Use COUNT() to count total students.
Query:
SELECT COUNT(*) AS total_students
FROM students;

Output:
+----------------+
| total_students |
+----------------+
| 4              |
+----------------+
Q2. Use SUM() to find the total of all ages.
Query:
SELECT SUM(age) AS total_age
FROM students;

Output:
+-----------+
| total_age |
+-----------+
| 92        |
+-----------+
Q3. Use AVG() to find the average age of students.
Query:
SELECT AVG(age) AS average_age
FROM students;

Output:
+-------------+
| average_age |
+-------------+
| 23.0        |
+-------------+
Q4. Use MAX() and MIN() to find oldest and youngest students.
Query:
SELECT MAX(age) AS oldest, MIN(age) AS youngest
FROM students;

Output:
+--------+---------+
| oldest | youngest|
+--------+---------+
| 24     | 22      |
+--------+---------+
Q5. Use UPPER() and LOWER() to change case of names.
Query:
SELECT UPPER(name) AS upper_name, LOWER(name) AS lower_name
FROM students;

Output:
+------------+------------+
| upper_name | lower_name |
+------------+------------+
| ARUN       | arun       |
| PRIYA      | priya      |
| KIRAN      | kiran      |
| MEENA      | meena      |
+------------+------------+
