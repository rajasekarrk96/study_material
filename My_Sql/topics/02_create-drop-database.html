Create / Drop Database

Commands to manage databases.

CREATE DATABASE

🔹 Definition
Used to create a new database in the SQL server.

🔹 Syntax
CREATE DATABASE database_name;
🔹 Example
CREATE DATABASE college;
Query OK, 1 row affected
📌 A new database named college is created.

CREATE DATABASE IF NOT EXISTS

🔹 Definition
Creates a database only if it does not already exist, preventing errors.

🔹 Syntax
CREATE DATABASE IF NOT EXISTS database_name;
🔹 Example
CREATE DATABASE IF NOT EXISTS company;
If database does not exist: Query OK, 1 row affected
If database already exists: Query OK, 0 rows affected
📌 No error is shown.

DROP DATABASE

🔹 Definition
Used to permanently delete a database and all its tables.
⚠️ Warning: Data cannot be recovered after dropping.

🔹 Syntax
DROP DATABASE database_name;
🔹 Example
DROP DATABASE college;
Query OK, 0 rows affected
📌 The college database is deleted.

DROP DATABASE IF EXISTS

🔹 Definition
Deletes a database only if it exists, avoiding errors.

🔹 Syntax
DROP DATABASE IF EXISTS database_name;
🔹 Example
DROP DATABASE IF EXISTS school;
Query OK, 0 rows affected
📌 No error shown if school didn't exist.
