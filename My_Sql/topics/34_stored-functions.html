Stored Functions

A Stored Function is a database object that returns exactly one value and can be used within SQL statements like built-in functions.

Key Points

    \n
  • Must return exactly one value.
    • \n
  • Used inside SELECT, WHERE, and ORDER BY clauses.
    • \n
  • Cannot perform DML operations like INSERT, UPDATE, or DELETE on tables.
    • \n
  • Uses the RETURN keyword to send back a value.
    • \n

🔹 Syntax
DELIMITER //\n\nCREATE FUNCTION function_name(parameter datatype)\nRETURNS datatype\nDETERMINISTIC\nBEGIN\n    RETURN value;\nEND //\n\nDELIMITER ;

Example 1: Create Stored Function

🔹 Goal
Create a reusable function to calculate a 10% bonus.

🔹 Example
DELIMITER //\n\nCREATE FUNCTION get_bonus(salary INT)\nRETURNS INT\nDETERMINISTIC\nBEGIN\n    RETURN salary * 0.10;\nEND //\n\nDELIMITER ;

Example 2: Use Stored Function

🔹 Goal
Call the custom function within a SELECT statement.

🔹 Example
SELECT name, salary, get_bonus(salary) AS bonus\nFROM employees;
Result:
namesalarybonus
Arun500005000
Ravi400004000
Meena600006000

Example 3: Stored Function with IF

🔹 Goal
Create a function that uses logic internally.

🔹 Example
DELIMITER //\n\nCREATE FUNCTION grade(marks INT)\nRETURNS VARCHAR(20)\nDETERMINISTIC\nBEGIN\n    RETURN IF(marks >= 50, 'Pass', 'Fail');\nEND //\n\nDELIMITER ;

Drop Stored Function

Remove the function from the database metadata.

🔹 Example
DROP FUNCTION get_bonus;

Practice Worksheet

Q1. Create a function to calculate square of a number.
Query:
DELIMITER $$

CREATE FUNCTION square_num(n INT)
RETURNS INT
DETERMINISTIC
BEGIN
    RETURN n * n;
END $$

DELIMITER ;

Usage Example:
SELECT square_num(5) AS square;

Output:
+--------+
| square |
+--------+
| 25     |
+--------+
Q2. Function to check if student is adult (age ≥ 18).
Query:
DELIMITER $$

CREATE FUNCTION is_adult(age INT)
RETURNS VARCHAR(10)
DETERMINISTIC
BEGIN
    RETURN IF(age >= 18, 'Adult', 'Minor');
END $$

DELIMITER ;

Usage Example:
SELECT is_adult(22) AS status;

Output:
+--------+
| status |
+--------+
| Adult  |
+--------+
Q3. Function to calculate total marks with bonus.
Query:
DELIMITER $$

CREATE FUNCTION total_marks(marks INT, bonus INT)
RETURNS INT
DETERMINISTIC
BEGIN
    RETURN marks + bonus;
END $$

DELIMITER ;

Usage Example:
SELECT total_marks(80, 5) AS total;

Output:
+-------+
| total |
+-------+
| 85    |
+-------+
Q4. Function to convert Celsius to Fahrenheit.
Query:
DELIMITER $$

CREATE FUNCTION c_to_f(celsius FLOAT)
RETURNS FLOAT
DETERMINISTIC
BEGIN
    RETURN (celsius * 9/5) + 32;
END $$

DELIMITER ;

Usage Example:
SELECT c_to_f(25) AS fahrenheit;

Output:
+------------+
| fahrenheit |
+------------+
| 77.0       |
+------------+
Q5. Function to get grade based on marks.
Query:
DELIMITER $$

CREATE FUNCTION get_grade(marks INT)
RETURNS VARCHAR(2)
DETERMINISTIC
BEGIN
    RETURN CASE
        WHEN marks >= 85 THEN 'A'
        WHEN marks >= 70 THEN 'B'
        ELSE 'C'
    END;
END $$

DELIMITER ;

Usage Example:
SELECT get_grade(90) AS grade;

Output:
+-------+
| grade |
+-------+
| A     |
+-------+
