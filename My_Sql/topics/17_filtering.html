Filtering

Filtering is used to retrieve specific rows from a table based on conditions. Mainly done using WHERE clause.

WHERE Clause (Basic Filtering)

🔹 Definition
Used to filter records based on a specified condition.

🔹 Syntax
SELECT * \nFROM table_name\nWHERE condition;
🔹 Example
SELECT * \nFROM students\nWHERE age > 20;
✔ Displays students older than 20

Comparison Operators

🔹 Definition
Operators used to compare values.

🔹 Syntax
SELECT * FROM students\nWHERE age >= 21;
OperatorMeaning
=Equal
!= / <>Not equal
>Greater than
<Less than
>=Greater or equal
<=Less or equal

AND / OR / NOT (Logical Filtering)

🔹 Definition
Used to combine multiple conditions.

🔹 Example
-- AND condition\nSELECT * FROM students\nWHERE age > 20 AND city = 'Chennai';\n\n-- OR condition\nSELECT * FROM students\nWHERE city = 'Delhi' OR city = 'Mumbai';\n\n-- NOT condition\nSELECT * FROM students\nWHERE NOT city = 'Chennai';
✔ Complex logical filtering applied.

IN (Multiple Values)

🔹 Definition
Specifies multiple possible values for a column.

🔹 Example
SELECT * FROM students\nWHERE city IN ('Chennai', 'Delhi');
✔ Returns students from Chennai or Delhi.

BETWEEN (Range)

🔹 Definition
Selects values within a given range (inclusive).

🔹 Example
SELECT * FROM students\nWHERE age BETWEEN 18 AND 22;
✔ Returns students aged 18 to 22.

LIKE (Pattern Matching)

🔹 Definition
Searches for a specified pattern in a column.

🔹 Example
-- Starts with A\nSELECT * FROM students\nWHERE name LIKE 'A%';\n\n-- Contains 'an'\nSELECT * FROM students\nWHERE name LIKE '%an%';
✔ Patterns matched using wildcards (% and _).

IS NULL / IS NOT NULL

🔹 Definition
Used to check for empty (NULL) values.

🔹 Example
SELECT * FROM students\nWHERE email IS NULL;
✔ Displays students without an email address.

Filtering with Dates

🔹 Definition
Applying conditions to date columns.

🔹 Example
SELECT * FROM orders\nWHERE order_date >= '2024-01-01';
✔ Returns orders from 2024 onwards.

HAVING (Filter Groups)

🔹 Definition
Used to filter groups created by GROUP BY (WHERE cannot filter groups).

🔹 Example
SELECT city, COUNT(*)\nFROM students\nGROUP BY city\nHAVING COUNT(*) > 2;
✔ Only shows cities with more than 2 students.

Practice Worksheet

Q1. Display students whose age is greater than 21.
Query:
SELECT * FROM students
WHERE age > 21;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q2. Display students whose age is between 21 and 23.
Query:
SELECT * FROM students
WHERE age BETWEEN 21 AND 23;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
+----+--------+------+
Q3. Display students whose name is either Arun or Meena.
Query:
SELECT * FROM students
WHERE name IN ('Arun', 'Meena');

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 4  | Meena  | 24   |
+----+--------+------+
Q4. Display students whose name starts with 'P'.
Query:
SELECT * FROM students
WHERE name LIKE 'P%';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
+----+--------+------+
Q5. Display students whose age is NOT NULL.
Query:
SELECT * FROM students
WHERE age IS NOT NULL;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
