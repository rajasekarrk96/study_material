Data Types

Data Types

Defines the type of data a column can hold.

Numeric Data Types

Used to store numbers

🔹 INT – Stores whole numbers
🔹 SMALLINT – Stores small whole numbers
🔹 BIGINT – Stores very large whole numbers
🔹 DECIMAL / NUMERIC – Stores exact decimal values
🔹 FLOAT – Stores approximate decimal numbers
🔹 DOUBLE – Stores large decimal numbers with precision

Character / String Data Types

Used to store text

🔹 CHAR – Fixed-length character data
🔹 VARCHAR – Variable-length character data
🔹 TEXT – Stores long text

Date & Time Data Types

Used to store date and time values

🔹 DATE – Stores date only
🔹 TIME – Stores time only
🔹 DATETIME – Stores date and time
🔹 TIMESTAMP – Stores current date & time automatically
🔹 YEAR – Stores year

Boolean Data Type

Used to store true or false

🔹 BOOLEAN / BOOL – Stores TRUE or FALSE values

ENUM Data Type

Used to store one value from a predefined list

🔹 ENUM – Allows only one value from a fixed set

SET Data Type

Used to store multiple values from a predefined list

🔹 SET – Allows multiple values from a fixed set

Binary Data Types

Used to store binary data

🔹 BINARY – Fixed-length binary data
🔹 VARBINARY – Variable-length binary data
🔹 BLOB – Stores large binary objects
