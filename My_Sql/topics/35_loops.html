Loops (Procedures)

Loops allow you to repeatedly execute a block of SQL statements within a stored procedure until a specific condition is met.

Basic Loop Types

MySQL supports three main loop constructs inside procedures:

  • LOOP: Continuous loop (uses LEAVE to exit).
  • WHILE: Repeats while a condition is TRUE.
  • REPEAT: Repeats until a condition becomes TRUE.

Basic LOOP with LEAVE

🔹 Logic
Uses a label and the LEAVE command to break out of an infinite loop (similar to 'break' in Java/C++).

🔹 Syntax
label_name: LOOP\n    IF condition THEN\n        LEAVE label_name;\n    END IF;\nEND LOOP label_name;
🔹 Example
DELIMITER //\n\nCREATE PROCEDURE LoopExample()\nBEGIN\n    DECLARE i INT DEFAULT 1;\n    number_loop: LOOP\n        IF i > 5 THEN\n            LEAVE number_loop;\n        END IF;\n        INSERT INTO numbers_log(value) VALUES(i);\n        SET i = i + 1;\n    END LOOP number_loop;\nEND //\n\nDELIMITER ;

WHILE Loop

🔹 Logic
Checks the condition before each iteration. The loop only runs if the condition is TRUE.

🔹 Syntax
WHILE condition DO\n    -- statements\nEND WHILE;
🔹 Example
DELIMITER //\n\nCREATE PROCEDURE WhileExample()\nBEGIN\n    DECLARE i INT DEFAULT 1;\n    WHILE i <= 5 DO\n        INSERT INTO numbers_log(value) VALUES(i);\n        SET i = i + 1;\n    END WHILE;\nEND //\n\nDELIMITER ;

Practice Worksheet

Q1. Simple LOOP – Print numbers 1 to 5 using a procedure.
Query:
DELIMITER $$

CREATE PROCEDURE simple_loop()
BEGIN
    DECLARE i INT DEFAULT 1;
    simple_loop_label: LOOP
        IF i > 5 THEN
            LEAVE simple_loop_label;
        END IF;
        SELECT i;
        SET i = i + 1;
    END LOOP simple_loop_label;
END $$

DELIMITER ;

Usage Example:
CALL simple_loop();

Output:
+---+
| i |
+---+
| 1 |
| 2 |
| 3 |
| 4 |
| 5 |
+---+
Q2. WHILE Loop – Display numbers 1 to 5.
Query:
DELIMITER $$

CREATE PROCEDURE while_loop()
BEGIN
    DECLARE i INT DEFAULT 1;
    WHILE i <= 5 DO
        SELECT i;
        SET i = i + 1;
    END WHILE;
END $$

CALL while_loop();

+---+

Q3. REPEAT Loop – Print numbers 1 to 5.
Query:
DELIMITER $$

CREATE PROCEDURE repeat_loop()
BEGIN
    DECLARE i INT DEFAULT 1;
    repeat_loop_label: REPEAT
        SELECT i;
        SET i = i + 1;
    UNTIL i > 5
    END REPEAT repeat_loop_label;
END $$

DELIMITER ;

Usage Example:
CALL repeat_loop();

Output:
+---+
| i |
+---+
| 1 |
| 2 |
| 3 |
| 4 |
| 5 |
+---+
Q4. Loop with IF condition – Break when number is 3.
Query:
DELIMITER $$

CREATE PROCEDURE loop_with_break()
BEGIN
    DECLARE i INT DEFAULT 1;
    my_loop: LOOP
        IF i > 5 THEN
            LEAVE my_loop;
        END IF;
        IF i = 3 THEN
            LEAVE my_loop;
        END IF;
        SELECT i;
        SET i = i + 1;
    END LOOP my_loop;
END $$

DELIMITER ;

Usage Example:
CALL loop_with_break();

Output:
+---+
| i |
+---+
| 1 |
| 2 |
+---+
Q5. Nested Loops – Multiply table 1 to 3 by 1 to 3.
Query:
DELIMITER $$

CREATE PROCEDURE nested_loop()
BEGIN
    DECLARE i INT DEFAULT 1;
    DECLARE j INT DEFAULT 1;
    outer_loop: LOOP
        IF i > 3 THEN
            LEAVE outer_loop;
        END IF;
        SET j = 1;
        inner_loop: LOOP
            IF j > 3 THEN
                LEAVE inner_loop;
            END IF;
            SELECT i, j, i*j AS product;
            SET j = j + 1;
        END LOOP inner_loop;
        SET i = i + 1;
    END LOOP outer_loop;
END $$

DELIMITER ;

Usage Example:
CALL nested_loop();

Output:
+---+---+---------+
| i | j | product |
+---+---+---------+
| 1 | 1 | 1       |
| 1 | 2 | 2       |
| 1 | 3 | 3       |
| 2 | 1 | 2       |
| 2 | 2 | 4       |
| 2 | 3 | 6       |
| 3 | 1 | 3       |
| 3 | 2 | 6       |
| 3 | 3 | 9       |
+---+---+---------+
