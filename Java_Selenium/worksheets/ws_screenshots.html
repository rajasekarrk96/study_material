Practice: Screenshots

Task 01: Capture as File
Take a screenshot of the current page and save it as error.png.
Task 02: Element Screenshot
Capture only the Submit button element as a screenshot.
Solutions 
package Selenium;
 
import java.io.File;
import org.openqa.selenium.By;
import org.openqa.selenium.OutputType;
import org.openqa.selenium.TakesScreenshot;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.apache.commons.io.FileUtils;
 
public class ScreenshotsDemo {
    public static void main(String[] args) throws Exception {
        System.out.println("--- Selenium Lab Session Started ---");
        WebDriver driver = new ChromeDriver();
 
        System.out.println("Navigating to Screenshots worksheet...");
        driver.get("http://127.0.0.1:5501/Java_Selinium/worksheets/ws_screenshots.html");
 
        // Task 01: Capture full page screenshot as file
        System.out.println("Task 01: Taking full page screenshot...");
        File screenshot = ((TakesScreenshot) driver)
            .getScreenshotAs(OutputType.FILE);
        FileUtils.copyFile(screenshot, new File("error.png"));
 
        // Task 02: Capture element screenshot
        System.out.println("Task 02: Taking element screenshot of Submit button...");
        WebElement button = driver.findElement(By.id("submit"));
        File elementScreenshot = button
            .getScreenshotAs(OutputType.FILE);
        FileUtils.copyFile(elementScreenshot, 
            new File("button_screenshot.png"));
 
        System.out.println("--- Selenium Lab Session Ended ---");
        driver.quit();
    }
}

Click is not required. Selenium will capture screenshots automatically.
