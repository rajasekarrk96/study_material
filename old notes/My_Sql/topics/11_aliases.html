Aliases (AS)

Aliases (AS)

AS is used to give a temporary alias (name) to a column or table. It makes the output more readable.

AS with Column Alias

🔹 Definition
Used to rename a column in the result set for better readability.

🔹 Syntax
SELECT column_name AS alias_name\nFROM table_name;
🔹 Example
SELECT name AS student_name, age AS student_age\nFROM students;
student_namestudent_age
Ravi21
Anu20

AS with Table Alias

🔹 Definition
Used to give a table a short name, especially useful when joining tables.

🔹 Syntax
SELECT column\nFROM table_name AS alias;
🔹 Example
SELECT s.name, s.age\nFROM students AS s;
✔ Query executed using table alias 's'

AS with Calculated Column

🔹 Definition
Used to give a name to columns created by mathematical operations or expressions.

🔹 Example
SELECT name, salary * 12 AS annual_salary\nFROM employees;
Result
nameannual_salary
John360000

AS in Aggregate Functions

🔹 Definition
Used to name the result of aggregate functions like COUNT, SUM, AVG, etc.

🔹 Example
SELECT COUNT(*) AS total_students\nFROM students;
Result
total_students
10

Practice Worksheet

Q1. Display name column with alias student_name.
Query:
SELECT name AS student_name
FROM students;

Output:
+--------------+
| student_name |
+--------------+
| Arun         |
| Priya        |
| Kiran        |
| Meena        |
+--------------+
Q2. Display age column with alias student_age.
Query:
SELECT age AS student_age
FROM students;

Output:
+-------------+
| student_age |
+-------------+
| 20          |
| 21          |
| 22          |
| NULL        |
+-------------+
Q3. Use alias for table name.
Query:
SELECT s.name, s.age
FROM students AS s;

Output:
+--------+------+
| name   | age  |
+--------+------+
| Arun   | 20   |
| Priya  | 21   |
| Kiran  | 22   |
| Meena  | NULL |
+--------+------+
Q4. Display calculated column using alias.
Query:
SELECT name, age + 1 AS next_year_age
FROM students;

Output:
+--------+---------------+
| name   | next_year_age |
+--------+---------------+
| Arun   | 21            |
| Priya  | 22            |
| Kiran  | 23            |
| Meena  | NULL          |
+--------+---------------+
Q5. Use alias in ORDER BY clause.
Query:
SELECT name AS student_name, age
FROM students
ORDER BY student_name;

Output:
+--------------+------+
| student_name | age  |
+--------------+------+
| Arun         | 20   |
| Kiran        | 22   |
| Meena        | NULL |
| Priya        | 21   |
+--------------+------+
