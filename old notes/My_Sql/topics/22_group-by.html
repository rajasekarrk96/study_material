Group By

Groups records with identical values into summary rows. Often used with aggregate functions (COUNT, SUM, AVG, MIN, MAX).

Example Table: employees

Sample data used for the following examples.

emp_idnamedepartmentsalary
1ArunIT50000

Basic Syntax

🔹 Note
The GROUP BY statement is often used with aggregate functions (COUNT, MAX, MIN, SUM, AVG) to group the result-set by one or more columns.

🔹 Syntax
SELECT column_name(s)\nFROM table_name\nWHERE condition\nGROUP BY column_name(s)\nORDER BY column_name(s);

Example 1: Total salary by department

🔹 Definition
Calculates the sum of salaries for each department using the SUM() function.

🔹 Syntax
SELECT department, SUM(salary) AS total_salary\nFROM employees\nGROUP BY department;
🔹 Example
SELECT department, SUM(salary) AS total_salary\nFROM employees\nGROUP BY department;
Result:
departmenttotal_salary

Example 2: Count employees in each department

🔹 Definition
Counts the number of employees in each department using the COUNT() function.

🔹 Syntax
SELECT department, COUNT(emp_id) AS total_employees\nFROM employees\nGROUP BY department;
🔹 Example
SELECT department, COUNT(emp_id) AS total_employees\nFROM employees\nGROUP BY department;

departmenttotal_employees

GROUP BY with WHERE

🔹 Definition
The WHERE clause filters rows BEFORE they are grouped.

🔹 Example
SELECT department, AVG(salary) AS avg_salary\nFROM employees\nWHERE salary > 45000\nGROUP BY department;
✔ Calculation performed only on employees with salary > 45000.

Practice Worksheet

Q1. Count the number of students in each age group.
Query:
SELECT age, COUNT(*) AS student_count
FROM students
GROUP BY age;

Output:
+------+---------------+
| age  | student_count |
+------+---------------+
| 22   | 1             |
| 23   | 2             |
| 24   | 1             |
+------+---------------+
Q2. Sum of ages for each age group.
Query:
SELECT age, SUM(age) AS total_age
FROM students
GROUP BY age;

Output:
+------+-----------+
| age  | total_age |
+------+-----------+
| 22   | 22        |
| 23   | 46        |
| 24   | 24        |
+------+-----------+
Q3. Display average age for each age group.
Query:
SELECT age, AVG(age) AS average_age
FROM students
GROUP BY age;

Output:
+------+-------------+
| age  | average_age |
+------+-------------+
| 22   | 22.0        |
| 23   | 23.0        |
| 24   | 24.0        |
+------+-------------+
Q4. Count students by age where age is greater than 22.
Query:
SELECT age, COUNT(*) AS student_count
FROM students
WHERE age > 22
GROUP BY age;

Output:
+------+---------------+
| age  | student_count |
+------+---------------+
| 23   | 2             |
| 24   | 1             |
+------+---------------+
Q5. Count students by age and order by count descending.
Query:
SELECT age, COUNT(*) AS student_count
FROM students
GROUP BY age
ORDER BY student_count DESC;

Output:
+------+---------------+
| age  | student_count |
+------+---------------+
| 23   | 2             |
| 22   | 1             |
| 24   | 1             |
+------+---------------+
