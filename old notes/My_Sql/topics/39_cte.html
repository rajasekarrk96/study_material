CTE (Common Table Expression)

CTE (Common Table Expression)

A CTE is a temporary result set that you can reference within a SELECT, INSERT, UPDATE, or DELETE statement, making complex queries readable.

Basic Concept & Syntax

🔹 Logic
Think of it as a temporary named view that only exists for the duration of the query execution.

🔹 Syntax
WITH cte_name (col1, col2) AS (\n    SELECT ...\n)\nSELECT * FROM cte_name;

Example: Non-Recursive CTE

🔹 Goal
Simplify filtering IT employees into a readable sub-result.

🔹 Example
WITH IT_Employees AS (\n    SELECT EmployeeID, Name, Salary\n    FROM Employees\n    WHERE Department = 'IT'\n)\nSELECT * FROM IT_Employees;
Result:
EmployeeID Name Salary
1 John 5000
3 Bob 5500

Example: Recursive CTE

🔹 Goal
Generate a numeric sequence from 1 to 5.

🔹 Example
WITH RECURSIVE Numbers AS (\n    SELECT 1 AS n          -- Anchor\n    UNION ALL\n    SELECT n + 1\n    FROM Numbers\n    WHERE n < 5            -- Condition\n)\nSELECT * FROM Numbers;
Result (n):
1
2
3
4
5

Practice Worksheet

Q1. Simple CTE – Select students older than 22.
Query:
WITH older_students AS (
    SELECT * FROM students
    WHERE age > 22
)
SELECT * FROM older_students;

Output:
+----+-------+-----+---------+
| id | name  | age | dept_id |
+----+-------+-----+---------+
| 2  | Priya | 23  | 2       |
| 3  | Kiran | 23  | 1       |
| 4  | Meena | 24  | NULL    |
+----+-------+-----+---------+
Q2. CTE with aggregate – Count students per department.
Query:
WITH dept_count AS (
    SELECT dept_id, COUNT(*) AS total_students
    FROM students
    GROUP BY dept_id
)
SELECT * FROM dept_count;

Output:
+---------+----------------+
| dept_id | total_students |
+---------+----------------+
| 1       | 2              |
| 2       | 1              |
| NULL    | 1              |
+---------+----------------+
Q3. CTE with JOIN – Students with department names.
Query:
WITH student_dept AS (
    SELECT s.id, s.name, d.dept_name
    FROM students s
    LEFT JOIN department d
    ON s.dept_id = d.dept_id
)
SELECT * FROM student_dept;

Output:
+----+-------+------------+
| id | name  | dept_name  |
+----+-------+------------+
| 1  | Arun  | IT         |
| 2  | Priya | HR         |
| 3  | Kiran | IT         |
| 4  | Meena | NULL       |
+----+-------+------------+
Q4. Recursive CTE – Generate numbers from 1 to 5.
Query:
WITH RECURSIVE numbers AS (
    SELECT 1 AS n
    UNION ALL
    SELECT n + 1 FROM numbers WHERE n < 5
)
SELECT * FROM numbers;

Output:
+---+
| n |
+---+
| 1 |
| 2 |
| 3 |
| 4 |
| 5 |
+---+
Q5. CTE with multiple steps – Students older than 22 and count per department.
Query:
WITH older_students AS (
    SELECT * FROM students WHERE age > 22
),
dept_count AS (
    SELECT dept_id, COUNT(*) AS total_students
    FROM older_students
    GROUP BY dept_id
)
SELECT * FROM dept_count;

Output:
+---------+----------------+
| dept_id | total_students |
+---------+----------------+
| 1       | 1              |
| 2       | 1              |
| NULL    | 1              |
+---------+----------------+
