Select

The SELECT statement is used to retrieve data from one or more tables in a database.

Basic Concepts

🔹 Definition
Used to view data without changing it. It is the primary command of Data Query Language (DQL).

🔹 Syntax
SELECT column1, column2 FROM table_name;
🔹 Example
-- Select specific columns\nSELECT emp_name, salary FROM employees;

Select All Columns

🔹 Definition
Uses the asterisk (*) wildcard to fetch all columns from a table.

🔹 Syntax
SELECT * FROM table_name;
🔹 Example
SELECT * FROM employees;
emp_idemp_namedeptsalary
101ArunIT60000
102PriyaHR45000
103RaviIT70000

Select Specific Columns

🔹 Definition
Retrieves only the columns you specify, separating them with commas.

🔹 Syntax
SELECT column1, column2 FROM table_name;
🔹 Example
SELECT emp_name, salary FROM employees;
emp_namesalary
Arun60000
Priya45000
Ravi70000

Practice Worksheet

Q1. Display all columns from the students table.
Query:
SELECT * FROM students;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 20   |
| 2  | Priya  | 21   |
| 3  | Kiran  | 22   |
| 4  | Meena  | NULL |
+----+--------+------+
Q2. Display only name and age columns from students.
Query:
SELECT name, age FROM students;

Output:
+--------+------+
| name   | age  |
+--------+------+
| Arun   | 20   |
| Priya  | 21   |
| Kiran  | 22   |
| Meena  | NULL |
+--------+------+
Q3. Display students whose age is greater than 20.
Query:
SELECT * FROM students
WHERE age > 20;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 21   |
| 3  | Kiran  | 22   |
+----+--------+------+
Q4. Display students ordered by age in descending order.
Query:
SELECT * FROM students
ORDER BY age DESC;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 3  | Kiran  | 22   |
| 2  | Priya  | 21   |
| 1  | Arun   | 20   |
| 4  | Meena  | NULL |
+----+--------+------+
Q5. Display only first 2 records from students.
Query:
SELECT * FROM students
LIMIT 2;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 20   |
| 2  | Priya  | 21   |
+----+--------+------+
