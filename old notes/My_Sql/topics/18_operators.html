Operators

Operators

Operators are symbols used to perform operations on data in SQL queries.

Arithmetic Operators

🔹 Definition
Used for mathematical calculations.

🔹 Example
SELECT salary, salary * 12 AS annual_salary\nFROM employees;
OperatorMeaning
+Addition
-Subtraction
*Multiplication
/Division
%Modulus

Comparison (Relational) Operators

🔹 Definition
Used to compare values.

🔹 Example
SELECT * FROM students\nWHERE age >= 21;
OperatorMeaning
=Equal
!= or <>Not equal
>Greater than
<Less than
>=Greater or equal
<=Less or equal

Logical Operators

🔹 Definition
Used to combine conditions.

🔹 Example
SELECT * FROM students\nWHERE age > 20 AND city = 'Chennai';
OperatorMeaning
ANDAll conditions true
ORAny condition true
NOTNegates condition

BETWEEN Operator

🔹 Definition
Checks if a value is within a range.

🔹 Example
SELECT * FROM students\nWHERE age BETWEEN 18 AND 22;
✔ Success: Filters students in the specified age range.

IN Operator

🔹 Definition
Checks if a value matches any value in a list.

🔹 Example
SELECT * FROM students\nWHERE city IN ('Chennai', 'Delhi');
✔ Success: Filters students from the listed cities.

LIKE Operator

🔹 Definition
Used for pattern matching with wildcards (% and _).

🔹 Example
SELECT * FROM students\nWHERE name LIKE 'A%';
✔ Success: Matches names starting with 'A'.

IS NULL Operator

🔹 Definition
Used to check for NULL values.

🔹 Syntax
SELECT * FROM students\nWHERE email IS NULL;
✔ Success: Filters rows where email is missing.

EXISTS Operator

🔹 Definition
Checks if a subquery returns any rows.

🔹 Example
SELECT * FROM orders o\nWHERE EXISTS (\n  SELECT 1 FROM customers c\n  WHERE c.id = o.customer_id\n);
✔ Success: Returns orders only for existing customers.

ANY / ALL Operators

🔹 Definition
Used to compare a value against a set of values returned by a subquery.

🔹 Example
SELECT * FROM students\nWHERE marks > ALL (\n  SELECT marks FROM exams WHERE subject = 'Math'\n);
✔ Success: Filters students with marks higher than all math exam marks.

Practice Worksheet

Q1. Use Arithmetic operator (+) to increase age by 1.
Query:
SELECT name, age + 1 AS next_age
FROM students;

Output:
+--------+----------+
| name   | next_age |
+--------+----------+
| Arun   | 23       |
| Priya  | 24       |
| Kiran  | 24       |
| Meena  | 25       |
+--------+----------+
Q2. Use Comparison operator (>) to filter students older than 22.
Query:
SELECT * FROM students
WHERE age > 22;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
| 4  | Meena  | 24   |
+----+--------+------+
Q3. Use Logical operator (AND).
Query:
SELECT * FROM students
WHERE age > 22 AND name <> 'Meena';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 2  | Priya  | 23   |
| 3  | Kiran  | 23   |
+----+--------+------+
Q4. Use Logical operator (OR).
Query:
SELECT * FROM students
WHERE age = 22 OR name = 'Meena';

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 4  | Meena  | 24   |
+----+--------+------+
Q5. Use NOT operator.
Query:
SELECT * FROM students
WHERE NOT age = 23;

Output:
+----+--------+------+
| id | name   | age  |
+----+--------+------+
| 1  | Arun   | 22   |
| 4  | Meena  | 24   |
+----+--------+------+
