WebDriver Commands

Learning Objectives

  • Master finding elements using CSS and XPath selectors
  • Understand element methods like click(), sendKeys(), getText()
  • Retrieve element attributes and properties
  • Handle element visibility and enabled states
  • Work with collections of WebElements

Practice Scenario

Target Application: Practice Test Automation Login

Your Task:

  1. Task 01: Find the username input field using multiple locator strategies
  2. Task 02: Find the password input field by its ID attribute
    (Instruction: Use the ID locator 'password' for this task)
  3. Task 03: Extract the text from the "Submit" button
    (Instruction: Use the ID locator 'submit' for this task)
  4. Task 04: Check if the username field is enabled before typing
  5. Task 05: Get all form labels to understand the form structure
  6. Task 06: Find all input fields on the form and print their tag names

Expected Output: Successfully locate elements and retrieve their properties without errors

Possible Solution
package Selenium;
 
import org.openqa.selenium.*;
import org.openqa.selenium.chrome.ChromeDriver;
import java.time.Duration;
import java.util.List;
 
public class WebDriverCommands {
    public static void main(String[] args) {
        System.out.println("--- Selenium Lab Session Started ---");
        WebDriver driver = new ChromeDriver();
        driver.manage().window().maximize();
        driver.manage().timeouts().implicitlyWait(Duration.ofSeconds(10));
 
        try {
            System.out.println("Navigating to application...");
            driver.get("https://practicetestautomation.com/practice-test-login/");
 
            // Task 01: Find username field using multiple strategies
            System.out.println("Executing Task 01: Finding username field using multiple strategies...");
            WebElement usernameById = driver.findElement(By.id("username"));
            WebElement usernameByName = driver.findElement(By.name("username"));
            WebElement usernameByCss = driver.findElement(By.cssSelector("input[name='username']"));
            System.out.println("Username field successfully located via ID, Name, and CSS Selector.");
 
            // Task 02: Find password field by ID
            System.out.println("Executing Task 02: Locating password field by ID...");
            WebElement passwordField = driver.findElement(By.id("password"));
            System.out.println("Password field found.");
 
            // Task 03: Extract text from Submit button by ID
            System.out.println("Executing Task 03: Extracting text from Submit button...");
            WebElement submitBtn = driver.findElement(By.id("submit"));
            System.out.println("Submit button text: " + submitBtn.getText());
 
            // Task 04: Check if username field is enabled
            System.out.println("Executing Task 04: Checking if username field is enabled...");
            System.out.println("Is Username Field Enabled: " + usernameById.isEnabled());
 
            // Task 05: Get all form labels
            System.out.println("Executing Task 05: Retrieving all form labels...");
            List labels = driver.findElements(By.tagName("label"));
            System.out.println("Total labels found: " + labels.size());
            for (WebElement label : labels) {
                System.out.println("- Label Text: " + label.getText());
            }
 
            // Task 06: Find all input fields and print their details
            System.out.println("Executing Task 06: Identifying all input fields...");
            List inputs = driver.findElements(By.tagName("input"));
            System.out.println("Total input fields: " + inputs.size());
            for (WebElement input : inputs) {
                String type = input.getAttribute("type");
                String name = input.getAttribute("name");
                System.out.println("- Input [Type: " + type + ", Name: " + name + "]");
            }
 
        } finally {
            System.out.println("--- Selenium Lab Session Ended ---");
            driver.quit();
        }
    }
}

Helpful Hints

💡 Tip 1: Use By.id() for elements with ID attributes, it's the fastest

💡 Tip 2: You can find elements by name, class, CSS selector, or XPath

💡 Tip 3: Use isEnabled() and isDisplayed() to check element state

💡 Tip 4: Use findElements() (plural) to get multiple elements as a List

💡 Tip 5: Always check the browser's Inspector (F12) to find correct selectors

Key Takeaways
  • ID is fastest: Use By.id() whenever available
  • CSS Selectors are flexible: Learn CSS selectors for complex scenarios
  • XPath is powerful: Use for dynamic and complex element selection
  • Validate element state: Always check isEnabled() and isDisplayed() before interacting
  • Reuse locators: Create a separate locators class for maintainability
  • Test on different browsers: Locators might work differently on Edge, Firefox, etc.
