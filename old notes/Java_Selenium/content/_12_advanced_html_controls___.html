12. Advanced HTML Controls

📖 Introduction

Blindly interacting with elements is a recipe for flaky tests. Before you click or type, you often need to verify the state of an element: Is it visible? Is it enabled? Is it already checked? Furthermore, you'll arguably spend more time reading data from the screen (validation) than entering it.

💡 What You'll Learn

  • Verifying if elements are displayed, enabled, or selected
  • Retrieving text and attribute values for assertions
  • The difference between getText() and getAttribute("value")

🔍 Detailed Explanation

1. Verification Commands (Boolean)

These return true or false. Use them in if conditions or assertions.

  • isDisplayed(): Checks if the element is rendered and visible on the page.
  • isEnabled(): Checks if the element is interactive (not disabled/grayed out).
  • isSelected(): Checks if a Checkbox or Radio Button is checked.

2. Retrieval Commands (String)

Used to extract data for validation.

  • getText(): Gets the inner text between opening and closing tags (e.g., <h1>Text</h1>).
  • getAttribute("attributeName"): Gets the value of a specific attribute (e.g., href, class, value).

💻 Code Examples

Validation logic example:

Safe Interaction

WebElement submitBtn = driver.findElement(By.id("submit"));

// 1. Check if safe to click
if(submitBtn.isDisplayed() && submitBtn.isEnabled()) {
    submitBtn.click();
} else {
    System.out.println("Button is disabled!");
}

// 2. Validate Checkbox
WebElement chk = driver.findElement(By.id("terms"));
if(chk.isSelected()) {
    System.out.println("Already agreed to terms");
}

⚠️ Common Pitfalls

getText() vs getAttribute("value"): This is the #1 confusion. If you type valid into an <input> box, getText() will return empty string! You MUST use getAttribute("value") to get typed text.

🎯 Real-World Scenario

Practice Scenario

Goal: Validate a Profile Page.

  1. Verify the "Save" button is disabled initially.
  2. Type a new email address.
  3. Verify the "Save" button becomes enabled.
  4. Click "Save".
  5. Verify the success message text says "Profile Updated".

📝 Interview Questions

Q1: Can you use isSelected() on a button?

Answer: No, it generally returns false. It is designed for <input type='checkbox'> and <input type='radio'> elements.
