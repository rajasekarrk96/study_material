20. Scrolling

📖 Introduction

Sometimes the elements you need to interact with are not visible on the screen and require scrolling to bring them into view. Since Selenium doesn't have a built-in scroll method in the WebDriver interface, we typically use JavaScript via the JavascriptExecutor to perform scrolling operations.

💡 What You'll Learn

  • How to use JavascriptExecutor for scrolling
  • How to scroll by a specific number of pixels
  • How to scroll to a specific element (Scroll into View)
  • How to scroll to the bottom or top of a page

🔍 Detailed Explanation

JavascriptExecutor

To execute JavaScript in Selenium, you need to cast your WebDriver instance to JavascriptExecutor:

JavascriptExecutor js = (JavascriptExecutor) driver;

Common Scrolling Methods

  • Scroll by Pixels: window.scrollBy(x, y) 
    js.executeScript("window.scrollBy(0, 1000)"); // Scrolls down 1000px
  • Scroll to Element: scrollIntoView() 
    WebElement element = driver.findElement(By.id("footer"));
js.executeScript("arguments[0].scrollIntoView(true);", element);
  • Scroll to Bottom: 
    js.executeScript("window.scrollTo(0, document.body.scrollHeight)");

💻 Code Examples

Scrolling to a Specific Element

WebElement footer = driver.findElement(By.id("footer"));
JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("arguments[0].scrollIntoView(true);", footer);

🎯 Real-World Scenario

Practice Scenario

Goal: Handle an infinite scrolling page or a long landing page.

  1. Identify an element at the very bottom of the page (e.g., a "Terms of Service" link).
  2. Use JavaScript to scroll until that element is visible.
  3. Assert that the element is displayed before clicking it.
