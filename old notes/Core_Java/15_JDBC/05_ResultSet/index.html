ResultSet in JDBC

Navigate and retrieve data from query results

What is ResultSet?

ResultSet represents the result set of a database query, allowing you to navigate through rows and retrieve column values.

Common ResultSet Methods

ResultSet rs = stmt.executeQuery("SELECT * FROM users");

while (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    System.out.println(id + " - " + name);
}
