CallableStatement in JDBC

Execute stored procedures and functions in databases

What is CallableStatement?

CallableStatement is used to execute stored procedures in the database.

CallableStatement cstmt = conn.prepareCall("{call getUser(?)}");
cstmt.setInt(1, 101);
ResultSet rs = cstmt.executeQuery();
