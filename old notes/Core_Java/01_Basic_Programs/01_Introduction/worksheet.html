Question 1: Hello World Program

Problem Statement

Write a Java program that prints "Hello, World!" to the console. This is the most basic Java program that demonstrates the structure of a Java class and the main method.

Sample Input and Output

Sample Input 1:
(No input required)

Sample Output 1:
Hello, World!

Sample Input 2:
(No input required)

Sample Output 2:
Hello, World!
Show Solution

Solution (Java Program)

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

Step-by-Step Explanation

  1. Class Declaration: public class HelloWorld - Declares a public class named HelloWorld. In Java, every program must have at least one class.
  2. Main Method: public static void main(String[] args) - This is the entry point of any Java program. The JVM looks for this method to start execution.
  3. Print Statement: System.out.println("Hello, World!"); - Prints the text "Hello, World!" to the console and moves to a new line.
  4. Semicolon: Every statement in Java must end with a semicolon (;).
  5. Curly Braces: The { } braces define the scope of the class and method.

Question 2: Print Your Name

Problem Statement

Write a Java program that takes your name as input from the user and prints a greeting message "Hello, [Your Name]!".

Sample Input and Output

Sample Input 1:
John

Sample Output 1:
Hello, John!

Sample Input 2:
Sarah

Sample Output 2:
Hello, Sarah!
Show Solution

Solution (Java Program)

import java.util.Scanner; public class PrintName { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter your name: "); String name = sn.nextLine(); System.out.println("Hello, " + name + "!"); sn.close(); } }

Step-by-Step Explanation

  1. Import Scanner: import java.util.Scanner; - Imports the Scanner class for reading user input.
  2. Create Scanner Object: Scanner sn = new Scanner(System.in); - Creates a Scanner object to read input from the console.
  3. Prompt User: System.out.print("Enter your name: "); - Displays a message asking for input (without moving to a new line).
  4. Read Input: String name = sn.nextLine(); - Reads the entire line of text entered by the user and stores it in the variable 'name'.
  5. Print Greeting: System.out.println("Hello, " + name + "!"); - Concatenates strings and prints the greeting message.
  6. Close Scanner: sn.close(); - Closes the Scanner to free up resources.

Question 3: Simple Addition

Problem Statement

Write a Java program that takes two numbers as input and prints their sum.

Sample Input and Output

Sample Input 1:
10
20

Sample Output 1:
Sum: 30

Sample Input 2:
45
55

Sample Output 2:
Sum: 100
Show Solution

Solution (Java Program)

import java.util.Scanner; public class SimpleAddition { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int num1 = sn.nextInt(); System.out.print("Enter second number: "); int num2 = sn.nextInt(); int sum = num1 + num2; System.out.println("Sum: " + sum); sn.close(); } }

Step-by-Step Explanation

  1. Create Scanner: Initialize Scanner object to read user input.
  2. Read First Number: int num1 = sn.nextInt(); - Reads an integer from the user and stores it in num1.
  3. Read Second Number: int num2 = sn.nextInt(); - Reads another integer and stores it in num2.
  4. Calculate Sum: int sum = num1 + num2; - Adds the two numbers and stores the result in the sum variable.
  5. Display Result: Prints the sum to the console.
  6. Close Scanner: Always close the Scanner when done to prevent resource leaks.

Question 4: Display Multiple Lines

Problem Statement

Write a Java program that displays your name, age, and city on separate lines.

Sample Input and Output

Sample Input 1:
Alice
25
New York

Sample Output 1:
Name: Alice
Age: 25
City: New York

Sample Input 2:
Bob
30
London

Sample Output 2:
Name: Bob
Age: 30
City: London
Show Solution

Solution (Java Program)

import java.util.Scanner; public class DisplayInfo { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter your name: "); String name = sn.nextLine(); System.out.print("Enter your age: "); int age = sn.nextInt(); sn.nextLine(); // Consume newline System.out.print("Enter your city: "); String city = sn.nextLine(); System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("City: " + city); sn.close(); } }

Step-by-Step Explanation

  1. Read Name: String name = sn.nextLine(); - Reads the entire line for the name.
  2. Read Age: int age = sn.nextInt(); - Reads an integer for age.
  3. Consume Newline: sn.nextLine(); - This is important! After reading an integer, we need to consume the leftover newline character.
  4. Read City: String city = sn.nextLine(); - Reads the city name.
  5. Display Information: Uses three separate println statements to display each piece of information on a new line.
  6. Close Scanner: Clean up resources by closing the Scanner.

Question 5: Simple Calculator (Addition and Multiplication)

Problem Statement

Write a Java program that takes two numbers as input and displays both their sum and product.

Sample Input and Output

Sample Input 1:
5
3

Sample Output 1:
Sum: 8
Product: 15

Sample Input 2:
12
4

Sample Output 2:
Sum: 16
Product: 48
Show Solution

Solution (Java Program)

import java.util.Scanner; public class SimpleCalculator { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); int num1 = sn.nextInt(); System.out.print("Enter second number: "); int num2 = sn.nextInt(); int sum = num1 + num2; int product = num1 * num2; System.out.println("Sum: " + sum); System.out.println("Product: " + product); sn.close(); } }

Step-by-Step Explanation

  1. Initialize Scanner: Create Scanner object for reading input.
  2. Read Two Numbers: Use nextInt() to read two integer values from the user.
  3. Calculate Sum: int sum = num1 + num2; - Adds the two numbers using the + operator.
  4. Calculate Product: int product = num1 * num2; - Multiplies the two numbers using the * operator.
  5. Display Results: Print both the sum and product on separate lines.
  6. Close Scanner: Release the Scanner resource.
