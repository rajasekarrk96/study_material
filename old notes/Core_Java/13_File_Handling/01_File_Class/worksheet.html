Question 1: Check Read and Write Permissions

Problem Statement

Write a Java program to check if a file or directory has read and write permission. Use the File class methods to determine the access permissions.

Solution

import java.io.File; class _01_ { public static void main(String[] args) { File f = new File("test.txt"); System.out.println("Readable: " + f.canRead()); System.out.println("Writable: " + f.canWrite()); } }

Sample Output

Readable: true Writable: true (or) Readable: false Writable: false (if file doesn't exist or no permissions)
Key Concepts:
  • File Class: Represents file and directory pathnames
  • canRead(): Tests whether application can read the file
  • canWrite(): Tests whether application can modify the file
  • canExecute(): Tests whether application can execute the file
Additional Methods:
  • setReadable(boolean) - Set read permission
  • setWritable(boolean) - Set write permission
  • setExecutable(boolean) - Set execute permission

Question 2: Get Last Modification Date and Time

Problem Statement

Write a Java program to get the last modification date and time of a file. Display the timestamp in a readable format using the Date class.

Solution

import java.io.File; import java.util.Date; class _02_ { public static void main(String[] args) { File f = new File("test.txt"); Date d = new Date(f.lastModified()); System.out.println("Last Modified: " + d); } }

Sample Output

Last Modified: Tue Jan 07 14:30:25 IST 2026 (or) Last Modified: Thu Jan 01 00:00:00 IST 1970 (if file doesn't exist, returns 0L)

Alternative Approaches

// Approach 1: Using SimpleDateFormat import java.text.SimpleDateFormat; SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy HH:mm:ss"); String formattedDate = sdf.format(new Date(f.lastModified())); System.out.println("Last Modified: " + formattedDate); // Approach 2: Using Java 8+ (Files and Instant) import java.nio.file.*; import java.time.Instant; Path path = Paths.get("test.txt"); Instant instant = Files.getLastModifiedTime(path).toInstant(); System.out.println("Last Modified: " + instant); // Approach 3: Get milliseconds directly long millis = f.lastModified(); System.out.println("Timestamp: " + millis + " ms");
Key Concepts:
  • lastModified(): Returns time in milliseconds since epoch (Jan 1, 1970)
  • Date Class: Converts milliseconds to readable date
  • Epoch Time: Number of milliseconds since January 1, 1970, 00:00:00 GMT
Use Cases: File synchronization, backup systems, cache validation

Question 3: Get File Size in Bytes, KB, MB

Problem Statement

Write a Java program to get file size in bytes, kilobytes (KB), and megabytes (MB). Convert the file size from bytes to larger units for better readability.

Solution

import java.io.File; class _03_ { public static void main(String[] args) { File f = new File("test.txt"); long bytes = f.length(); double kb = bytes / 1024.0; double mb = kb / 1024.0; System.out.println("Bytes: " + bytes); System.out.println("KB: " + kb); System.out.println("MB: " + mb); } }

Sample Output

Bytes: 2048 KB: 2.0 MB: 0.001953125 (for a 2KB file)

Enhanced Version with Formatting

import java.io.File; import java.text.DecimalFormat; class FileSizeFormatted { public static void main(String[] args) { File f = new File("test.txt"); long bytes = f.length(); double kb = bytes / 1024.0; double mb = kb / 1024.0; double gb = mb / 1024.0; DecimalFormat df = new DecimalFormat("#.##"); System.out.println("Bytes: " + bytes); System.out.println("KB: " + df.format(kb)); System.out.println("MB: " + df.format(mb)); System.out.println("GB: " + df.format(gb)); // Smart formatting System.out.println("\nBest Format: " + getReadableSize(bytes)); } static String getReadableSize(long bytes) { if (bytes < 1024) return bytes + " B"; if (bytes < 1024 * 1024) return (bytes / 1024.0) + " KB"; if (bytes < 1024 * 1024 * 1024) return (bytes / (1024.0 * 1024)) + " MB"; return (bytes / (1024.0 * 1024 * 1024)) + " GB"; } }
Key Concepts:
  • length(): Returns file size in bytes (returns 0 if file doesn't exist)
  • Unit Conversion: 1 KB = 1024 bytes, 1 MB = 1024 KB, 1 GB = 1024 MB
  • Floating Point: Use 1024.0 (not 1024) for decimal results
Storage Units:
  • 1 Byte (B) = 8 bits
  • 1 Kilobyte (KB) = 1,024 bytes
  • 1 Megabyte (MB) = 1,024 KB = 1,048,576 bytes
  • 1 Gigabyte (GB) = 1,024 MB

Question 4: Store File Content Line by Line into Array

Problem Statement

Write a Java program to store text file content line by line into an array. Read each line from the file and store it in an ArrayList for processing.

Solution

import java.io.*; import java.util.ArrayList; class _04_ { public static void main(String[] args) { ArrayList lines = new ArrayList<>(); try (BufferedReader br = new BufferedReader(new FileReader("test.txt"))) { String line; while ((line = br.readLine()) != null) { lines.add(line); } } catch (Exception e) { System.out.println("Error reading file"); } for (String s : lines) { System.out.println(s); } } }

Sample Output

Line 1 content Line 2 content Line 3 content ... (or) Error reading file (if file doesn't exist)

Alternative Approaches

// Approach 1: Using Files.readAllLines() (Java 7+) import java.nio.file.*; import java.util.List; List lines = Files.readAllLines(Paths.get("test.txt")); for (String line : lines) { System.out.println(line); } // Approach 2: Using Scanner import java.util.*; import java.io.*; ArrayList lines = new ArrayList<>(); try (Scanner sc = new Scanner(new File("test.txt"))) { while (sc.hasNextLine()) { lines.add(sc.nextLine()); } } // Approach 3: Convert to String Array String[] linesArray = lines.toArray(new String[0]);
Key Concepts:
  • BufferedReader: Efficient reading of text files
  • FileReader: Reads character files
  • readLine(): Reads a line of text, returns null at EOF
  • try-with-resources: Automatically closes resources
Why BufferedReader? More efficient than FileReader alone due to buffering

Question 5: Append Text to Existing File

Problem Statement

Write a Java program to append text to an existing file without overwriting its current content. Use FileWriter with append mode enabled.

Solution

import java.io.FileWriter; class _05_ { public static void main(String[] args) { try (FileWriter fw = new FileWriter("test.txt", true)) { fw.write("\nThis text is appended."); System.out.println("Text appended successfully"); } catch (Exception e) { System.out.println("Error writing file"); } } }

Sample Output

Text appended successfully

FileWriter Modes

// Mode 1: Overwrite (default) FileWriter fw = new FileWriter("test.txt"); // Deletes existing content and writes new content // Mode 2: Append FileWriter fw = new FileWriter("test.txt", true); // Keeps existing content and adds new content at the end // Example: Appending multiple lines try (FileWriter fw = new FileWriter("test.txt", true)) { fw.write("Line 1\n"); fw.write("Line 2\n"); fw.write("Line 3\n"); } // Using BufferedWriter for better performance import java.io.*; try (BufferedWriter bw = new BufferedWriter(new FileWriter("test.txt", true))) { bw.write("Appended line"); bw.newLine(); // Platform-independent newline }
Key Concepts:
  • FileWriter(file, true): Second parameter enables append mode
  • write(): Writes string to file
  • \n: Newline character (use bw.newLine() for platform independence)
  • Auto-close: try-with-resources ensures file is closed
Important: Always close FileWriter to flush data to disk

Question 6: Check if Path is File or Directory

Problem Statement

Write a Java program to check if given pathname is a directory or a file. Use File class methods to determine the type of the path.

Solution

import java.io.File; class _06_ { public static void main(String[] args) { File f = new File("test.txt"); if (f.isFile()) System.out.println("It is a file"); else if (f.isDirectory()) System.out.println("It is a directory"); else System.out.println("Path does not exist"); } }

Sample Output

It is a file (or) It is a directory (or) Path does not exist

Enhanced Version with More Details

import java.io.File; class FileInfo { public static void main(String[] args) { File f = new File("test.txt"); System.out.println("Path: " + f.getPath()); System.out.println("Absolute Path: " + f.getAbsolutePath()); System.out.println("Exists: " + f.exists()); if (f.exists()) { System.out.println("Is File: " + f.isFile()); System.out.println("Is Directory: " + f.isDirectory()); System.out.println("Is Hidden: " + f.isHidden()); System.out.println("Name: " + f.getName()); System.out.println("Parent: " + f.getParent()); if (f.isDirectory()) { String[] files = f.list(); System.out.println("Contains " + files.length + " items"); } } } } // List all files in a directory File dir = new File("."); if (dir.isDirectory()) { File[] files = dir.listFiles(); for (File file : files) { System.out.println((file.isDirectory() ? "[DIR] " : "[FILE] ") + file.getName()); } }
Key Concepts:
  • isFile(): Returns true if path is a file
  • isDirectory(): Returns true if path is a directory
  • exists(): Returns true if file/directory exists
  • Both return false: If path doesn't exist
Additional File Methods:
  • getName() - Get file/directory name
  • getPath() - Get path string
  • getAbsolutePath() - Get full path
  • getParent() - Get parent directory
  • list() - List directory contents (names)
  • listFiles() - List directory contents (File objects)
