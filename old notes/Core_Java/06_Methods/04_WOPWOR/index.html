Methods Without Parameters Without Return (WOPWOR)

Methods Without Parameters Without Return

Methods that neither accept input nor return values.

What is WOPWOR?

WOPWOR = Without Parameters Without Return

These methods don't accept parameters and don't return values. They perform standalone actions.

  • No input parameters
  • No return value (void)
  • Perform self-contained actions

Syntax

public void methodName() {
    // Method body
}

Example: Simple Actions

public class Greeter {
    public void sayHello() {
        System.out.println("Hello, World!");
    }
    
    public void displayMenu() {
        System.out.println("1. Start");
        System.out.println("2. Exit");
    }
    
    public static void main(String[] args) {
        Greeter g = new Greeter();
        g.sayHello();
        g.displayMenu();
    }
}
Hello, World!

2. Exit
