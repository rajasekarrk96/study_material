HashMap

HashMap in Java

Store key-value pairs efficiently with HashMap.

What is HashMap?

HashMap stores data in key-value pairs. Each key is unique and maps to exactly one value.

  • Key-value pairs
  • Fast lookup by key
  • No duplicate keys allowed
  • Allows one null key and multiple null values

Common Methods

Method Description
put(K, V) Add key-value pair
get(K) Get value by key
remove(K) Remove by key
containsKey(K) Check if key exists
keySet() Get all keys
values() Get all values

Example: Student Grades

import java.util.HashMap;

public class HashMapDemo {
    public static void main(String[] args) {
        HashMap grades = new HashMap<>();
        
        // Add key-value pairs
        grades.put("Alice", 95);
        grades.put("Bob", 87);
        grades.put("Charlie", 92);
        
        // Get value
        System.out.println("Alice's grade: " + grades.get("Alice"));
        
        // Check if key exists
        if (grades.containsKey("Bob")) {
            System.out.println("Bob's grade: " + grades.get("Bob"));
        }
        
        // Iterate
        for (String name : grades.keySet()) {
            System.out.println(name + ": " + grades.get(name));
        }
        
        // Size
        System.out.println("Total students: " + grades.size());
    }
}
Alice's grade: 95
Bob's grade: 87
Alice: 95
Bob: 87
Charlie: 92
Total students: 3
