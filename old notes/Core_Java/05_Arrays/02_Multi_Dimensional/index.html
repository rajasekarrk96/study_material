Multi-Dimensional Arrays in Java

Work with 2D and 3D arrays for complex data structure representation.

What are Multi-Dimensional Arrays?

A multi-dimensional array is an array of arrays. The most common is the 2D array (like a table with rows and columns). Java also supports 3D arrays and higher dimensions.

  • 2D Array: Array with rows and columns (like a matrix)
  • 3D Array: Array of 2D arrays (like a cube)
  • Used for matrices, tables, grids, game boards, etc.
💡 Visualization: Think of a 2D array as a spreadsheet with rows and columns.

Declaration and Syntax

// 2D Array Declaration
dataType[][] arrayName;

// Examples
int[][] matrix;
String[][] table;
double[][] grid;

// 3D Array
int[][][] cube;

Initialization Methods

// Method 1: Direct initialization
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Method 2: Using new keyword
int[][] arr = new int[3][4];  // 3 rows, 4 columns

// Method 3: Jagged array (different column sizes)
int[][] jagged = new int[3][];
jagged[0] = new int[2];
jagged[1] = new int[4];
jagged[2] = new int[3];

Accessing Elements

public class ArrayAccess {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        
        // Access element at row 1, column 2
        System.out.println("Element: " + matrix[1][2]);  // Output: 6
        
        // Modify element
        matrix[0][0] = 10;
        
        // Get dimensions
        System.out.println("Rows: " + matrix.length);
        System.out.println("Columns: " + matrix[0].length);
    }
}
Element: 6
Rows: 3
Columns: 3

Example: Matrix Operations

public class MatrixExample {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        
        // Print matrix
        System.out.println("Matrix:");
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
        
        // Calculate sum of all elements
        int sum = 0;
        for (int[] row : matrix) {
            for (int num : row) {
                sum += num;
            }
        }
        System.out.println("\nSum of all elements: " + sum);
        
        // Find diagonal sum
        int diagonalSum = 0;
        for (int i = 0; i < matrix.length; i++) {
            diagonalSum += matrix[i][i];
        }
        System.out.println("Diagonal sum: " + diagonalSum);
    }
}

Sum of all elements: 45
Diagonal sum: 15
