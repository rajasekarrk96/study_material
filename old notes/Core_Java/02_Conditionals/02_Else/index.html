Else Statement

Else Statement in Java

Execute alternative code when the if condition is false.

Syntax

if (condition) {
    // Code executes if condition is true
} else {
    // Code executes if condition is false
}
  • The else block executes only when the if condition is false
  • You cannot have else without an if
  • Only one block (either if or else) will execute

Example: Check Even or Odd

public class ElseExample {
    public static void main(String[] args) {
        int number = 7;
        
        if (number % 2 == 0) {
            System.out.println(number + " is Even");
        } else {
            System.out.println(number + " is Odd");
        }
        
        // Another example: Age check
        int age = 15;
        
        if (age >= 18) {
            System.out.println("You are eligible to vote");
        } else {
            System.out.println("You are not eligible to vote");
        }
    }
}
7 is Odd
You are not eligible to vote

Control Flow

Start
  ↓
Check Condition
  ↓
  ├─ True  → Execute if block → End
  └─ False → Execute else block → End
💡 Key Point: The else block provides a default action when the condition fails, ensuring your program always has a response.
