Question 1: Extract All Words Ending With "ing"

Problem Statement

Write a Java program to extract all words from a sentence that end with the suffix "ing". Use word boundaries \b to ensure you match only the end of the word.

Example Sentence: "Singing and dancing are inspiring activities for coding."

Solution

import java.util.regex.*; public class ExtractWords { public static void main(String[] args) { String text = "Singing and dancing are inspiring activities for coding."; // Pattern: any word characters ending with 'ing' at a word boundary Pattern p = Pattern.compile("\\b\\w+ing\\b"); Matcher m = p.matcher(text); System.out.println("Words ending with -ing:"); while (m.find()) { System.out.println(m.group()); } } }

Sample Output

Words ending with -ing: Singing dancing inspiring coding

Regex Pattern Explanation

\\b\\w+ing\\b Breakdown: - \\b : Start of a word boundary - \\w+ : One or more word characters - ing : Literal suffix "ing" - \\b : End of a word boundary
Key Concepts:
  • Word Boundary \b: Matches the position between a word character and a non-word character (or start/end of string).
  • Matcher.find(): Scans the input sequence for the next subsequence that matches the pattern.
  • Matcher.group(): Returns the input subsequence matched by the previous match.

Question 2: Check if a String Starts and Ends with Only Vowels

Problem Statement

Write a Java program to check if a single word starts and ends with a vowel (a, e, i, o, u). Use ^ and $ anchors.

Solution

public class VowelAnchor { public static void main(String[] args) { String[] words = {"apple", "orange", "java", "idea", "eye"}; // ^[aeiouAEIOU] matches starting vowel // .* matches any characters in between // [aeiouAEIOU]$ matches ending vowel String regex = "^[aeiouAEIOU].*[aeiouAEIOU]$"; for (String word : words) { if (word.matches(regex)) { System.out.println(word + " matches"); } else { System.out.println(word + " does not match"); } } } }

Sample Output

apple matches orange matches java does not match idea matches eye matches

Regex Pattern Explanation

^[aeiouAEIOU].*[aeiouAEIOU]$ Breakdown: - ^ : Start of string anchor - [aeiouAEIOU] : Vowel character class (matches first char) - .* : Any characters (matches middle) - [aeiouAEIOU] : Vowel character class (matches last char) - $ : End of string anchor
Key Concepts:
  • Caret (^): Anchors the match to the beginning of the string.
  • Dollar ($): Anchors the match to the end of the string.
