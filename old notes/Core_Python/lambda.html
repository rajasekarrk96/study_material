Python Lambda

Introduction

A lambda function is a small anonymous function.

A lambda function can take any number of arguments, but can only have one expression.

Syntax

The syntax of a lambda function is:

lambda arguments : expression

The expression is executed and the result is returned:

x = lambda a : a + 10
print(x(5))

Lambda functions can take any number of arguments:

x = lambda a, b : a * b
print(x(5, 6))

x = lambda a, b, c : a + b + c
print(x(5, 6, 2))

Why Use Lambda Functions?

The power of lambda is better shown when you use them as an anonymous function inside another function.

Say you have a function definition that takes one argument, and that argument will be multiplied with an unknown number:

def myfunc(n):
    return lambda a : a * n

mydoubler = myfunc(2)

print(mydoubler(11))

Usage with Map and Filter

Lambda functions are frequently used with filter() and map() functions. 

# Using with filter()
numbers = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)

# Using with map()
squares = list(map(lambda x: x**2, numbers))
print(squares)

Worksheet

1. Write a lambda function to add 15 to a given number.

add_15 = lambda x: x + 15
print(add_15(10))

2. Write a lambda function that multiplies argument x with argument y.

multiply = lambda x, y: x * y
print(multiply(5, 6))

3. Write a lambda function to square a number.

square = lambda x: x ** 2
print(square(8))

4. Write a lambda function to check if a number is even.

is_even = lambda x: x % 2 == 0
print(is_even(10))
print(is_even(7))

5. Write a lambda function to reverse a string.

reverse_str = lambda s: s[::-1]
print(reverse_str("Python"))

6. Write a lambda function that returns the maximum of two numbers.

max_num = lambda x, y: x if x > y else y
print(max_num(10, 20))

7. Write a program to sort a list of tuples based on the second element using lambda.

data = [(1, 5), (3, 2), (2, 8)]
data.sort(key=lambda x: x[1])
print(data)

8. Write a program to filter words that start with 'a' using lambda.

words = ["apple", "banana", "avocado", "cherry"]
starts_with_a = list(filter(lambda x: x.startswith('a'), words))
print(starts_with_a)

9. Write a program to square every number in a list using map and lambda.

nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
print(squared)

10. Write a lambda function to check if a string contains the letter 'a'.

has_a = lambda s: 'a' in s
print(has_a("Hello"))
print(has_a("Apple"))
