Pattern Printing in Python

Introduction

Pattern printing is a great way to understand nested loops (loops inside loops).

Usually, we use two loops:

  • Outer loop: Controls the number of rows.
  • Inner loop: Controls the number of columns (stars or numbers in each row).

1. Square Pattern

Prints a square of stars.

n = 5
for i in range(n):
    for j in range(n):
        print("*", end=" ")
    print()

Output:

* * * * *
* * * * *
* * * * *
* * * * *
* * * * *

2. Right-Angled Triangle

Prints a triangle where stars increase in each row.

n = 5
for i in range(n):
    for j in range(i + 1):
        print("*", end=" ")
    print()

Output:

*
* *
* * *
* * * *
* * * * *

3. Inverted Right-Angled Triangle

Prints a triangle where stars decrease in each row.

n = 5
for i in range(n):
    for j in range(n - i):
        print("*", end=" ")
    print()

Output:

* * * * *
* * * *
* * *
* *
*

4. Pyramid Pattern

Prints a centered pyramid.

n = 5
for i in range(n):
    # Print spaces
    for j in range(n - i - 1):
        print(" ", end=" ")
    # Print stars
    for k in range(2 * i + 1):
        print("*", end=" ")
    print()

Output:

        *
      * * *
    * * * * *
  * * * * * * *
* * * * * * * * *

5. Diamond Pattern

Combines a pyramid and an inverted pyramid.

n = 5

# Top half
for i in range(n):
    print("  " * (n - i - 1) + "* " * (2 * i + 1))

# Bottom half
for i in range(n - 2, -1, -1):
    print("  " * (n - i - 1) + "* " * (2 * i + 1))

Worksheet

1. Write a program to print a 5x5 Matrix of numbers.

n = 5
for i in range(n):
    for j in range(n):
        print(j + 1, end=" ")
    print()

2. Write a program to print a number triangle (1, 12, 123...).

n = 5
for i in range(1, n + 1):
    for j in range(1, i + 1):
        print(j, end=" ")
    print()

3. Write a program to print the same number in each row (1, 22, 333...).

n = 5
for i in range(1, n + 1):
    for j in range(i):
        print(i, end=" ")
    print()

4. Write a program to print an inverted number triangle (12345, 1234...).

n = 5
for i in range(n, 0, -1):
    for j in range(1, i + 1):
        print(j, end=" ")
    print()

5. Write a program to print Floyd's Triangle (continuous numbers).

n = 5
num = 1
for i in range(1, n + 1):
    for j in range(1, i + 1):
        print(num, end=" ")
        num += 1
    print()

6. Write a program to print a Reverse Pyramid of stars.

n = 5
for i in range(n, 0, -1):
    # Spaces
    print("  " * (n - i), end="")
    # Stars
    print("* " * (2 * i - 1))

7. Write a program to print an Alphabet Triangle (A, BB, CCC...).

n = 5
for i in range(n):
    for j in range(i + 1):
        print(chr(65 + i), end=" ")
    print()

8. Write a program to print a Hollow Square pattern.

n = 5
for i in range(n):
    for j in range(n):
        if i == 0 or i == n - 1 or j == 0 or j == n - 1:
            print("*", end=" ")
        else:
            print(" ", end=" ")
    print()

9. Write a program to print an X shape using stars.

n = 5
for i in range(n):
    for j in range(n):
        if i == j or i + j == n - 1:
            print("*", end=" ")
        else:
            print(" ", end=" ")
    print()

10. Write a program to print Pascal's Triangle.

n = 5
for i in range(n):
    # Spaces
    print(" " * (n - i), end="")
    
    # Calculate coefficients
    val = 1
    for j in range(i + 1):
        print(val, end=" ")
        val = val * (i - j) // (j + 1)
    print()
