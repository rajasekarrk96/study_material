Python Sets

Introduction

Sets are used to store multiple items in a single variable.

A set is a collection which is unordered, unchangeable*, and unindexed.

* Note: Set items are unchangeable, but you can remove items and add new items.

Sets are written with curly brackets.

thisset = {"apple", "banana", "cherry"}
print(thisset)

Set Items

  • Unordered: Unordered means that the items in a set do not have a defined order. Set items can appear in a different order every time you use them, and cannot be referred to by index or key.
  • Unchangeable: Set items are unchangeable, meaning that we cannot change the items after the set has been created.
  • Duplicates Not Allowed: Sets cannot have two items with the same value.

Access Items

You cannot access items in a set by referring to an index or a key.

But you can loop through the set items using a for loop, or ask if a specified value is present in a set, by using the in keyword.

thisset = {"apple", "banana", "cherry"}

for x in thisset:
  print(x)

print("banana" in thisset) # True

Add Items

Add Items

To add one item to a set use the add() method.

thisset = {"apple", "banana", "cherry"}
thisset.add("orange")

Add Sets

To add items from another set into the current set, use the update() method.

thisset = {"apple", "banana", "cherry"}
tropical = {"pineapple", "mango", "papaya"}

thisset.update(tropical)

Create Set

A set is created by placing all the items (elements) inside curly braces {}, separated by commas.

Syntax:

set_name = {item1, item2, item3, ...}

Example:

thisset = {"apple", "banana", "cherry"}
thisset.remove("banana")

Remove Items

To remove an item in a set, use the remove(), or the discard() method. 

thisset = {"apple", "banana", "cherry"}
thisset.remove("banana")

Note: If the item to remove does not exist, remove() will raise an error, while discard() will NOT raise an error.

Loop Sets

You can loop through the set items by using a for loop:

thisset = {"apple", "banana", "cherry"}

for x in thisset:
  print(x)

Join Sets

There are several ways to join two or more sets in Python.

  • The union() method returns a new set with all items from both sets.
  • The update() method inserts all the items from one set into another.
set1 = {"a", "b" , "c"}
set2 = {1, 2, 3}

set3 = set1.union(set2)
print(set3)

Set Methods

Method Description
add() Adds an element to the set
clear() Removes all the elements from the set
copy() Returns a copy of the set
difference() Returns a set containing the difference between two or more sets
discard() Remove the specified item
intersection() Returns a set, that is the intersection of two other sets
issubset() Returns whether another set contains this set or not
pop() Removes an element from the set
remove() Removes the specified element
union() Return a set containing the union of sets
update() Update the set with the union of this set and others

Worksheet

1. Write a Python program to create a set.

x = set()
print(x)
n = {0, 1, 2, 3, 4}
print(n)

2. Write a Python program to add member(s) in a set.

color_set = set()
color_set.add("Red")
print(color_set)
# Add multiple items
color_set.update(["Blue", "Green"])
print(color_set)

3. Write a Python program to remove an item from a set if it is present in the set.

num_set = {0, 1, 2, 3, 4, 5}
num_set.discard(4)
print(num_set)

4. Write a Python program to create an intersection of sets.

setx = set(["green", "blue"])
sety = set(["blue", "yellow"])
setz = setx & sety
print(setz)

5. Write a Python program to create a union of sets.

setx = set(["green", "blue"])
sety = set(["blue", "yellow"])
seta = setx | sety
print(seta)

6. Write a Python program to create set difference.

setx = set(["green", "blue"])
sety = set(["blue", "yellow"])
setz = setx - sety
print(setz)

7. Write a Python program to create a symmetric difference.

setx = set(["green", "blue"])
sety = set(["blue", "yellow"])
setz = setx ^ sety
print(setz)

8. Write a Python program to check if a set is a subset of another set.

x = {"a", "b", "c", "d", "e"}
y = {"c", "d"}
print("Is y a subset of x?")
print(y.issubset(x))

9. Write a Python program to find maximum and the minimum value in a set.

setn = {5, 10, 3, 15, 2, 20}
print(max(setn))
print(min(setn))

10. Write a Python program to clear a set.

setc = {"Red", "Green", "Black"}
setc.clear()
print(setc)
