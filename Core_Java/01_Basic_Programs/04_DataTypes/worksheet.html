Question 1: Primitive Data Types Demonstration

Problem Statement

Write a Java program that demonstrates all primitive data types by taking input for each type and displaying them.

Sample Input and Output

Sample Input 1:
25
5000000000
3.14
9.99
A
true

Sample Output 1:
byte: 25
long: 5000000000
float: 3.14
double: 9.99
char: A
boolean: true

Sample Input 2:
100
9876543210
2.5
7.89
Z
false

Sample Output 2:
byte: 100
long: 9876543210
float: 2.5
double: 7.89
char: Z
boolean: false
Show Solution

Solution (Java Program)

import java.util.Scanner; public class PrimitiveTypes { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter byte value: "); byte b = sn.nextByte(); System.out.print("Enter long value: "); long l = sn.nextLong(); System.out.print("Enter float value: "); float f = sn.nextFloat(); System.out.print("Enter double value: "); double d = sn.nextDouble(); System.out.print("Enter char value: "); char c = sn.next().charAt(0); System.out.print("Enter boolean value: "); boolean bool = sn.nextBoolean(); System.out.println("byte: " + b); System.out.println("long: " + l); System.out.println("float: " + f); System.out.println("double: " + d); System.out.println("char: " + c); System.out.println("boolean: " + bool); sn.close(); } }

Step-by-Step Explanation

  1. Byte: 8-bit integer, range -128 to 127. Read using nextByte().
  2. Long: 64-bit integer for large numbers. Read using nextLong().
  3. Float: 32-bit floating point number. Read using nextFloat().
  4. Double: 64-bit floating point for more precision. Read using nextDouble().
  5. Char: Single character. Read using next().charAt(0).
  6. Boolean: true or false value. Read using nextBoolean().

Question 2: Type Casting - Widening

Problem Statement

Write a Java program that demonstrates automatic type casting (widening) by converting an int to a double.

Sample Input and Output

Sample Input 1:
42

Sample Output 1:
Integer value: 42
Double value: 42.0

Sample Input 2:
100

Sample Output 2:
Integer value: 100
Double value: 100.0
Show Solution

Solution (Java Program)

import java.util.Scanner; public class WideningCast { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter an integer: "); int intValue = sn.nextInt(); // Automatic widening conversion double doubleValue = intValue; System.out.println("Integer value: " + intValue); System.out.println("Double value: " + doubleValue); sn.close(); } }

Step-by-Step Explanation

  1. Read Integer: Get an integer value from the user.
  2. Automatic Conversion: double doubleValue = intValue; - Java automatically converts int to double (widening).
  3. Widening Cast: Smaller data types can be automatically converted to larger types without data loss.
  4. Conversion Order: byte → short → int → long → float → double
  5. No Explicit Cast Needed: This conversion happens automatically.
  6. Display Results: Show both the original integer and converted double value.

Question 3: Type Casting - Narrowing

Problem Statement

Write a Java program that demonstrates explicit type casting (narrowing) by converting a double to an int.

Sample Input and Output

Sample Input 1:
9.99

Sample Output 1:
Double value: 9.99
Integer value: 9

Sample Input 2:
15.75

Sample Output 2:
Double value: 15.75
Integer value: 15
Show Solution

Solution (Java Program)

import java.util.Scanner; public class NarrowingCast { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a double value: "); double doubleValue = sn.nextDouble(); // Explicit narrowing conversion int intValue = (int) doubleValue; System.out.println("Double value: " + doubleValue); System.out.println("Integer value: " + intValue); sn.close(); } }

Step-by-Step Explanation

  1. Read Double: Get a double value from the user.
  2. Explicit Cast: (int) doubleValue - We must explicitly cast double to int using (int).
  3. Data Loss: The decimal part is truncated (not rounded), so 9.99 becomes 9.
  4. Narrowing Cast: Converting larger types to smaller types requires explicit casting.
  5. Potential Issues: If the value is too large for int, it will overflow.
  6. Syntax: Use parentheses with the target type: (targetType) value

Question 4: String to Primitive Conversion

Problem Statement

Write a Java program that converts String inputs to different primitive types (int, double, boolean) and performs calculations.

Sample Input and Output

Sample Input 1:
10
5.5
true

Sample Output 1:
Integer: 10
Double: 5.5
Boolean: true
Sum: 15.5

Sample Input 2:
20
3.2
false

Sample Output 2:
Integer: 20
Double: 3.2
Boolean: false
Sum: 23.2
Show Solution

Solution (Java Program)

import java.util.Scanner; public class StringConversion { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter integer as string: "); String strInt = sn.nextLine(); int num = Integer.parseInt(strInt); System.out.print("Enter double as string: "); String strDouble = sn.nextLine(); double decimal = Double.parseDouble(strDouble); System.out.print("Enter boolean as string: "); String strBool = sn.nextLine(); boolean flag = Boolean.parseBoolean(strBool); double sum = num + decimal; System.out.println("Integer: " + num); System.out.println("Double: " + decimal); System.out.println("Boolean: " + flag); System.out.println("Sum: " + sum); sn.close(); } }

Step-by-Step Explanation

  1. Read Strings: Read all inputs as strings using nextLine().
  2. Parse Integer: Integer.parseInt() converts String to int.
  3. Parse Double: Double.parseDouble() converts String to double.
  4. Parse Boolean: Boolean.parseBoolean() converts String to boolean.
  5. Calculate Sum: Add the integer and double values.
  6. Wrapper Classes: Integer, Double, and Boolean are wrapper classes that provide parsing methods.

Question 5: Data Type Size and Range

Problem Statement

Write a Java program that displays the size (in bytes) and range of int and long data types, then takes input and shows if it fits in each type.

Sample Input and Output

Sample Input 1:
2147483647

Sample Output 1:
int size: 4 bytes
long size: 8 bytes
Your number: 2147483647
Fits in int: true
Fits in long: true

Sample Input 2:
9223372036854775807

Sample Output 2:
int size: 4 bytes
long size: 8 bytes
Your number: 9223372036854775807
Fits in int: false
Fits in long: true
Show Solution

Solution (Java Program)

import java.util.Scanner; public class DataTypeSize { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.println("int size: " + Integer.BYTES + " bytes"); System.out.println("long size: " + Long.BYTES + " bytes"); System.out.print("Enter a number: "); long number = sn.nextLong(); boolean fitsInInt = (number >= Integer.MIN_VALUE && number <= Integer.MAX_VALUE); boolean fitsInLong = true; // Always true if we can read it as long System.out.println("Your number: " + number); System.out.println("Fits in int: " + fitsInInt); System.out.println("Fits in long: " + fitsInLong); sn.close(); } }

Step-by-Step Explanation

  1. Display Sizes: Use Integer.BYTES and Long.BYTES to show size in bytes.
  2. Read Number: Read as long to handle large numbers.
  3. Check Int Range: Compare with Integer.MIN_VALUE and Integer.MAX_VALUE.
  4. Int Range: -2,147,483,648 to 2,147,483,647 (4 bytes)
  5. Long Range: -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 (8 bytes)
  6. Practical Use: Choose appropriate data types based on the range of values you need to store.
