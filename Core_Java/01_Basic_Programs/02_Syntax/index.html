Java Syntax Basics

Learn the fundamental syntax rules and structure that every Java program must follow.

Syntax Rules

Java syntax consists of rules that define how Java programs are written. Following these rules is essential for code to compile and run.

  • Every statement must end with a semicolon (;)
  • Java is case-sensitive (Main and main are different)
  • Curly braces {} define code blocks
  • Indentation improves readability (not required but recommended)
  • Comments are ignored by the compiler
✓ Valid vs Invalid:
✓ System.out.println("Hello");
✗ System.out.println("Hello") // Missing semicolon
✓ public class Main {}
✗ public class main {} // Wrong - should be Main

Program Structure

// Import statements (at top)
import java.util.Scanner;

// Class declaration (must match filename)
public class HelloWorld {
    
    // Main method (entry point)
    public static void main(String[] args) {
        // Executable statements
        System.out.println("Hello, World!");
    }
    
    // Other methods
    public void displayMessage() {
        System.out.println("This is a method");
    }
}

Key Components:

  • import - Brings in external classes
  • public class - Declares the main class
  • public static void main() - Program entry point
  • String[] args - Command line arguments

Comments in Java

Comments are non-executable lines used to explain code and improve readability.

Type Syntax Example
Single-line // // This is a comment
Multi-line /* */ /* This is a
multi-line comment */
Documentation /** */ /** JavaDoc comment */
// Used for API documentation		 
public class Comments {
    public static void main(String[] args) {
        // Single-line comment
        int age = 25;  // Variable for age
        
        /* Multi-line comment
           This can span multiple lines
           Used for complex explanations */
        System.out.println(age);
        
        /**
         * Documentation comment
         * Used for generating JavaDoc
         * @param args Command line arguments
         */
    }
}

Packages

Packages organize Java classes into namespaces. They prevent naming conflicts and organize code logically.

// Package declaration (must be first line)
package com.example.application;

// Now this class belongs to the package
public class MyClass {
    public static void main(String[] args) {
        System.out.println("Hello from package!");
    }
}

Package Naming Convention:

  • Use reverse domain names: com.company.project
  • All lowercase letters
  • Separated by dots (.)
  • Matches folder structure
💡 Example: For company "google.com", use "com.google.projectname"

Import Statements

Import statements bring classes from other packages into your program.

// Import specific class
import java.util.Scanner;

// Import all classes from a package
import java.util.*;

// Import multiple specific classes
import java.io.File;
import java.io.FileReader;

public class ImportExample {
    public static void main(String[] args) {
        // Scanner is available because of import
        Scanner sc = new Scanner(System.in);
        
        // ArrayList is available due to import *
        java.util.ArrayList list = new java.util.ArrayList();
    }
}

Common Packages:

Package Purpose
java.lang Core Java classes (auto-imported)
java.util Utility classes like Scanner, ArrayList
java.io Input/Output classes
java.sql Database classes

ASCII Notes

ASCII values are important when working with characters in Java code. Understanding ASCII helps in character comparisons and conversions within your programs.

Quick ASCII Reference:

Control Characters:  32=SPACE  48-57=Numbers 0-9
Uppercase:           65-90 = A-Z
Lowercase:           97-122 = a-z
Special:             33-47, 58-64, 91-96, 123-126

Character to ASCII Conversion in Java:

public class ASCIIConversion {
    public static void main(String[] args) {
        // Method 1: Direct casting char to int
        char ch = 'A';
        System.out.println("Char 'A' = " + (int)ch);
        
        // Method 2: Casting other characters
        System.out.println("Char 'Z' = " + (int)'Z');
        System.out.println("Char 'a' = " + (int)'a');
        System.out.println("Char '0' = " + (int)'0');
        System.out.println("Char ' ' = " + (int)' ');
    }
}
Char 'A' = 65
Char 'Z' = 90
Char 'a' = 97
Char '0' = 48
Char ' ' = 32
✓ Useful Fact: Difference between uppercase and lowercase is 32. 'A'=65, 'a'=97 (65+32=97)
