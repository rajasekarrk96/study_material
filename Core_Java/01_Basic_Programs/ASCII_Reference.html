📋 Quick Reference

Control Characters
0-31, 127: Non-printable special characters for device control and formatting
Printable ASCII
32-126: Standard Latin characters, digits, and symbols
Extended ASCII
128-256: Accented letters and special symbols (varies by encoding)
Common ASCII Values
Space: 32 | '0': 48 | 'A': 65 | 'a': 97

🔄 Java Conversion Examples

Get ASCII value from character:
char ch = 'A';
int ascii = (int) ch; // Result: 65
Get character from ASCII value:
int ascii = 65;
char ch = (char) ascii; // Result: 'A'
Loop through ASCII values:
for (int i = 65; i <= 90; i++) {
  System.out.println((char) i);
} // Prints: A B C ... Z
Using Character class methods:
int ascii = Character.getNumericValue('5'); // Result: 5
boolean isDigit = Character.isDigit('5'); // Result: true
boolean isLetter = Character.isLetter('A'); // Result: true
