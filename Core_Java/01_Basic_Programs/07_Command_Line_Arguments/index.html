Command Line Arguments in Java

Learn how to pass arguments to Java programs from the command line and process them within your code.

What are Command Line Arguments?

Command line arguments are parameters passed to a Java program when it is executed from the command line or terminal. They allow you to provide input to your program without hardcoding values or using Scanner.

  • Passed as strings in the String[] args parameter of the main method
  • Accessed using array indexing: args[0], args[1], etc.
  • Useful for configuration, file paths, or runtime parameters
  • All arguments are received as String type
💡 Note: The name "args" is conventional but you can use any valid identifier. However, it must be a String array.

Syntax

public class ClassName {
    public static void main(String[] args) {
        // args[0] - first argument
        // args[1] - second argument
        // args.length - number of arguments
    }
}

Running with arguments:

// Compilation
javac ClassName.java

// Execution with arguments
java ClassName arg1 arg2 arg3

Example: Basic Command Line Arguments

public class CommandLineDemo {
    public static void main(String[] args) {
        // Check if arguments are provided
        if (args.length == 0) {
            System.out.println("No arguments provided!");
            return;
        }
        
        // Display number of arguments
        System.out.println("Number of arguments: " + args.length);
        
        // Display all arguments
        System.out.println("\nArguments received:");
        for (int i = 0; i < args.length; i++) {
            System.out.println("Argument " + i + ": " + args[i]);
        }
    }
}

Execution:

java CommandLineDemo Hello World Java
Number of arguments: 3

Arguments received:
Argument 0: Hello
Argument 1: World
Argument 2: Java

Type Conversion

Since all command line arguments are strings, you need to convert them to other data types when needed. 

public class Calculator {
    public static void main(String[] args) {
        if (args.length < 2) {
            System.out.println("Usage: java Calculator num1 num2");
            return;
        }
        
        // Convert String to int
        int num1 = Integer.parseInt(args[0]);
        int num2 = Integer.parseInt(args[1]);
        
        // Perform calculations
        System.out.println("Sum: " + (num1 + num2));
        System.out.println("Difference: " + (num1 - num2));
        System.out.println("Product: " + (num1 * num2));
        
        if (num2 != 0) {
            System.out.println("Division: " + (num1 / num2));
        }
    }
}

Execution:

java Calculator 10 5
Sum: 15
Difference: 5
Product: 50
Division: 2
✓ Common Conversions:
• Integer.parseInt(str) - String to int
• Double.parseDouble(str) - String to double
• Boolean.parseBoolean(str) - String to boolean
• Long.parseLong(str) - String to long
