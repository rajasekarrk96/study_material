Java Operators

Learn about different types of operators in Java and how to use them effectively in your programs.

Section 1: Arithmetic Operators

Overview

Arithmetic operators are used to perform basic mathematical operations on numeric values. These operators work with integers and floating-point numbers. Java supports five basic arithmetic operators: addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).

Operators Table

Operator Symbol Description Example Result
Addition + Adds two numbers 10 + 5 15
Subtraction - Subtracts second from first 10 - 5 5
Multiplication * Multiplies two numbers 10 * 5 50
Division / Divides first by second 10 / 5 2
Modulus % Returns remainder 10 % 3 1

Code Example

public class ArithmeticOperators { public static void main(String[] args) { int a = 20; int b = 10; // Addition int add = a + b; System.out.println("Addition: " + add); // Subtraction int sub = a - b; System.out.println("Subtraction: " + sub); // Multiplication int mul = a * b; System.out.println("Multiplication: " + mul); // Division int div = a / b; System.out.println("Division: " + div); // Modulus int mod = a % b; System.out.println("Modulus: " + mod); // With floating-point double x = 20.5; double y = 10.2; System.out.println("Float Division: " + (x / y)); } }

Output

Addition: 30 Subtraction: 10 Multiplication: 200 Division: 2 Modulus: 0 Float Division: 2.0098039215686274
Note: When dividing two integers, the result is truncated (decimal part is removed). Use floating-point numbers for decimal results. The modulus operator (%) works only with integers.

Section 2: Relational Operators

Overview

Relational operators are used to compare two values. They return a boolean result (true or false). Relational operators are commonly used in conditional statements to make decisions based on comparisons.

Operators Table

Operator Symbol Description Example Result
Equal to == Checks if equal 10 == 10 true
Not equal to != Checks if not equal 10 != 5 true
Less than < Checks if less than 5 < 10 true
Greater than > Checks if greater than 10 > 5 true
Less than or equal <= Checks if less than or equal 10 <= 10 true
Greater than or equal >= Checks if greater than or equal 10 >= 5 true

Code Example

public class RelationalOperators { public static void main(String[] args) { int a = 15; int b = 10; System.out.println("a == b: " + (a == b)); System.out.println("a != b: " + (a != b)); System.out.println("a < b: " + (a < b)); System.out.println("a > b: " + (a > b)); System.out.println("a <= b: " + (a <= b)); System.out.println("a >= b: " + (a >= b)); // Comparing with same values int x = 20; int y = 20; System.out.println("\nWhen x == y:"); System.out.println("x == y: " + (x == y)); System.out.println("x != y: " + (x != y)); System.out.println("x >= y: " + (x >= y)); System.out.println("x <= y: " + (x <= y)); } }

Output

a == b: false a != b: true a < b: false a > b: true a <= b: false a >= b: true When x == y: x == y: true x != y: false x >= y: true x <= y: true
Example: Use relational operators to validate user input or check conditions before performing operations.

Section 3: Logical Operators

Overview

Logical operators are used to combine two or more boolean conditions. They return a boolean result and are essential for creating complex conditional statements. Java has three main logical operators: AND (&&), OR (||), and NOT (!).

Operators Table

Operator Symbol Description Example Result
AND && True if both conditions are true true && false false
OR || True if at least one condition is true true || false true
NOT ! Reverses the boolean value !true false

Code Example

public class LogicalOperators { public static void main(String[] args) { int age = 25; int income = 50000; boolean hasLicense = true; // AND operator (&&) if (age >= 18 && hasLicense) { System.out.println("Can drive: YES"); } // OR operator (||) if (age < 18 || income < 30000) { System.out.println("Eligible for subsidy: YES"); } // NOT operator (!) if (!hasLicense) { System.out.println("Need to get a license"); } else { System.out.println("Have a valid license"); } // Complex condition if ((age >= 21 && income >= 50000) || (hasLicense && age >= 18)) { System.out.println("All conditions satisfied"); } // Truth table demonstration System.out.println("\nTruth Table for AND:"); System.out.println("true && true = " + (true && true)); System.out.println("true && false = " + (true && false)); System.out.println("false && true = " + (false && true)); System.out.println("false && false = " + (false && false)); } }

Output

Can drive: YES Eligible for subsidy: YES Have a valid license All conditions satisfied Truth Table for AND: true && true = true true && false = false false && true = false false && false = false
Note: The && operator uses short-circuit evaluation: if the first condition is false, it doesn't evaluate the second. Similarly, || stops evaluating once it finds a true condition.

Section 4: Assignment Operators

Overview

Assignment operators are used to assign values to variables. The simple assignment operator is =, but Java also provides compound assignment operators that combine assignment with other operations, making code more concise.

Operators Table

Operator Description Example Equivalent to
= Assign value a = 10 Assign 10 to a
+= Add and assign a += 5 a = a + 5
-= Subtract and assign a -= 5 a = a - 5
*= Multiply and assign a *= 5 a = a * 5
/= Divide and assign a /= 5 a = a / 5
%= Modulus and assign a %= 5 a = a % 5

Code Example

public class AssignmentOperators { public static void main(String[] args) { int a = 20; // Simple assignment int b = a; System.out.println("b = " + b); // += operator a += 10; // a = a + 10 System.out.println("After a += 10: " + a); // -= operator a -= 5; // a = a - 5 System.out.println("After a -= 5: " + a); // *= operator a *= 2; // a = a * 2 System.out.println("After a *= 2: " + a); // /= operator a /= 4; // a = a / 4 System.out.println("After a /= 4: " + a); // %= operator a %= 3; // a = a % 3 System.out.println("After a %= 3: " + a); // With strings String str = "Hello"; str += " World"; System.out.println(str); } }

Output

b = 20 After a += 10: 30 After a -= 5: 25 After a *= 2: 50 After a /= 4: 12 After a %= 3: 0 Hello World
Example: Compound assignment operators are useful in loops where you need to update variable values repeatedly.

Section 5: Ternary Operator

Overview

The ternary operator (? :) is Java's only operator that takes three operands. It provides a concise way to make decisions and assign values conditionally. It's also called the conditional operator. The syntax is: condition ? value_if_true : value_if_false

Syntax

variable = (condition) ? value_if_true : value_if_false;

Code Example

public class TernaryOperator { public static void main(String[] args) { // Basic ternary operator int age = 20; String status = (age >= 18) ? "Adult" : "Minor"; System.out.println("Status: " + status); // Finding maximum of two numbers int a = 50; int b = 30; int max = (a > b) ? a : b; System.out.println("Maximum: " + max); // Finding minimum of two numbers int min = (a < b) ? a : b; System.out.println("Minimum: " + min); // Nested ternary operator int marks = 75; String grade = (marks >= 90) ? "A" : (marks >= 80) ? "B" : (marks >= 70) ? "C" : (marks >= 60) ? "D" : "F"; System.out.println("Grade: " + grade); // Comparing strings String name = "Java"; String language = (name.equals("Java")) ? "Object-Oriented" : "Other"; System.out.println(name + " is: " + language); // With arithmetic int number = 7; String evenOdd = (number % 2 == 0) ? "Even" : "Odd"; System.out.println(number + " is " + evenOdd); } }

Output

Status: Adult Maximum: 50 Minimum: 30 Grade: C Java is: Object-Oriented 7 is Odd
Note: The ternary operator is useful for simple conditions. For complex decision-making, use if-else statements. Avoid deeply nested ternary operators as they reduce code readability.
Example: Use the ternary operator to assign values based on conditions or to return values from methods concisely.
