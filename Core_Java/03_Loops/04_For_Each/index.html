For-Each Loop (Enhanced For Loop)

For-Each Loop in Java

Simplified way to iterate through arrays and collections without using index.

Syntax

for (dataType variable : arrayOrCollection) {
    // Code to execute for each element
}
  • dataType: Type of elements in the array/collection
  • variable: Temporary variable holding current element
  • arrayOrCollection: The array or collection to iterate
💡 Also Called: Enhanced for loop or for-each loop. Introduced in Java 5.

Example: Iterate Arrays

public class ForEachExample {
    public static void main(String[] args) {
        // Array of numbers
        int[] numbers = {10, 20, 30, 40, 50};
        
        System.out.println("Using for-each loop:");
        for (int num : numbers) {
            System.out.println("Number: " + num);
        }
        
        // Array of strings
        String[] fruits = {"Apple", "Banana", "Orange", "Mango"};
        
        System.out.println("\nFruits:");
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
        
        // Calculate sum
        int sum = 0;
        for (int num : numbers) {
            sum += num;
        }
        System.out.println("\nSum: " + sum);
    }
}

Advantages & Limitations

✓ Advantages:
• Cleaner and more readable code
• No index management needed
• Eliminates off-by-one errors
• Works with arrays and collections
⚠️ Limitations:
• Cannot modify array elements
• Cannot access index directly
• Only forward iteration
• Cannot iterate multiple arrays simultaneously
