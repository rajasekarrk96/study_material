Methods Without Parameters With Return (WOPWR)

Methods Without Parameters With Return

Methods that don't accept input but return a value.

What is WOPWR?

WOPWR = Without Parameters With Return

These methods don't accept any parameters but return a value to the caller. They often work with class variables or generate values.

  • No input parameters needed
  • Return a value using return statement
  • Useful for getters, calculations, or value generation

Syntax

public returnType methodName() {
    // Method body
    return value;
}

Example 1: Getter Methods

public class Student {
    private String name = "Alice";
    private int age = 20;
    
    public String getName() {
        return name;
    }
    
    public int getAge() {
        return age;
    }
    
    public static void main(String[] args) {
        Student s = new Student();
        System.out.println("Name: " + s.getName());
        System.out.println("Age: " + s.getAge());
    }
}
Name: Alice
Age: 20

Example 2: Random Number Generator

public class RandomGen {
    public int getRandomNumber() {
        return (int)(Math.random() * 100);
    }
    
    public static void main(String[] args) {
        RandomGen gen = new RandomGen();
        System.out.println("Random: " + gen.getRandomNumber());
    }
}

Example 3: Current Time

public class TimeUtil {
    public String getCurrentTime() {
        return java.time.LocalTime.now().toString();
    }
    
    public static void main(String[] args) {
        TimeUtil util = new TimeUtil();
        System.out.println("Time: " + util.getCurrentTime());
    }
}
