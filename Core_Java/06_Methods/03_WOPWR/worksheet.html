Question 1: Random Number Generator

Write a Java method named getRandomNumber that returns a random integer between 1 and 100. Call it from main() and print the result.
class _01_ { static int getRandomNumber() { return (int) (Math.random() * 100) + 1; } public static void main(String[] args) { int random = getRandomNumber(); System.out.println("Random Number: " + random); } }
Random Number: 42 (Example)

Question 2: Current Date String

Write a Java method named getCurrentDate that returns the current date as a String using java.time.LocalDate.now().toString().
class _02_ { static String getCurrentDate() { return java.time.LocalDate.now().toString(); } public static void main(String[] args) { System.out.println("Current Date: " + getCurrentDate()); } }
Current Date: 2026-01-08 (Example)
