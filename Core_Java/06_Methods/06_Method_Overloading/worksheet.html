Question 1: Area Calculator

Overload a method named calculateArea to calculate the area of a Circle (1 parameter: radius) and a Rectangle (2 parameters: width and height).
class _01_ { // Circle static double calculateArea(double radius) { return Math.PI * radius * radius; } // Rectangle static double calculateArea(double width, double height) { return width * height; } public static void main(String[] args) { System.out.println("Circle Area: " + calculateArea(5.0)); System.out.println("Rectangle Area: " + calculateArea(10.0, 5.0)); } }
Circle Area: 78.53981633974483 Rectangle Area: 50.0

Question 2: Data Printer

Overload a method named printValue to print an int, a String, and a boolean with appropriate labels.
class _02_ { static void printValue(int val) { System.out.println("Integer Value: " + val); } static void printValue(String val) { System.out.println("String Value: " + val); } static void printValue(boolean val) { System.out.println("Boolean Value: " + val); } public static void main(String[] args) { printValue(100); printValue("Hello Overloading"); printValue(true); } }
Integer Value: 100 String Value: Hello Overloading Boolean Value: true
