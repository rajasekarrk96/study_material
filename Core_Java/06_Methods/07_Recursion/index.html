Recursion

Recursion in Java

Write methods that call themselves to solve complex problems elegantly.

What is Recursion?

Recursion is when a method calls itself. It breaks down complex problems into smaller, simpler sub-problems.

  • Method calls itself
  • Must have a base case (stopping condition)
  • Each recursive call should move toward the base case
💡 Key Components: Base case (stop condition) + Recursive case (method calls itself)

Base Case - Critical!

Without a base case, recursion will continue infinitely and cause StackOverflowError.

// BAD - No base case!
public void infiniteRecursion() {
    infiniteRecursion();  // Never stops!
}

// GOOD - Has base case
public void countdown(int n) {
    if (n == 0) return;  // Base case
    System.out.println(n);
    countdown(n - 1);    // Recursive case
}

Example 1: Factorial

public class Factorial {
    public int factorial(int n) {
        // Base case
        if (n == 0 || n == 1) {
            return 1;
        }
        // Recursive case
        return n * factorial(n - 1);
    }
    
    public static void main(String[] args) {
        Factorial f = new Factorial();
        System.out.println("5! = " + f.factorial(5));
        System.out.println("4! = " + f.factorial(4));
    }
}
5! = 120
4! = 24

Example 2: Fibonacci

public class Fibonacci {
    public int fib(int n) {
        if (n <= 1) return n;  // Base case
        return fib(n - 1) + fib(n - 2);  // Recursive case
    }
    
    public static void main(String[] args) {
        Fibonacci f = new Fibonacci();
        for (int i = 0; i < 8; i++) {
            System.out.print(f.fib(i) + " ");
        }
    }
}
0 1 1 2 3 5 8 13

Example 3: Sum of Numbers

public class RecursiveSum {
    public int sum(int n) {
        if (n == 1) return 1;  // Base case
        return n + sum(n - 1);  // Recursive case
    }
    
    public static void main(String[] args) {
        RecursiveSum rs = new RecursiveSum();
        System.out.println("Sum 1 to 5: " + rs.sum(5));
        System.out.println("Sum 1 to 10: " + rs.sum(10));
    }
}
Sum 1 to 5: 15
Sum 1 to 10: 55
