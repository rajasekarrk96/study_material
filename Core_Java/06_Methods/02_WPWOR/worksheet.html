Question 1: Check Positive, Negative, or Zero

Write a Java method named checkNumber that accepts an integer and checks if it is negative, positive, or zero. Print the result directly from the method.
import java.util.Scanner; class _01_ { static void checkNumber(int n) { if (n > 0) System.out.println("Positive"); else if (n < 0) System.out.println("Negative"); else System.out.println("Zero"); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a number: "); int n = sn.nextInt(); checkNumber(n); sn.close(); } }
Enter a number: -5 Negative

Question 2: Smallest of Three Numbers

Write a Java method named printSmallest that accepts three integers and prints the smallest number among them.
import java.util.Scanner; class _02_ { static void printSmallest(int a, int b, int c) { int smallest = Math.min(a, Math.min(b, c)); System.out.println("Smallest: " + smallest); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter three numbers: "); int a = sn.nextInt(); int b = sn.nextInt(); int c = sn.nextInt(); printSmallest(a, b, c); sn.close(); } }
Enter three numbers: 12 5 8 Smallest: 5

Question 3: Greatest Common Divisor (GCD)

Write a Java method named printGCD that accepts two integers and prints their Greatest Common Divisor (GCD).
import java.util.Scanner; class _03_ { static void printGCD(int a, int b) { int originalA = a, originalB = b; while (b != 0) { int t = b; b = a % b; a = t; } System.out.println("GCD of " + originalA + " and " + originalB + " is: " + a); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter two numbers: "); int x = sn.nextInt(); int y = sn.nextInt(); printGCD(x, y); sn.close(); } }
Enter two numbers: 12 18 GCD of 12 and 18 is: 6
