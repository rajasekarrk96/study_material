Question 1: Sum of Variable Arguments

Write a Java method named sumAll that accepts a variable number of integers and returns their sum. Call it with different numbers of arguments.
class _01_ { static int sumAll(int... numbers) { int total = 0; for (int n : numbers) { total += n; } return total; } public static void main(String[] args) { System.out.println("Sum 1: " + sumAll(1, 2, 3)); System.out.println("Sum 2: " + sumAll(10, 20, 30, 40, 50)); System.out.println("Sum 3: " + sumAll()); } }
Sum 1: 6 Sum 2: 150 Sum 3: 0

Question 2: Print Names

Write a Java method named printNames that accepts a variable number of String arguments and prints each one on a new line.
class _02_ { static void printNames(String... names) { for (String name : names) { System.out.println("Hello, " + name); } } public static void main(String[] args) { printNames("Alice", "Bob", "Charlie"); } }
Hello, Alice Hello, Bob Hello, Charlie
