Question 1: BankAccount - Private, Protected, Public

Problem Statement

Create a BankAccount class with the following attributes and methods: • A private attribute balance • A protected attribute accountNumber • A public method deposit to add money to the balance • A public method withdraw to subtract money from the balance • A protected method displayAccountNumber to print the account number

Solution

import java.util.Scanner; class _01_ { private double balance; protected String accountNumber; public void deposit(double amt) { balance += amt; } public void withdraw(double amt) { balance -= amt; } protected void displayAccountNumber() { System.out.println("Account Number: " + accountNumber); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); _01_ acc = new _01_(); System.out.print("Enter account number: "); acc.accountNumber = sn.next(); System.out.print("Enter deposit amount: "); acc.deposit(sn.nextDouble()); System.out.print("Enter withdraw amount: "); acc.withdraw(sn.nextDouble()); acc.displayAccountNumber(); System.out.println("Balance = " + acc.balance); sn.close(); } }

Sample Output

Enter account number: ACC001 Enter deposit amount: 5000 Enter withdraw amount: 1000 Account Number: ACC001 Balance = 4000.0
Key Concepts:
  • Private: balance is accessible only within the class
  • Protected: accountNumber accessible in same package and subclasses
  • Public Methods: deposit() and withdraw() provide controlled access
  • Encapsulation: Balance protected from direct external modification
Access Levels: private < default < protected < public

Question 2: Employee - Private, Package-Private, Public

Problem Statement

Create an Employee class with the following attributes and methods: • A private attribute salary • A package-private attribute name • A public method setSalary to set the salary • A public method getSalary to get the salary • A package-private method displayEmployeeDetails to print the employee's name and salary

Solution

import java.util.Scanner; class _02_ { private double salary; String name; // package-private public void setSalary(double s) { salary = s; } public double getSalary() { return salary; } void displayEmployeeDetails() { // package-private System.out.println("Name: " + name); System.out.println("Salary: " + salary); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); _02_ emp = new _02_(); System.out.print("Enter name: "); emp.name = sn.next(); System.out.print("Enter salary: "); emp.setSalary(sn.nextDouble()); emp.displayEmployeeDetails(); sn.close(); } }

Sample Output

Enter name: Ravi Enter salary: 50000 Name: Ravi Salary: 50000.0
Key Concepts:
  • Private: salary is hidden from external access
  • Package-Private (Default): name accessible within same package only
  • Public Getters/Setters: Controlled access to private salary
  • No Modifier: displayEmployeeDetails() has default (package-private) access
Note: Package-private means no access modifier is specified

Question 3: Student - Private, Protected, Public

Problem Statement

Create a Student class with the following attributes and methods: • A private attribute grade • A protected attribute studentID • A public method setGrade to set the grade • A public method getGrade to get the grade • A protected method displayStudentID to print the student ID

Solution

import java.util.Scanner; class _03_ { private char grade; protected int studentID; public void setGrade(char g) { grade = g; } public char getGrade() { return grade; } protected void displayStudentID() { System.out.println("Student ID: " + studentID); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); _03_ s = new _03_(); System.out.print("Enter student ID: "); s.studentID = sn.nextInt(); System.out.print("Enter grade: "); s.setGrade(sn.next().charAt(0)); s.displayStudentID(); System.out.println("Grade: " + s.getGrade()); sn.close(); } }

Sample Output

Enter student ID: 101 Enter grade: A Student ID: 101 Grade: A
Key Concepts:
  • Private Attribute: grade cannot be accessed directly
  • Protected Attribute: studentID accessible in subclasses
  • Public Methods: setGrade() and getGrade() provide controlled access
  • Protected Method: displayStudentID() accessible in subclasses
Design: Sensitive data (grade) is private, ID is protected for inheritance

Question 4: LibraryBook - Private, Public, Protected

Problem Statement

Create a LibraryBook class with the following attributes and methods: • A private attribute isbn • A public attribute title • A protected attribute author • A public method setIsbn to set the ISBN • A public method getIsbn to get the ISBN • A protected method displayAuthor to print the author

Solution

import java.util.Scanner; class _04_ { private String isbn; public String title; protected String author; public void setIsbn(String i) { isbn = i; } public String getIsbn() { return isbn; } protected void displayAuthor() { System.out.println("Author: " + author); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); _04_ book = new _04_(); System.out.print("Enter title: "); book.title = sn.next(); System.out.print("Enter author: "); book.author = sn.next(); System.out.print("Enter ISBN: "); book.setIsbn(sn.next()); System.out.println("Title: " + book.title); System.out.println("ISBN: " + book.getIsbn()); book.displayAuthor(); sn.close(); } }

Sample Output

Enter title: Java Enter author: Gosling Enter ISBN: 978-0134685991 Title: Java ISBN: 978-0134685991 Author: Gosling
Key Concepts:
  • Private: isbn requires getter/setter for access
  • Public: title can be accessed and modified directly
  • Protected: author accessible in same package and subclasses
  • Mixed Access: Different attributes have different visibility levels
Warning: Public attributes bypass encapsulation - use with caution

Question 5: BankAccount - Transfer Between Accounts

Problem Statement

Create a BankAccount class with the following attributes and methods: • A private attribute accountNumber • A private attribute balance • A protected attribute accountHolderName • A public method setAccountNumber to set the account number • A public method getAccountNumber to get the account number • A public method setBalance to set the balance • A public method getBalance to get the balance • A protected method displayAccountHolderName to print the account holder name • A public method deposit to add money to the balance • A public method withdraw to subtract money from the balance • A public method transfer to transfer money to another BankAccount

Solution

import java.util.Scanner; class _05_ { private String accountNumber; private double balance; protected String accountHolderName; public void setAccountNumber(String acc) { accountNumber = acc; } public String getAccountNumber() { return accountNumber; } public void setBalance(double b) { balance = b; } public double getBalance() { return balance; } protected void displayAccountHolderName() { System.out.println("Account Holder: " + accountHolderName); } public void deposit(double amt) { balance += amt; } public void withdraw(double amt) { balance -= amt; } public void transfer(_05_ other, double amt) { this.withdraw(amt); other.deposit(amt); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); _05_ acc1 = new _05_(); _05_ acc2 = new _05_(); System.out.print("Enter first account holder name: "); acc1.accountHolderName = sn.next(); System.out.print("Enter first account number: "); acc1.setAccountNumber(sn.next()); System.out.print("Enter initial balance: "); acc1.setBalance(sn.nextDouble()); System.out.print("Enter second account holder name: "); acc2.accountHolderName = sn.next(); System.out.print("Enter second account number: "); acc2.setAccountNumber(sn.next()); System.out.print("Enter transfer amount: "); double amt = sn.nextDouble(); acc1.transfer(acc2, amt); acc1.displayAccountHolderName(); System.out.println("Balance = " + acc1.getBalance()); acc2.displayAccountHolderName(); System.out.println("Balance = " + acc2.getBalance()); sn.close(); } }

Sample Output

Enter first account holder name: Alice Enter first account number: ACC001 Enter initial balance: 10000 Enter second account holder name: Bob Enter second account number: ACC002 Enter transfer amount: 2000 Account Holder: Alice Balance = 8000.0 Account Holder: Bob Balance = 2000.0
Key Concepts:
  • Private Attributes: accountNumber and balance are fully encapsulated
  • Protected Attribute: accountHolderName accessible in subclasses
  • Public Methods: All transaction methods are publicly accessible
  • Object Interaction: transfer() method works with another BankAccount object
Real-World: Demonstrates proper encapsulation in banking systems
