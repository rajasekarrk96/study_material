Super Keyword

Super Keyword in Java

Access parent class members and constructors using the super keyword.

What is the Super Keyword?

The super keyword in Java is a reference variable used to refer to the immediate parent class object. It is used to access parent class members (variables, methods, constructors) from the child class.

Uses of Super Keyword:
Use Case Syntax Purpose
Access Parent Variable super.variableName Access parent class variable when child has same name
Call Parent Constructor super(arguments) Invoke parent class constructor from child
Call Parent Method super.methodName() Call parent class method (especially when overridden)
💡 Important: super() must be the first statement in a child class constructor.

1. Access Parent Class Variables

When both parent and child classes have variables with the same name, use super to access the parent class variable.

Example 1: Accessing Parent Variable
class Animal {
    String color = "White";
}

class Dog extends Animal {
    String color = "Brown";
    
    void displayColors() {
        System.out.println("Dog color: " + color);           // Child class variable
        System.out.println("Animal color: " + super.color);  // Parent class variable
    }
}

public class SuperVariableDemo {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.displayColors();
    }
}
Dog color: Brown
Animal color: White
Example 2: Differentiating Variables
class Vehicle {
    int speed = 60;
}

class Car extends Vehicle {
    int speed = 120;
    
    void showSpeed() {
        System.out.println("Car speed: " + speed);
        System.out.println("Vehicle speed: " + super.speed);
        System.out.println("Speed difference: " + (speed - super.speed));
    }
}

public class SpeedDemo {
    public static void main(String[] args) {
        Car car = new Car();
        car.showSpeed();
    }
}
Car speed: 120
Vehicle speed: 60
Speed difference: 60

2. Call Parent Class Constructor

Use super() to call the parent class constructor. This is useful for initializing parent class members.

Example 1: Basic Constructor Call
class Person {
    String name;
    int age;
    
    Person(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("Person constructor called");
    }
}

class Student extends Person {
    int rollNumber;
    
    Student(String name, int age, int roll) {
        super(name, age);  // Call parent constructor
        this.rollNumber = roll;
        System.out.println("Student constructor called");
    }
    
    void display() {
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Roll Number: " + rollNumber);
    }
}

public class ConstructorDemo {
    public static void main(String[] args) {
        Student student = new Student("Alice", 20, 101);
        student.display();
    }
}
Person constructor called
Student constructor called
Name: Alice
Age: 20
Roll Number: 101
Example 2: Constructor Chaining
class Shape {
    String color;
    
    Shape() {
        color = "Red";
        System.out.println("Shape() - Default color: " + color);
    }
    
    Shape(String color) {
        this.color = color;
        System.out.println("Shape(String) - Color: " + color);
    }
}

class Circle extends Shape {
    double radius;
    
    Circle(double radius) {
        super();  // Calls Shape()
        this.radius = radius;
    }
    
    Circle(String color, double radius) {
        super(color);  // Calls Shape(String)
        this.radius = radius;
    }
    
    void display() {
        System.out.println("Circle - Color: " + color + ", Radius: " + radius);
    }
}

public class ChainDemo {
    public static void main(String[] args) {
        System.out.println("Creating Circle 1:");
        Circle c1 = new Circle(5.0);
        c1.display();
        
        System.out.println("\nCreating Circle 2:");
        Circle c2 = new Circle("Blue", 7.5);
        c2.display();
    }
}
Creating Circle 1:
Shape() - Default color: Red
Circle - Color: Red, Radius: 5.0

Creating Circle 2:
Shape(String) - Color: Blue
Circle - Color: Blue, Radius: 7.5
⚠️ Important: If super() is not explicitly called, Java automatically inserts super() as the first statement in the child constructor.

3. Call Parent Class Methods

Use super.methodName() to call parent class methods, especially when they are overridden in the child class.

Example 1: Calling Overridden Method
class Animal {
    void eat() {
        System.out.println("Animal is eating");
    }
}

class Dog extends Animal {
    @Override
    void eat() {
        super.eat();  // Call parent method
        System.out.println("Dog is eating dog food");
    }
}

public class MethodDemo {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.eat();
    }
}
Animal is eating
Dog is eating dog food
Example 2: Extending Parent Functionality
class Employee {
    double salary = 50000;
    
    void calculateBonus() {
        System.out.println("Base bonus: " + (salary * 0.1));
    }
}

class Manager extends Employee {
    @Override
    void calculateBonus() {
        super.calculateBonus();  // Call parent method
        double managerBonus = salary * 0.2;
        System.out.println("Manager bonus: " + managerBonus);
        System.out.println("Total bonus: " + (salary * 0.3));
    }
}

public class BonusDemo {
    public static void main(String[] args) {
        Manager manager = new Manager();
        manager.calculateBonus();
    }
}
Base bonus: 5000.0
Manager bonus: 10000.0
Total bonus: 15000.0
Example 3: Method Resolution
class Parent {
    void display() {
        System.out.println("Parent display()");
    }
    
    void show() {
        System.out.println("Parent show()");
    }
}

class Child extends Parent {
    @Override
    void display() {
        System.out.println("Child display()");
        super.display();  // Call parent's display()
    }
    
    void callParentShow() {
        super.show();  // Call parent's show()
    }
}

public class ResolutionDemo {
    public static void main(String[] args) {
        Child child = new Child();
        child.display();
        System.out.println();
        child.callParentShow();
    }
}
Child display()
Parent display()

Parent show()

Practical Examples

Example 1: Bank Account Hierarchy
class Account {
    protected String accountNumber;
    protected double balance;
    
    Account(String accNum, double bal) {
        accountNumber = accNum;
        balance = bal;
    }
    
    void deposit(double amount) {
        balance += amount;
        System.out.println("Deposited: " + amount);
    }
    
    void displayBalance() {
        System.out.println("Balance: " + balance);
    }
}

class SavingsAccount extends Account {
    private double interestRate;
    
    SavingsAccount(String accNum, double bal, double rate) {
        super(accNum, bal);  // Initialize parent class members
        this.interestRate = rate;
    }
    
    @Override
    void deposit(double amount) {
        super.deposit(amount);  // Call parent deposit
        double interest = amount * interestRate / 100;
        balance += interest;
        System.out.println("Interest added: " + interest);
    }
    
    void displayDetails() {
        System.out.println("Account: " + accountNumber);
        super.displayBalance();  // Call parent method
        System.out.println("Interest Rate: " + interestRate + "%");
    }
}

public class BankDemo {
    public static void main(String[] args) {
        SavingsAccount acc = new SavingsAccount("SA1001", 1000, 5);
        acc.deposit(500);
        acc.displayDetails();
    }
}
Deposited: 500.0
Interest added: 25.0
Account: SA1001
Balance: 1525.0
Interest Rate: 5.0%
Comparison: this vs super
Feature this super

Method Call this.method() super.method()
Constructor Call this() - same class super() - parent class
