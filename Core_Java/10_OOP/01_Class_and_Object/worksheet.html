Question 1: Rectangle Class - Area & Perimeter

Problem Statement

Implement a Rectangle class in Java. The class should have two attributes for the width and height of the rectangle. It should also have methods to calculate the area and the perimeter of the rectangle. Provide a constructor to initialize the attributes.

Solution

import java.util.Scanner; class _01_ { double width, height; _01_(double w, double h) { width = w; height = h; } double area() { return width * height; } double perimeter() { return 2 * (width + height); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter width: "); double w = sn.nextDouble(); System.out.print("Enter height: "); double h = sn.nextDouble(); _01_ r = new _01_(w, h); System.out.println("Area = " + r.area()); System.out.println("Perimeter = " + r.perimeter()); sn.close(); } }

Sample Output

Enter width: 5 Enter height: 3 Area = 15.0 Perimeter = 16.0
Key Concepts:
  • Class Attributes: width and height store rectangle dimensions
  • Constructor: Initializes attributes when object is created
  • Methods: area() and perimeter() perform calculations
  • Object Creation: new keyword creates Rectangle instance
Formulas: Area = width × height, Perimeter = 2(width + height)

Question 2: Student Class - Details & Honor Roll

Problem Statement

Create a Student class that includes attributes for the student's name, ID, and GPA. The class should have methods to display the student's information and to check if the student is on the honor roll (GPA 3.5 or higher). Include a constructor to initialize the attributes.

Solution

import java.util.Scanner; class _02_ { String name; int id; double gpa; _02_(String n, int i, double g) { name = n; id = i; gpa = g; } void display() { System.out.println("Name: " + name); System.out.println("ID: " + id); System.out.println("GPA: " + gpa); } void honorRoll() { if (gpa >= 3.5) System.out.println("Honor Roll Student"); else System.out.println("Not in Honor Roll"); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter name: "); String name = sn.next(); System.out.print("Enter ID: "); int id = sn.nextInt(); System.out.print("Enter GPA: "); double gpa = sn.nextDouble(); _02_ s = new _02_(name, id, gpa); s.display(); s.honorRoll(); sn.close(); } }

Sample Output

Enter name: Alice Enter ID: 101 Enter GPA: 3.8 Name: Alice ID: 101 GPA: 3.8 Honor Roll Student
Key Concepts:
  • Multiple Attributes: name, id, and gpa store student data
  • Parameterized Constructor: Accepts three parameters for initialization
  • Display Method: Prints all student information
  • Conditional Logic: honorRoll() checks GPA threshold
Business Logic: Honor roll requires GPA ≥ 3.5

Question 3: Circle Class - Radius, Area & Circumference

Problem Statement

Create a Circle class with an attribute for the radius. The class should have methods to set and get the radius, and to calculate the area and circumference of the circle.

Solution

import java.util.Scanner; class _03_ { double radius; void setRadius(double r) { radius = r; } double getRadius() { return radius; } double area() { return 3.14 * radius * radius; } double circumference() { return 2 * 3.14 * radius; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter radius: "); double r = sn.nextDouble(); _03_ c = new _03_(); c.setRadius(r); System.out.println("Radius = " + c.getRadius()); System.out.println("Area = " + c.area()); System.out.println("Circumference = " + c.circumference()); sn.close(); } }

Sample Output

Enter radius: 5 Radius = 5.0 Area = 78.5 Circumference = 31.400000000000002
Key Concepts:
  • Getter/Setter: setRadius() and getRadius() control access to radius
  • Encapsulation: Radius accessed through methods, not directly
  • Calculation Methods: area() and circumference() use π ≈ 3.14
  • No Constructor: Default constructor used, radius set via setter
Formulas: Area = πr², Circumference = 2πr

Question 4: Static MathUtils Class

Problem Statement

Create a static MathUtils class that includes methods for common mathematical operations: addition, subtraction, multiplication, and division. Each method should take two parameters and return the result.

Solution

import java.util.Scanner; class _04_ { static double add(double a, double b) { return a + b; } static double sub(double a, double b) { return a - b; } static double mul(double a, double b) { return a * b; } static double div(double a, double b) { return a / b; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); double a = sn.nextDouble(); System.out.print("Enter second number: "); double b = sn.nextDouble(); System.out.println("Addition = " + add(a, b)); System.out.println("Subtraction = " + sub(a, b)); System.out.println("Multiplication = " + mul(a, b)); System.out.println("Division = " + div(a, b)); sn.close(); } }

Sample Output

Enter first number: 10 Enter second number: 5 Addition = 15.0 Subtraction = 5.0 Multiplication = 50.0 Division = 2.0
Key Concepts:
  • Static Methods: Called without creating object instance
  • Utility Class: Provides common operations without state
  • Method Parameters: Each method accepts two double values
  • Return Values: All methods return calculated results
Usage: Static methods called directly: add(a, b), no object needed

Question 5: Static DateUtils Class

Problem Statement

Create a static DateUtils class that includes methods to: • Get the current date and time • Calculate the difference in days between two dates • Check if a given year is a leap year

Solution

import java.time.LocalDate; import java.time.temporal.ChronoUnit; import java.util.Scanner; class _05_ { static void currentDateTime() { System.out.println("Current Date = " + LocalDate.now()); } static void dateDifference(LocalDate d1, LocalDate d2) { long days = ChronoUnit.DAYS.between(d1, d2); System.out.println("Difference in days = " + days); } static void leapYear(int year) { if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) System.out.println("Leap Year"); else System.out.println("Not a Leap Year"); } public static void main(String[] args) { Scanner sn = new Scanner(System.in); currentDateTime(); System.out.print("Enter first date (yyyy-mm-dd): "); LocalDate d1 = LocalDate.parse(sn.next()); System.out.print("Enter second date (yyyy-mm-dd): "); LocalDate d2 = LocalDate.parse(sn.next()); dateDifference(d1, d2); System.out.print("Enter year: "); int year = sn.nextInt(); leapYear(year); sn.close(); } }

Sample Output

Current Date = 2026-01-08 Enter first date (yyyy-mm-dd): 2026-01-01 Enter second date (yyyy-mm-dd): 2026-01-08 Difference in days = 7 Enter year: 2024 Leap Year
Key Concepts:
  • LocalDate API: Java's modern date/time handling
  • ChronoUnit: Calculates time differences between dates
  • Leap Year Logic: Divisible by 4, except centuries unless divisible by 400
  • Static Utility Methods: Date operations without object state
Leap Year Rule: (year % 4 == 0 && year % 100 != 0) || year % 400 == 0
