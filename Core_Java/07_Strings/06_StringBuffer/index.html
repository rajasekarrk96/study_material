StringBuffer

StringBuffer in Java

Mutable, thread-safe string class for efficient string manipulation.

What is StringBuffer?

StringBuffer is a mutable sequence of characters. Unlike String, StringBuffer can be modified without creating new objects.

  • Mutable - can be changed
  • Thread-safe - synchronized methods
  • Slower than StringBuilder but safe for multi-threading
💡 Use When: You need thread-safe mutable strings with frequent modifications.

Common Methods

StringBuffer sb = new StringBuffer("Hello");

// append() - add to end
sb.append(" World");
System.out.println(sb);  // Hello World

// insert() - add at position
sb.insert(5, ",");
System.out.println(sb);  // Hello, World

// delete() - remove characters
sb.delete(5, 6);
System.out.println(sb);  // Hello World

// reverse() - reverse string
sb.reverse();
System.out.println(sb);  // dlroW olleH

Example: Building String

public class StringBufferDemo {
    public static void main(String[] args) {
        StringBuffer sb = new StringBuffer();
        
        // Efficient string building
        for (int i = 1; i <= 5; i++) {
            sb.append(i).append(" ");
        }
        
        System.out.println("Numbers: " + sb);
        System.out.println("Length: " + sb.length());
        System.out.println("Capacity: " + sb.capacity());
    }
}
Numbers: 1 2 3 4 5
Length: 10
Capacity: 16
