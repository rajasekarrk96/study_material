StringBuilder - Practice

Practice Questions

Practice fast mutable string operations using the StringBuilder class.

Question 1: Concatenate Strings

Concatenate the string “World “ with string “Ending..” using StringBuilder.
class _01_ { public static void main(String[] args) { StringBuilder sb = new StringBuilder("World "); sb.append("Ending.."); System.out.println("Result: " + sb); } }
Result: World Ending..

Question 2: Reverse String

Write a Java program to reverse a given string “Palindrome” using StringBuilder.
class _02_ { public static void main(String[] args) { StringBuilder sb = new StringBuilder("Palindrome"); sb.reverse(); System.out.println("Reversed: " + sb); } }
Reversed: emordnilaP

Question 3: Insert Substring

Write a Java program to insert a substring “and apple” into a given string “ Mango is a fruit” at position 6 using StringBuilder.
class _03_ { public static void main(String[] args) { StringBuilder sb = new StringBuilder(" Mango is a fruit"); sb.insert(6, "and apple"); System.out.println("Modified: " + sb); } }
Modified: Mangoand apple is a fruit

Question 4: Delete Substring

Write a Java program to delete a specific substring “happiness and” from a given string “Plants gives happiness and joy” using StringBuilder.
class _04_ { public static void main(String[] args) { StringBuilder sb = new StringBuilder("Plants gives happiness and joy"); int start = sb.indexOf("happiness and"); int end = start + "happiness and".length(); if (start != -1) { sb.delete(start, end); } System.out.println("After deletion: " + sb); } }
After deletion: Plants gives joy

Question 5: Replace Substring

Write a Java program to replace a specific substring “World” with another substring “Everyone” in a given string “Hello, World!” using StringBuilder.
class _05_ { public static void main(String[] args) { StringBuilder sb = new StringBuilder("Hello, World!"); int start = sb.indexOf("World"); int end = start + "World".length(); if (start != -1) { sb.replace(start, end, "Everyone"); } System.out.println("Final String: " + sb); } }
Final String: Hello, Everyone!
