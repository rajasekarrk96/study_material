StringBuilder

StringBuilder in Java

Fast, mutable string class for single-threaded string manipulation.

What is StringBuilder?

StringBuilder is similar to StringBuffer but NOT thread-safe. It's faster because methods are not synchronized.

  • Mutable - can be changed
  • NOT thread-safe - no synchronization
  • Faster than StringBuffer
  • Preferred for single-threaded applications
💡 Use When: You need fast mutable strings in single-threaded code.

StringBuilder vs StringBuffer

StringBuilder:
• Faster (not synchronized)
• NOT thread-safe
• Use in single-threaded applications

StringBuffer:
• Slower (synchronized)
• Thread-safe
• Use in multi-threaded applications

Example: Efficient String Building

public class StringBuilderDemo {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Java");
        
        // append()
        sb.append(" Programming");
        System.out.println(sb);  // Java Programming
        
        // insert()
        sb.insert(4, " 17");
        System.out.println(sb);  // Java 17 Programming
        
        // replace()
        sb.replace(5, 7, "21");
        System.out.println(sb);  // Java 21 Programming
        
        // delete()
        sb.delete(4, 7);
        System.out.println(sb);  // Java Programming
        
        // Convert to String
        String result = sb.toString();
        System.out.println("Final: " + result);
    }
}
