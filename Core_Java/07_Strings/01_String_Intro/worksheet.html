Practice Questions

Master the basics of Java Strings, including literals, memory Pool, and immutability.

Question 1: Literal vs New Keyword

Write a Java program to demonstrate the difference between String Literals and the new keyword using the == operator.
class _01_ { public static void main(String[] args) { String s1 = "Java"; String s2 = "Java"; String s3 = new String("Java"); System.out.println("s1 == s2: " + (s1 == s2)); // true (String Pool) System.out.println("s1 == s3: " + (s1 == s3)); // false (Heap) } }

Question 2: String Immutability

Write a Java program to prove that Strings are immutable by trying to change a value through a method.
class _02_ { public static void main(String[] args) { String s = "Hello"; s.concat(" World"); System.out.println("String after concatenation: " + s); // "Hello" (unchanged) s = s.concat(" World"); System.out.println("String after reassignment: " + s); // "Hello World" (new object) } }

Question 3: Memory Pool Interning

Use the intern() method to manually add a String created with new to the String Pool.
class _03_ { public static void main(String[] args) { String s1 = "Java"; String s2 = new String("Java"); String s3 = s2.intern(); System.out.println("s1 == s2: " + (s1 == s2)); // false System.out.println("s1 == s3: " + (s1 == s3)); // true (interned) } }
