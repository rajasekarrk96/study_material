Question 1: Lexicographical Comparison

Write a Java program to compare two strings lexicographically. Two strings are lexicographically equal if they are the same length and contain the same characters in the same positions.
import java.util.Scanner; class _01_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter string 1: "); String s1 = sn.next(); System.out.print("Enter string 2: "); String s2 = sn.next(); int result = s1.compareTo(s2); if (result == 0) System.out.println("Strings are equal"); else if (result > 0) System.out.println("First string is greater"); else System.out.println("Second string is greater"); sn.close(); } }
Enter string 1: Apple Enter string 2: Banana Second string is greater

Question 5: Count Vowels

Write a Java method that takes a string as input and returns the number of vowels (a, e, i, o, u) in the string.
import java.util.Scanner; class _05_ { static int countVowels(String s) { int count = 0; for (char c : s.toLowerCase().toCharArray()) if ("aeiou".indexOf(c) != -1) count++; return count; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter string: "); String s = sn.nextLine(); System.out.println("Vowel count: " + countVowels(s)); sn.close(); } }

Question 6: Count "the" Occurrences

Write a Java method that takes a string as input and returns the number of times the word "the" appears in the string.
import java.util.Scanner; class _06_ { static int countThe(String s) { int count = 0; String[] words = s.toLowerCase().split("\\s+"); for (String w : words) if (w.equals("the")) count++; return count; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter sentence: "); String s = sn.nextLine(); System.out.println("Count of 'the': " + countThe(s)); sn.close(); } }

Question 7: Unique Characters

Write a Java method that returns true if the string contains only unique characters; otherwise, it returns false.
import java.util.Scanner; import java.util.HashSet; class _07_ { static boolean isUnique(String s) { HashSet set = new HashSet<>(); for (char c : s.toCharArray()) { if (!set.add(c)) return false; } return true; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter word: "); String s = sn.next(); System.out.println("Is unique: " + isUnique(s)); sn.close(); } }

Question 8: Count Words

Write a Java method that takes a string as input and returns the number of words in the string.
import java.util.Scanner; class _08_ { static int wordCount(String s) { if (s == null || s.isEmpty()) return 0; String[] words = s.trim().split("\\s+"); return words.length; } public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter sentence: "); String s = sn.nextLine(); System.out.println("Word count: " + wordCount(s)); sn.close(); } }

Question 9: Reverse String Characters

Write a program in Java to print individual characters of a string in reverse order.
import java.util.Scanner; class _09_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter string: "); String s = sn.next(); System.out.print("Reversed: "); for (int i = s.length() - 1; i >= 0; i--) System.out.print(s.charAt(i)); System.out.println(); sn.close(); } }

Question 10: Second Most Frequent Character

Write a Java program to find the second most frequent character in a given string.
import java.util.Scanner; import java.util.HashMap; class _10_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter string: "); String s = sn.next(); HashMap map = new HashMap<>(); for (char c : s.toCharArray()) map.put(c, map.getOrDefault(c, 0) + 1); char first = 0, second = 0; int fmax = 0, smax = 0; for (char c : map.keySet()) { int freq = map.get(c); if (freq > fmax) { smax = fmax; second = first; fmax = freq; first = c; } else if (freq > smax && freq != fmax) { smax = freq; second = c; } } System.out.println("Second most frequent character: " + second); sn.close(); } }
