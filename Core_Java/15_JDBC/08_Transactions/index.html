Database Transactions in JDBC

Ensure data consistency with ACID transactions

What are Transactions?

Transactions ensure that a series of database operations either all succeed or all fail together.

conn.setAutoCommit(false); // Start transaction
try {
    // Execute multiple SQL statements
    stmt1.executeUpdate("...");
    stmt2.executeUpdate("...");
    conn.commit(); // Commit if all successful
} catch (SQLException e) {
    conn.rollback(); // Rollback on error
}
