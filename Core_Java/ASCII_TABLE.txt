================================================================================
                            KEY RANGES
================================================================================

CONTROL CHARACTERS:     0-31   (NUL, SOH, STX, etc. - non-printable)
PRINTABLE ASCII:        32-126 (Standard keyboard and symbols)
EXTENDED ASCII:         128-255 (Special characters, accents, etc.)

SPECIFIC RANGES:
- Digits (0-9):         48-57
- Uppercase (A-Z):      65-90
- Lowercase (a-z):      97-122
- Special Symbols:      33-47, 58-64, 91-96, 123-126
- Whitespace:           9 (TAB), 10 (LF), 13 (CR), 32 (SPACE)

================================================================================
                    COMMON ASCII VALUE CONVERSIONS
================================================================================

JAVA CONVERSION CODE:
    char ch = 'A';
    int ascii = (int) ch;          // Result: 65
    
    int code = 65;
    char character = (char) code;  // Result: 'A'

COMMON VALUES:
    '0' = 48,  '9' = 57         Digits range
    'A' = 65,  'Z' = 90         Uppercase range
    'a' = 97,  'z' = 122        Lowercase range
    ' ' = 32                    Space
    '\n' = 10                   Newline
    '\t' = 9                    Tab
    '!' = 33                    Exclamation
    '@' = 64                    At symbol
    
DIFFERENCES:
    'a' - 'A' = 97 - 65 = 32    (lowercase is 32 higher than uppercase)
    '9' - '0' = 57 - 48 = 9     (each digit is 1 apart)
