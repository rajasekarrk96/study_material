Question 1: Full Star Pyramid

Problem Statement

Print a centered pyramid of stars.

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*
***
*****
*******
*********
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class FullPyramid { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { // Print spaces for (int j = 1; j <= n - i; j++) { System.out.print(" "); } // Print stars for (int k = 1; k <= 2 * i - 1; k++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Three loops for pyramid: Outer loop for rows, first inner loop for leading spaces, second inner loop for stars. This spaces-then-stars pattern is fundamental to all centered patterns.
  2. Space formula: n - i: Row 1 has n-1 spaces, row 2 has n-2 spaces, etc. As row increases, spaces decrease, pushing stars toward center. This creates pyramid alignment.
  3. Stars formula: 2*i - 1: Creates odd number sequence (1, 3, 5, 7, 9). Row 1 has 1 star, row 2 has 3, row 3 has 5. Odd numbers create symmetric expansion.
  4. Why 2*i - 1?: For symmetry! Row i needs i stars on left of center, center star, i-1 stars on right = 2i-1 total. This formula builds symmetric pyramids automatically.

Question 2: Inverted Pyramid

Problem Statement

Print an upside-down pyramid (starts wide, narrows to point).

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*********
*******
*****
***
*
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class InvertedPyramid { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = n; i >= 1; i--) { for (int j = 1; j <= n - i; j++) { System.out.print(" "); } for (int k = 1; k <= 2 * i - 1; k++) { System.out.print("*"); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Reverse outer loop for inversion: Counting i from n down to 1 flips the pyramid. First row (i=n) has most stars, last row (i=1) has one star. Direction reversal inverts the shape.
  2. Same formulas work in reverse: n-i for spaces and 2*i-1 for stars still work! When i is large (start), spaces are few and stars are many. As i decreases, spaces increase and stars decrease.
  3. Pattern symmetry principle: Many patterns have inverse versions. To invert, usually just reverse the loop direction. The mathematical relationships often don't need changing.
  4. Diamond is pyramid + inverted: Understanding this pyramid prepares you for diamond patterns, which combine upward and inverted pyramids. Building blocks combine into complex shapes.

Question 3: Number Pyramid

Problem Statement

Print a pyramid with numbers increasing towards center (1 2 3 2 1 pattern).

Sample Input and Output

Sample Input 1:
4

Sample Output 1:
1
121
12321
1234321
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class NumberPyramid { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= n - i; j++) { System.out.print(" "); } for (int k = 1; k <= i; k++) { System.out.print(k); } for (int m = i - 1; m >= 1; m--) { System.out.print(m); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Four loops for complex pattern: Spaces, ascending numbers, descending numbers, then newline. Breaking complex patterns into phases (spaces, left half, right half) makes them manageable.
  2. Ascending then descending creates symmetry: First number loop prints 1,2,3... up to i. Second prints i-1, i-2... down to 1. Together they form mirrored number sequence.
  3. Why start descending at i-1?: If we started at i, center would repeat (1231321 not 123321). Starting one below avoids duplication, creating proper palindrome.
  4. Palindromic pattern technique: This ascending-descending approach works for any symmetric pattern. Numbers, letters, symbols - same technique. Master it for all palindromic patterns.

Question 4: Hollow Pyramid

Problem Statement

Print a pyramid with stars only on edges, hollow inside.

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
*
* *
* *
* *
*********
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class HollowPyramid { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= n - i; j++) { System.out.print(" "); } for (int k = 1; k <= 2 * i - 1; k++) { if (k == 1 || k == 2 * i - 1 || i == n) { System.out.print("*"); } else { System.out.print(" "); } } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Conditional printing for hollow effect: Print star if first position (k==1) OR last position (k==2*i-1) OR bottom row (i==n). Else print space. This creates border-only pattern.
  2. Base must be filled: Condition i==n ensures bottom row is completely filled. Without it, base would be hollow too, which looks wrong for pyramid.
  3. Spaces maintain alignment: Printing space (not nothing) for hollow parts keeps structure aligned. Hollow doesn't mean absent - it means visible empty space character.
  4. OR logic for borders: Left edge OR right edge OR base - combining these with || handles all border cases. This OR pattern applies to hollowing any shape.

Question 5: Pascal's Triangle Numbers

Problem Statement

Print Pascal's Triangle pattern (each number is sum of two above it).

Sample Input and Output

Sample Input 1:
5

Sample Output 1:
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
▼ Show Solution

Solution (Java Program)

import java.util.Scanner; public class PascalTriangle { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter N: "); int n = sn.nextInt(); for (int i = 0; i < n; i++) { for (int j = 0; j < n - i - 1; j++) { System.out.print(" "); } int num = 1; for (int k = 0; k <= i; k++) { System.out.print(num + " "); num = num * (i - k) / (k + 1); } System.out.println(); } sn.close(); } }

Step-by-Step Explanation

  1. Mathematical formula for coefficients: Each number in Pascal's triangle is a binomial coefficient. Formula num = num * (i-k) / (k+1) generates next number from current without storing previous rows.
  2. Start each row with num=1: Every row in Pascal's triangle starts with 1. We initialize num=1 per row, then use formula to generate subsequent numbers.
  3. Pattern has deep math connections: Pascal's triangle appears in probability, algebra, combinatorics. It's not just a pattern exercise - it's fundamental to mathematics and computer science.
  4. Algorithmic vs storage approach: Could store previous row in array and sum pairs. This formula approach calculates on-the-fly, showing how math can replace storage. Both valid, formula is more elegant.
