Question 1: Check Two Floating-Point Numbers Same Up to Three Decimal Places

Problem Statement

Write a Java program that reads two floating-point numbers and tests whether they are the same up to three decimal places.

Solution

import java.util.Scanner; class _01_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first floating number: "); double a = sn.nextDouble(); System.out.print("Enter second floating number: "); double b = sn.nextDouble(); int x = (int)(a * 1000); int y = (int)(b * 1000); if (x == y) System.out.println("Same up to three decimal places"); else System.out.println("Not same up to three decimal places"); sn.close(); } }

Sample Output

Enter first floating number: 3.14159 Enter second floating number: 3.14156 Same up to three decimal places
Key Concepts:
  • Precision Comparison: Multiply by 1000 to shift 3 decimal places
  • Type Casting: Convert to int to truncate decimal places
  • Wrapper Class: Uses Double wrapper for floating-point operations
  • Equality Check: Compare integer representations for precision
Logic: 3.14159 × 1000 = 3141.59 → (int) 3141, 3.14156 × 1000 = 3141.56 → (int) 3141

Question 2: Check Whether a String is a Valid Number Using Wrapper Class

Problem Statement

Write a Java program that validates whether the string is a valid number using wrapper class.

Solution

import java.util.Scanner; class _02_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter a string: "); String s = sn.next(); try { Double d = Double.parseDouble(s); System.out.println("Valid Number"); } catch (NumberFormatException e) { System.out.println("Invalid Number"); } sn.close(); } }

Sample Output

Enter a string: 123.45 Valid Number Enter a string: abc Invalid Number
Key Concepts:
  • parseDouble(): Wrapper class method to convert String to Double
  • Exception Handling: try-catch block handles invalid input
  • NumberFormatException: Thrown when string cannot be parsed as number
  • Validation: Safe way to check if string represents valid number
Wrapper Methods: Integer.parseInt(), Double.parseDouble(), Boolean.parseBoolean()

Question 3: Compare Two Numbers Using compareTo() Method

Problem Statement

Write a Java program by using compareTo() method in wrapper class for comparing two numbers.

Solution

import java.util.Scanner; class _03_ { public static void main(String[] args) { Scanner sn = new Scanner(System.in); System.out.print("Enter first number: "); Integer a = sn.nextInt(); System.out.print("Enter second number: "); Integer b = sn.nextInt(); int result = a.compareTo(b); if (result == 0) System.out.println("Both numbers are equal"); else if (result > 0) System.out.println("First number is greater"); else System.out.println("Second number is greater"); sn.close(); } }

Sample Output

Enter first number: 25 Enter second number: 15 First number is greater Enter first number: 10 Enter second number: 10 Both numbers are equal
Key Concepts:
  • compareTo(): Wrapper class method for object comparison
  • Return Values: 0 (equal), positive (greater), negative (less)
  • Integer Wrapper: Converts primitive int to Integer object
  • Object Comparison: Enables sorting and ordering operations
compareTo() Returns: a.compareTo(b) → 0 if a==b, >0 if a>b, <0 if a
