Autoboxing & Unboxing

Autoboxing and Unboxing

Automatic conversion between primitives and wrapper objects.

Autoboxing

Automatic conversion of primitive types to their corresponding wrapper objects.

// Manual boxing (old way)
Integer num1 = Integer.valueOf(10);

// Autoboxing (automatic)
Integer num2 = 10;  // Compiler converts int to Integer

Unboxing

Automatic conversion of wrapper objects to primitive types.

Integer num = 100;

// Manual unboxing (old way)
int value1 = num.intValue();

// Auto-unboxing (automatic)
int value2 = num;  // Compiler converts Integer to int

Example: Collections with Autoboxing

import java.util.ArrayList;

public class AutoboxingDemo {
    public static void main(String[] args) {
        ArrayList numbers = new ArrayList<>();
        
        // Autoboxing - int to Integer
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        
        // Auto-unboxing - Integer to int
        int sum = 0;
        for (int num : numbers) {  // Auto-unboxing here
            sum += num;
        }
        
        System.out.println("Sum: " + sum);
        
        // Autoboxing in method calls
        Integer max = Math.max(10, 20);  // Returns Integer
        System.out.println("Max: " + max);
    }
}
Sum: 60
Max: 20
