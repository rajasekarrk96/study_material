ArrayList

ArrayList in Java

Dynamic, resizable array implementation from the Collections Framework.

What is ArrayList?

ArrayList is a resizable array implementation that can grow or shrink dynamically. Unlike arrays, you don't need to specify the size at creation.

ArrayList vs Array:
Feature Array ArrayList
Size Fixed Dynamic (resizable)
Type Primitives & Objects Objects only
Performance Faster Slightly slower
Methods Limited Many built-in methods
When to Use ArrayList:
  • Size is unknown or changes frequently
  • Need to add/remove elements dynamically
  • Want to use built-in methods (sort, search, etc.)

Declaration and Creation

Basic Syntax:
import java.util.ArrayList;

ArrayList<Type> listName = new ArrayList<>();
Examples:
import java.util.ArrayList;

public class ArrayListDeclaration {
    public static void main(String[] args) {
        // ArrayList of Strings
        ArrayList<String> names = new ArrayList<>();
        
        // ArrayList of Integers
        ArrayList<Integer> numbers = new ArrayList<>();
        
        // ArrayList of Doubles
        ArrayList<Double> prices = new ArrayList<>();
        
        // ArrayList with initial capacity
        ArrayList<String> fruits = new ArrayList<>(10);
        
        System.out.println("ArrayLists created successfully!");
    }
}
ArrayLists created successfully!
💡 Note: ArrayList stores objects only. For primitives, use wrapper classes (Integer, Double, Boolean, etc.)

Common ArrayList Methods

Adding Elements:
ArrayList<String> fruits = new ArrayList<>();

fruits.add("Apple");           // Add at end
fruits.add("Banana");
fruits.add(1, "Orange");       // Add at index 1

System.out.println(fruits);    // [Apple, Orange, Banana]
Accessing Elements:
String first = fruits.get(0);      // Get element at index 0
int size = fruits.size();          // Get size
boolean isEmpty = fruits.isEmpty(); // Check if empty

System.out.println("First fruit: " + first);
System.out.println("Total fruits: " + size);
Modifying Elements:
fruits.set(0, "Mango");    // Replace element at index 0
System.out.println(fruits); // [Mango, Orange, Banana]
Removing Elements:
fruits.remove(0);              // Remove by index
fruits.remove("Banana");       // Remove by value
fruits.clear();                // Remove all elements
Searching:
boolean hasApple = fruits.contains("Apple");
int index = fruits.indexOf("Orange");  // Returns -1 if not found
Complete Example:
import java.util.ArrayList;

public class ArrayListMethods {
    public static void main(String[] args) {
        ArrayList<String> colors = new ArrayList<>();
        
        // Add elements
        colors.add("Red");
        colors.add("Green");
        colors.add("Blue");
        System.out.println("Colors: " + colors);
        
        // Get element
        String firstColor = colors.get(0);
        System.out.println("First color: " + firstColor);
        
        // Size
        System.out.println("Total colors: " + colors.size());
        
        // Modify
        colors.set(1, "Yellow");
        System.out.println("After modification: " + colors);
        
        // Contains
        System.out.println("Has Blue? " + colors.contains("Blue"));
        
        // Remove
        colors.remove("Red");
        System.out.println("After removal: " + colors);
    }
}
Colors: [Red, Green, Blue]
First color: Red
Total colors: 3
After modification: [Red, Yellow, Blue]
Has Blue? true
After removal: [Yellow, Blue]

Iterating Through ArrayList

Method 1: For Loop
ArrayList<Integer> numbers = new ArrayList<>();
numbers.add(10);
numbers.add(20);
numbers.add(30);

for (int i = 0; i < numbers.size(); i++) {
    System.out.println(numbers.get(i));
}
Method 2: For-Each Loop
for (Integer num : numbers) {
    System.out.println(num);
}
Method 3: Iterator
import java.util.Iterator;

Iterator<Integer> it = numbers.iterator();
while (it.hasNext()) {
    System.out.println(it.next());
}
Method 4: forEach (Java 8+)
numbers.forEach(num -> System.out.println(num));

Practical Examples

Example 1: Student Management
import java.util.ArrayList;

public class StudentList {
    public static void main(String[] args) {
        ArrayList<String> students = new ArrayList<>();
        
        // Add students
        students.add("Alice");
        students.add("Bob");
        students.add("Charlie");
        students.add("Diana");
        
        System.out.println("Total students: " + students.size());
        System.out.println("\nStudent List:");
        
        for (int i = 0; i < students.size(); i++) {
            System.out.println((i + 1) + ". " + students.get(i));
        }
        
        // Search for a student
        String searchName = "Bob";
        if (students.contains(searchName)) {
            System.out.println("\n" + searchName + " is in the class!");
        }
    }
}
Total students: 4

Student List:
1. Alice
2. Bob
3. Charlie
4. Diana

Bob is in the class!
Example 2: Shopping Cart
import java.util.ArrayList;

public class ShoppingCart {
    public static void main(String[] args) {
        ArrayList<String> cart = new ArrayList<>();
        ArrayList<Double> prices = new ArrayList<>();
        
        // Add items
        cart.add("Laptop");
        prices.add(999.99);
        
        cart.add("Mouse");
        prices.add(25.50);
        
        cart.add("Keyboard");
        prices.add(75.00);
        
        // Display cart
        System.out.println("Shopping Cart:");
        double total = 0;
        
        for (int i = 0; i < cart.size(); i++) {
            System.out.printf("%s - $%.2f\n", cart.get(i), prices.get(i));
            total += prices.get(i);
        }
        
        System.out.printf("\nTotal: $%.2f\n", total);
    }
}
Shopping Cart:
Laptop - $999.99
Mouse - $25.50
Keyboard - $75.00

Total: $1100.49
Example 3: Sorting ArrayList
import java.util.ArrayList;
import java.util.Collections;

public class SortArrayList {
    public static void main(String[] args) {
        ArrayList<Integer> numbers = new ArrayList<>();
        numbers.add(45);
        numbers.add(12);
        numbers.add(78);
        numbers.add(23);
        numbers.add(56);
        
        System.out.println("Original: " + numbers);
        
        // Sort ascending
        Collections.sort(numbers);
        System.out.println("Sorted (ascending): " + numbers);
        
        // Sort descending
        Collections.sort(numbers, Collections.reverseOrder());
        System.out.println("Sorted (descending): " + numbers);
    }
}
Original: [45, 12, 78, 23, 56]
Sorted (ascending): [12, 23, 45, 56, 78]
Sorted (descending): [78, 56, 45, 23, 12]
