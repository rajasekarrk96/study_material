LinkedList

LinkedList in Java

Doubly-linked list implementation for efficient insertions and deletions.

What is LinkedList?

LinkedList is a doubly-linked list implementation where each element (node) contains references to both the next and previous elements. This makes insertions and deletions at any position very efficient.

LinkedList vs ArrayList:
Feature ArrayList LinkedList
Internal Structure Dynamic Array Doubly Linked List
Access Time Fast (O(1)) Slow (O(n))
Insert/Delete at Beginning Slow (O(n)) Fast (O(1))
Insert/Delete at End Fast (O(1)) Fast (O(1))
Insert/Delete in Middle Slow (O(n)) Fast (O(1)) after finding position
Memory Usage Less (only data) More (data + 2 pointers)
When to Use LinkedList:
  • Frequent insertions/deletions at beginning or middle
  • Implementing queues or deques
  • Don't need random access to elements
  • Memory is not a constraint
When to Use ArrayList:
  • Need fast random access by index
  • Mostly reading data, few modifications
  • Want to save memory

Declaration and Creation

Basic Syntax:
import java.util.LinkedList;

LinkedList<Type> listName = new LinkedList<>();
Examples:
import java.util.LinkedList;

public class LinkedListDeclaration {
    public static void main(String[] args) {
        // LinkedList of Strings
        LinkedList<String> names = new LinkedList<>();
        
        // LinkedList of Integers
        LinkedList<Integer> numbers = new LinkedList<>();
        
        // LinkedList of Doubles
        LinkedList<Double> prices = new LinkedList<>();
        
        System.out.println("LinkedLists created successfully!");
    }
}
LinkedLists created successfully!
💡 Note: LinkedList implements both List and Deque interfaces, so it can be used as a list, stack, or queue.

Common LinkedList Methods

Adding Elements:
LinkedList<String> fruits = new LinkedList<>();

fruits.add("Apple");           // Add at end
fruits.add("Banana");
fruits.add(1, "Orange");       // Add at index 1
fruits.addFirst("Mango");      // Add at beginning
fruits.addLast("Grapes");      // Add at end

System.out.println(fruits);    // [Mango, Apple, Orange, Banana, Grapes]
Accessing Elements:
String first = fruits.get(0);        // Get element at index 0
String firstElement = fruits.getFirst();  // Get first element
String lastElement = fruits.getLast();    // Get last element
int size = fruits.size();                 // Get size

System.out.println("First fruit: " + first);
System.out.println("Total fruits: " + size);
Modifying Elements:
fruits.set(0, "Pineapple");    // Replace element at index 0
System.out.println(fruits);     // [Pineapple, Apple, Orange, Banana, Grapes]
Removing Elements:
fruits.remove(0);              // Remove by index
fruits.remove("Banana");       // Remove by value
fruits.removeFirst();          // Remove first element
fruits.removeLast();           // Remove last element
fruits.clear();                // Remove all elements
Searching:
boolean hasApple = fruits.contains("Apple");
int index = fruits.indexOf("Orange");  // Returns -1 if not found
Queue Operations:
LinkedList<String> queue = new LinkedList<>();

queue.offer("First");          // Add to end (queue)
queue.offer("Second");
queue.offer("Third");

String head = queue.peek();    // View first element without removing
String removed = queue.poll(); // Remove and return first element

System.out.println("Head: " + head);
System.out.println("Removed: " + removed);
System.out.println("Queue: " + queue);
Stack Operations:
LinkedList<String> stack = new LinkedList<>();

stack.push("Bottom");          // Add to beginning (stack)
stack.push("Middle");
stack.push("Top");

String top = stack.peek();     // View top element
String popped = stack.pop();   // Remove and return top element

System.out.println("Top: " + top);
System.out.println("Popped: " + popped);
System.out.println("Stack: " + stack);
Complete Example:
import java.util.LinkedList;

public class LinkedListMethods {
    public static void main(String[] args) {
        LinkedList<String> colors = new LinkedList<>();
        
        // Add elements
        colors.add("Red");
        colors.add("Green");
        colors.add("Blue");
        colors.addFirst("Yellow");
        colors.addLast("Purple");
        System.out.println("Colors: " + colors);
        
        // Get elements
        String firstColor = colors.getFirst();
        String lastColor = colors.getLast();
        System.out.println("First: " + firstColor + ", Last: " + lastColor);
        
        // Size
        System.out.println("Total colors: " + colors.size());
        
        // Modify
        colors.set(2, "Orange");
        System.out.println("After modification: " + colors);
        
        // Contains
        System.out.println("Has Blue? " + colors.contains("Blue"));
        
        // Remove
        colors.removeFirst();
        colors.removeLast();
        System.out.println("After removal: " + colors);
    }
}
Colors: [Yellow, Red, Green, Blue, Purple]
First: Yellow, Last: Purple
Total colors: 5
After modification: [Yellow, Red, Orange, Blue, Purple]
Has Blue? true
After removal: [Red, Orange, Blue]

Iterating Through LinkedList

Method 1: For Loop
LinkedList<Integer> numbers = new LinkedList<>();
numbers.add(10);
numbers.add(20);
numbers.add(30);

for (int i = 0; i < numbers.size(); i++) {
    System.out.println(numbers.get(i));
}
Method 2: For-Each Loop (Recommended)
for (Integer num : numbers) {
    System.out.println(num);
}
Method 3: Iterator
import java.util.Iterator;

Iterator<Integer> it = numbers.iterator();
while (it.hasNext()) {
    System.out.println(it.next());
}
Method 4: Descending Iterator
Iterator<Integer> descIt = numbers.descendingIterator();
while (descIt.hasNext()) {
    System.out.println(descIt.next());  // Prints in reverse order
}
Method 5: forEach (Java 8+)
numbers.forEach(num -> System.out.println(num));
⚠️ Warning: Avoid using regular for loop with get(i) for LinkedList as it's inefficient (O(n²)). Use for-each loop or iterator instead.

Practical Examples

Example 1: Task Queue
import java.util.LinkedList;

public class TaskQueue {
    public static void main(String[] args) {
        LinkedList<String> tasks = new LinkedList<>();
        
        // Add tasks to queue
        tasks.offer("Send Email");
        tasks.offer("Write Report");
        tasks.offer("Call Client");
        tasks.offer("Update Database");
        
        System.out.println("Tasks in Queue: " + tasks.size());
        System.out.println("\nProcessing Tasks:");
        
        // Process tasks (FIFO - First In First Out)
        while (!tasks.isEmpty()) {
            String currentTask = tasks.poll();
            System.out.println("Processing: " + currentTask);
        }
        
        System.out.println("\nAll tasks completed!");
    }
}
Tasks in Queue: 4

Processing Tasks:
Processing: Send Email
Processing: Write Report
Processing: Call Client
Processing: Update Database

All tasks completed!
Example 2: Browser History (Stack)
import java.util.LinkedList;

public class BrowserHistory {
    public static void main(String[] args) {
        LinkedList<String> history = new LinkedList<>();
        
        // Visit pages (push to stack)
        history.push("google.com");
        history.push("youtube.com");
        history.push("github.com");
        history.push("stackoverflow.com");
        
        System.out.println("Current page: " + history.peek());
        System.out.println("\nGoing back through history:");
        
        // Go back (pop from stack - LIFO)
        while (!history.isEmpty()) {
            System.out.println("← " + history.pop());
        }
    }
}
Current page: stackoverflow.com

Going back through history:
← stackoverflow.com
← github.com
← youtube.com
← google.com
Example 3: Playlist Manager
import java.util.LinkedList;

public class PlaylistManager {
    public static void main(String[] args) {
        LinkedList<String> playlist = new LinkedList<>();
        
        // Add songs
        playlist.add("Song A");
        playlist.add("Song B");
        playlist.add("Song C");
        playlist.add("Song D");
        
        System.out.println("Original Playlist:");
        displayPlaylist(playlist);
        
        // Add song at beginning
        playlist.addFirst("Intro Song");
        
        // Add song at specific position
        playlist.add(3, "Bonus Track");
        
        System.out.println("\nUpdated Playlist:");
        displayPlaylist(playlist);
        
        // Remove first and last songs
        playlist.removeFirst();
        playlist.removeLast();
        
        System.out.println("\nFinal Playlist:");
        displayPlaylist(playlist);
    }
    
    static void displayPlaylist(LinkedList<String> list) {
        for (int i = 0; i < list.size(); i++) {
            System.out.println((i + 1) + ". " + list.get(i));
        }
    }
}

Example 4: Performance Comparison
import java.util.ArrayList;
import java.util.LinkedList;

public class PerformanceTest {
    public static void main(String[] args) {
        int size = 100000;
        
        // ArrayList - Insert at beginning
        ArrayList<Integer> arrayList = new ArrayList<>();
        long start = System.currentTimeMillis();
        for (int i = 0; i < size; i++) {
            arrayList.add(0, i);  // Insert at beginning
        }
        long arrayTime = System.currentTimeMillis() - start;
        
        // LinkedList - Insert at beginning
        LinkedList<Integer> linkedList = new LinkedList<>();
        start = System.currentTimeMillis();
        for (int i = 0; i < size; i++) {
            linkedList.addFirst(i);  // Insert at beginning
        }
        long linkedTime = System.currentTimeMillis() - start;
        
        System.out.println("Inserting " + size + " elements at beginning:");
        System.out.println("ArrayList: " + arrayTime + "ms");
        System.out.println("LinkedList: " + linkedTime + "ms");
        System.out.println("\nLinkedList is " + (arrayTime / linkedTime) + "x faster!");
    }
}
Inserting 100000 elements at beginning:
ArrayList: 2847ms
LinkedList: 12ms

LinkedList is 237x faster!
