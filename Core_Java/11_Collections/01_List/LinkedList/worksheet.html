Question 1: Append Element to the End of a LinkedList

Problem Statement

Write a Java program to append the specified element to the end of a linked list.

Solution

import java.util.LinkedList; class _01_ { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Red"); list.add("Green"); list.add("Blue"); list.addLast("Yellow"); System.out.println(list); } }

Sample Output

[Red, Green, Blue, Yellow]
Key Concepts:
  • addLast(): Adds element to the end of the list
  • add(): Also adds to end, same as addLast()
  • Doubly-Linked: Each node has next and previous pointers
  • O(1) Performance: Constant time for adding at end
Deque Interface: LinkedList implements Deque, providing addLast()

Question 2: Iterate a LinkedList in Reverse Order

Problem Statement

Write a Java program to iterate a linked list in reverse order.

Solution

import java.util.LinkedList; import java.util.Iterator; class _02_ { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("One"); list.add("Two"); list.add("Three"); Iterator<String> it = list.descendingIterator(); while (it.hasNext()) { System.out.println(it.next()); } } }

Sample Output

Three Two One
Key Concepts:
  • descendingIterator(): Returns iterator in reverse order
  • Bidirectional: Can traverse both forward and backward
  • Iterator Pattern: hasNext() and next() methods
  • Unique to LinkedList: ArrayList doesn't have this method
Alternative: Can use ListIterator with hasPrevious() and previous()

Question 3: Insert Elements at First and Last Position

Problem Statement

Write a Java program to insert elements into the linked list at the first and last position.

Solution

import java.util.LinkedList; class _03_ { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Green"); list.add("Blue"); list.addFirst("Red"); list.addLast("Yellow"); System.out.println(list); } }

Sample Output

[Red, Green, Blue, Yellow]
Key Concepts:
  • addFirst(): Inserts element at the beginning
  • addLast(): Inserts element at the end
  • O(1) Performance: Both operations are constant time
  • Efficient: Much faster than ArrayList for these operations
Use Case: Ideal for implementing queues and deques

Question 4: Insert Elements at a Specified Position

Problem Statement

Write a Java program to insert some elements at the specified position into a linked list.

Solution

import java.util.LinkedList; class _04_ { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Red"); list.add("Blue"); list.add("Yellow"); list.add(1, "Green"); System.out.println(list); } }

Sample Output

[Red, Green, Blue, Yellow]
Key Concepts:
  • add(index, element): Inserts at specified position
  • Index 1: Second position (0-based indexing)
  • Node Linking: Updates pointers to insert new node
  • Performance: O(n) to find position, O(1) to insert
Note: Must traverse to position first, then insert

Question 5: Get First and Last Occurrence of an Element

Problem Statement

Write a Java program to get the first and last occurrence of the specified elements in a linked list.

Solution

import java.util.LinkedList; class _05_ { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Red"); list.add("Green"); list.add("Blue"); list.add("Green"); list.add("Yellow"); System.out.println("First occurrence of Green: " + list.indexOf("Green")); System.out.println("Last occurrence of Green: " + list.lastIndexOf("Green")); } }

Sample Output

First occurrence of Green: 1 Last occurrence of Green: 3
Key Concepts:
  • indexOf(): Returns index of first occurrence
  • lastIndexOf(): Returns index of last occurrence
  • Returns -1: If element is not found
  • Linear Search: O(n) time complexity for both methods
Duplicates: LinkedList allows duplicate elements unlike Set
