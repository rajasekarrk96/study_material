Question 1: Count Number of Key-Value Mappings

Problem Statement

Write a Java program to count the number of key-value (size) mappings in a map.

Solution

import java.util.HashMap; class _01_ { public static void main(String[] args) { HashMap<Integer, String> map = new HashMap<>(); map.put(1, "Red"); map.put(2, "Green"); map.put(3, "Blue"); System.out.println("Size of map: " + map.size()); } }

Sample Output

Size of map: 3
Key Concepts:
  • size(): Returns the number of key-value pairs in the map
  • put(): Adds a key-value mapping to the HashMap
  • HashMap: Stores data in key-value pairs for fast retrieval
  • Integer Key: Keys can be any object type, here using Integer
Use Cases: Checking map capacity, validating data, loop conditions

Question 2: Get a Shallow Copy of a HashMap

Problem Statement

Write a Java program to get a shallow copy of a HashMap instance.

Solution

import java.util.HashMap; class _02_ { public static void main(String[] args) { HashMap<Integer, String> map1 = new HashMap<>(); map1.put(1, "Apple"); map1.put(2, "Banana"); HashMap<Integer, String> map2 = new HashMap<>(map1); System.out.println("Original Map: " + map1); System.out.println("Shallow Copy: " + map2); } }

Sample Output

Original Map: {1=Apple, 2=Banana} Shallow Copy: {1=Apple, 2=Banana}
Key Concepts:
  • Copy Constructor: new HashMap<>(map1) creates a shallow copy
  • Shallow Copy: Copies references, not the actual objects
  • Independent Maps: Changes to one map don't affect the other
  • Alternative: Can also use clone() method
Note: For deep copy of mutable objects, manual copying is needed

Question 3: Check if Map Contains a Specified Key

Problem Statement

Write a Java program to test if a map contains a mapping for the specified key.

Solution

import java.util.HashMap; class _03_ { public static void main(String[] args) { HashMap<Integer, String> map = new HashMap<>(); map.put(1, "Java"); map.put(2, "Python"); System.out.println(map.containsKey(1)); System.out.println(map.containsKey(3)); } }

Sample Output

true false
Key Concepts:
  • containsKey(): Returns true if key exists, false otherwise
  • Fast Lookup: HashMap provides O(1) average time complexity
  • Null Keys: HashMap allows one null key
  • Validation: Useful before accessing values to avoid null
Best Practice: Check key existence before using get() to avoid NullPointerException

Question 4: Check if Map Contains a Specified Value

Problem Statement

Write a Java program to test if a map contains a mapping for the specified value.

Solution

import java.util.HashMap; class _04_ { public static void main(String[] args) { HashMap<Integer, String> map = new HashMap<>(); map.put(1, "Cat"); map.put(2, "Dog"); System.out.println(map.containsValue("Dog")); System.out.println(map.containsValue("Lion")); } }

Sample Output

true false
Key Concepts:
  • containsValue(): Searches for value in the map
  • Linear Search: O(n) time complexity as it checks all values
  • Slower than containsKey(): Values are not indexed
  • Multiple Values: Returns true if at least one key maps to the value
Performance: Use containsKey() when possible for better performance

Question 5: Create a Set View of the Mappings

Problem Statement

Write a Java program to create a set view of the mappings contained in a map.

Solution

import java.util.HashMap; import java.util.Map; import java.util.Set; class _05_ { public static void main(String[] args) { HashMap<Integer, String> map = new HashMap<>(); map.put(1, "One"); map.put(2, "Two"); map.put(3, "Three"); Set<Map.Entry<Integer, String>> set = map.entrySet(); for (Map.Entry<Integer, String> e : set) { System.out.println(e.getKey() + " = " + e.getValue()); } } }

Sample Output

1 = One 2 = Two 3 = Three
Key Concepts:
  • entrySet(): Returns a Set view of all key-value pairs
  • Map.Entry: Interface representing a key-value pair
  • getKey(): Retrieves the key from an entry
  • getValue(): Retrieves the value from an entry
Alternatives: keySet() for keys only, values() for values only
