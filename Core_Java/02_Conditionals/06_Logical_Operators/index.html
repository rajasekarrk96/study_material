Logical Operators in Java

Combine multiple conditions using AND, OR, and NOT operators.

Logical Operators

Operator Name Description Example
&& Logical AND True if both conditions are true (a > 5 && b < 10)
|| Logical OR True if at least one condition is true (a > 5 || b < 10)
! Logical NOT Reverses the boolean value !(a > 5)

Example: Login Validation

public class LogicalOperatorsExample {
    public static void main(String[] args) {
        // AND operator example
        int age = 25;
        boolean hasLicense = true;
        
        if (age >= 18 && hasLicense) {
            System.out.println("You can drive");
        } else {
            System.out.println("You cannot drive");
        }
        
        // OR operator example
        String day = "Saturday";
        
        if (day.equals("Saturday") || day.equals("Sunday")) {
            System.out.println("It's a weekend!");
        } else {
            System.out.println("It's a weekday");
        }
        
        // NOT operator example
        boolean isRaining = false;
        
        if (!isRaining) {
            System.out.println("You don't need an umbrella");
        } else {
            System.out.println("Take an umbrella");
        }
        
        // Combined example
        int marks = 75;
        boolean attendance = true;
        
        if (marks >= 70 && attendance) {
            System.out.println("Passed with good attendance");
        } else if (marks >= 70 && !attendance) {
            System.out.println("Passed but poor attendance");
        } else {
            System.out.println("Failed");
        }
    }
}
