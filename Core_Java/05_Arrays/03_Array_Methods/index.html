Array Methods in Java

Use built-in array methods like sort(), binarySearch(), and toString().

java.util.Arrays Class

The Arrays class provides static methods to manipulate arrays. Import it to use these powerful utility methods.

import java.util.Arrays;
💡 Key Point: All methods in Arrays class are static, so call them using Arrays.methodName()

Sorting Arrays

import java.util.Arrays;

public class ArraySort {
    public static void main(String[] args) {
        int[] numbers = {5, 2, 8, 1, 9, 3};
        
        System.out.println("Before sort: " + Arrays.toString(numbers));
        
        // Sort in ascending order
        Arrays.sort(numbers);
        
        System.out.println("After sort: " + Arrays.toString(numbers));
        
        // Sort specific range (index 1 to 4)
        int[] arr = {9, 5, 3, 7, 1, 8};
        Arrays.sort(arr, 1, 4);
        System.out.println("Partial sort: " + Arrays.toString(arr));
    }
}
Before sort: [5, 2, 8, 1, 9, 3]
After sort: [1, 2, 3, 5, 8, 9]
Partial sort: [9, 3, 5, 7, 1, 8]

Binary Search

import java.util.Arrays;

public class ArraySearch {
    public static void main(String[] args) {
        int[] numbers = {1, 3, 5, 7, 9, 11, 13};
        
        // Array MUST be sorted for binarySearch
        int index = Arrays.binarySearch(numbers, 7);
        System.out.println("Index of 7: " + index);
        
        // Search for non-existent element
        int notFound = Arrays.binarySearch(numbers, 6);
        System.out.println("Index of 6: " + notFound);  // Returns negative value
    }
}
Index of 7: 3
Index of 6: -4
⚠️ Important: binarySearch() requires a sorted array. Returns negative value if element not found.

Copying Arrays

import java.util.Arrays;

public class ArrayCopy {
    public static void main(String[] args) {
        int[] original = {1, 2, 3, 4, 5};
        
        // Method 1: copyOf()
        int[] copy1 = Arrays.copyOf(original, original.length);
        System.out.println("Copy: " + Arrays.toString(copy1));
        
        // Method 2: copyOfRange()
        int[] copy2 = Arrays.copyOfRange(original, 1, 4);
        System.out.println("Range copy: " + Arrays.toString(copy2));
        
        // Method 3: clone()
        int[] copy3 = original.clone();
        System.out.println("Clone: " + Arrays.toString(copy3));
    }
}
Copy: [1, 2, 3, 4, 5]
Range copy: [2, 3, 4]
Clone: [1, 2, 3, 4, 5]

Other Useful Methods

import java.util.Arrays;

public class ArrayMethods {
    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3};
        int[] arr2 = {1, 2, 3};
        int[] arr3 = {1, 2, 4};
        
        // equals() - compare arrays
        System.out.println("arr1 equals arr2: " + Arrays.equals(arr1, arr2));
        System.out.println("arr1 equals arr3: " + Arrays.equals(arr1, arr3));
        
        // fill() - fill array with value
        int[] arr = new int[5];
        Arrays.fill(arr, 7);
        System.out.println("Filled: " + Arrays.toString(arr));
        
        // toString() - convert to string
        String[] names = {"Alice", "Bob", "Charlie"};
        System.out.println("Names: " + Arrays.toString(names));
    }
}
arr1 equals arr2: true
arr1 equals arr3: false
Filled: [7, 7, 7, 7, 7]
Names: [Alice, Bob, Charlie]
✓ Common Methods:
• Arrays.sort() - Sort array
• Arrays.binarySearch() - Search sorted array
• Arrays.toString() - Convert to string
• Arrays.equals() - Compare arrays
• Arrays.fill() - Fill with value
• Arrays.copyOf() - Copy array
