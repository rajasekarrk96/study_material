Pattern and Matcher Classes

Pattern and Matcher Classes

Use Pattern and Matcher classes for regex operations.

Pattern and Matcher

  • Pattern: Compiled regex pattern
  • Matcher: Performs matching operations
Pattern p = Pattern.compile("regex");
Matcher m = p.matcher("text to search");

Common Matcher Methods

Method Description
find() Find next match
matches() Match entire string
group() Get matched text
start() Get start index
end() Get end index
replaceAll() Replace all matches

Example: Complete Pattern Matching

import java.util.regex.*;

public class PatternMatcherDemo {
    public static void main(String[] args) {
        String text = "Email: john@example.com, jane@test.com";
        
        // Email pattern
        Pattern p = Pattern.compile("\\w+@\\w+\\.\\w+");
        Matcher m = p.matcher(text);
        
        System.out.println("Emails found:");
        while (m.find()) {
            System.out.println("- " + m.group());
            System.out.println("  Position: " + m.start() + "-" + m.end());
        }
        
        // Replace emails
        String masked = m.replaceAll("[HIDDEN]");
        System.out.println("\nMasked: " + masked);
    }
}
Emails found:
- john@example.com
  Position: 7-23
- jane@test.com
  Position: 25-38

Masked: Email: [HIDDEN], [HIDDEN]
