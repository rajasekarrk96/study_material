Question 1: Extract Area Code and Phone Number

Problem Statement

Write a Java program to extract the Area Code and the Main Number from a phone number string formatted as (123) 456-7890.

Requirements:
  • Capture the 3 digits inside the parentheses as Group 1 (Area Code).
  • Capture the remaining 7 digits as Group 2 (Phone Number).

Solution

import java.util.regex.*; public class ExtractGroups { public static void main(String[] args) { String phone = "(555) 987-1234"; // \\((\\d{3})\\) matches (555) and captures 555 in group 1 // \\s* matches optional space // (\\d{3}-\\d{4}) captures 987-1234 in group 2 Pattern p = Pattern.compile("\\((\\d{3})\\)\\s*(\\d{3}-\\d{4})"); Matcher m = p.matcher(phone); if (m.find()) { System.out.println("Full Match: " + m.group(0)); System.out.println("Area Code: " + m.group(1)); System.out.println("Phone Num: " + m.group(2)); } } }

Sample Output

Full Match: (555) 987-1234 Area Code: 555 Phone Num: 987-1234

Regex Pattern Explanation

\\((\\d{3})\\)\\s*(\\d{3}-\\d{4}) Breakdown: - \\( and \\) : Escaped parentheses to match literal '(' and ')' - (\\d{3}) : FIRST capturing group for 3 digits (Area Code) - \\s* : Zero or more whitespace characters - (\\d{3}-\\d{4}) : SECOND capturing group for the rest of the number
Key Concepts:
  • Capturing Groups (): Used to treat multiple characters as a single unit and save them for extraction.
  • Numbered Groups: group(0) refers to the entire match, group(1) to the first parentheses set, and so on.
  • Escaping: Special characters like ( or . must be escaped with \\ to be matched literally.
