Regex Groups

Regex Groups

Capture and extract parts of matched patterns.

What are Groups?

Groups allow you to capture parts of a match using parentheses (). Each group can be accessed separately.

  • () creates a capturing group
  • group(0) returns entire match
  • group(1), group(2)... return captured groups

Example: Extracting Date Parts

import java.util.regex.*;

public class GroupDemo {
    public static void main(String[] args) {
        String date = "2024-01-15";
        
        // Pattern with 3 groups: (year)-(month)-(day)
        Pattern p = Pattern.compile("(\\d{4})-(\\d{2})-(\\d{2})");
        Matcher m = p.matcher(date);
        
        if (m.find()) {
            System.out.println("Full match: " + m.group(0));
            System.out.println("Year: " + m.group(1));
            System.out.println("Month: " + m.group(2));
            System.out.println("Day: " + m.group(3));
        }
    }
}
Full match: 2024-01-15
Year: 2024
Month: 01
Day: 15
