Regex Predefined Classes

Predefined Character Classes

Use shorthand character classes like \d, \w, \s in regex patterns.

Common Predefined Classes

Class Meaning Example
\d Any digit [0-9] Matches: 5, 9
\D Non-digit Matches: a, @
\w Word character [a-zA-Z0-9_] Matches: a, 5, _
\W Non-word character Matches: @, #
\s Whitespace Matches: space, tab
\S Non-whitespace Matches: a, 5

Example: Using Predefined Classes

import java.util.regex.*;

public class PredefinedDemo {
    public static void main(String[] args) {
        String text = "Phone: 123-456-7890";
        
        // \d+ matches one or more digits
        Pattern p = Pattern.compile("\\d+");
        Matcher m = p.matcher(text);
        
        System.out.println("Numbers found:");
        while (m.find()) {
            System.out.println(m.group());
        }
    }
}
Numbers found:
123
456
7890
