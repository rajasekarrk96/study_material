Question 1: Validate a 10-Digit Mobile Number

Problem Statement

Write a Java program to validate a mobile number. A valid mobile number must:
  • Contain exactly 10 digits
  • Not contain any letters or special characters
Use the predefined character class \d for this validation.

Solution

import java.util.Scanner; public class MobileValidator { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Enter mobile number: "); String mobile = sc.next(); // \\d{10} means exactly 10 digits if (mobile.matches("\\d{10}")) { System.out.println("Valid Mobile Number"); } else { System.out.println("Invalid Mobile Number"); } } }

Sample Output

Input: 9876543210 Output: Valid Mobile Number Input: 12345 Output: Invalid Mobile Number (too short) Input: 98765abc10 Output: Invalid Mobile Number (contains letters)

Regex Pattern Explanation

\\d{10} Breakdown: - \\d : Predefined class matching any digit [0-9] - {10} : Quantifier - exactly 10 occurrences
Key Concepts:
  • Predefined Class \d: Shorthand for [0-9]. Remember to use double backslash \\d in Java strings.
  • Exact Match {n}: The quantifier ensures the length is exactly as specified.
  • matches() method: Validates the entire string against the pattern.

Question 2: Remove All Whitespaces from a String

Problem Statement

Write a Java program to remove all types of whitespace characters (spaces, tabs, newlines) from a given string using regex.

Example: "J a v a Is P o w e r f u l" → "JavaIsPowerful"

Solution

public class RemoveWhitespace { public static void main(String[] args) { String str = "J a v a Is P o w e r f u l"; // \\s matches any whitespace character String result = str.replaceAll("\\s", ""); System.out.println("Original: " + str); System.out.println("Cleaned: " + result); } }

Sample Output

Original: J a v a Is P o w e r f u l Cleaned: JavaIsPowerful

Regex Pattern Explanation

\\s Breakdown: - \\s : Predefined class matching any whitespace character (space, \t, \n, \r, \f)
Key Concepts:
  • Predefined Class \s: Matches spaces, tabs, and other invisible characters.
  • replaceAll(): Replaces every character matching the pattern with the specified replacement string.
