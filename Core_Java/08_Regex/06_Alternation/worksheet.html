Question 1: Match Domain Extensions

Problem Statement

Write a Java program to check if a website URL ends with either .com, .org, or .net using alternation.

Example URLs to test:
  • google.com
  • wikipedia.org
  • mynetwork.net
  • portfolio.me (should not match)

Solution

public class AlternationDemo { public static void main(String[] args) { String[] websites = {"google.com", "wikipedia.org", "mynetwork.net", "portfolio.me"}; // .* matches anything before the extension // \\. matches the literal dot // (com|org|net) matches one of the three extensions // $ ensures it matches at the end of the string String regex = ".*\\.(com|org|net)$"; for (String site : websites) { if (site.matches(regex)) { System.out.println(site + " is a valid match"); } else { System.out.println(site + " is NOT a valid match"); } } } }

Sample Output

google.com is a valid match wikipedia.org is a valid match mynetwork.net is a valid match portfolio.me is NOT a valid match

Regex Pattern Explanation

.*\\.(com|org|net)$ Breakdown: - .* : Matches any sequence of characters - \\. : Matches the literal dot '.' character - ( : Start of a capturing group (optional, can also be non-capturing) - com|org|net : The alternation logic - match "com" OR "org" OR "net" - ) : End of the group - $ : End of string anchor
Key Concepts:
  • Alternation (|): Acts like a logical OR between multiple patterns.
  • Grouping: Alternation is often placed inside parentheses () to limit its scope to that specific part of the regex.
  • Literal Dot: Remember that . in regex matches any character, so \\. is required for a literal period.
